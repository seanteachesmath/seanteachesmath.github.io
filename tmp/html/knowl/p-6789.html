We integrate by parts, using \(u = x\text{,}\) \(\dee{v} =\sin x \,\dee{x}\) so that \(v=-\cos x\) and \(\dee{u} = \dee{x}\text{:}\)

\begin{align*} \int_0^\pi x\sin x\,\dee{x} &= \underbrace{-x\cos x}_{uv} \Big|_0^\pi -\int_0^\pi \underbrace{(-\cos x)}_v\,\underbrace{\dee{x}}_{\dee{u}}\\ &= \Big[-x\cos x +\sin x\Big]_0^\pi =-\pi(-1) =\pi \end{align*}
