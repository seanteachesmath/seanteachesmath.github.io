2.4.2.50.a When we evaluate \(\displaystyle\lim_{x \rightarrow -4^-} f(x)\text{,}\) we only consider values of \(x\) that are less than \(-4\text{.}\) For these values, \(f(x)=x^3+8x^2+16x\text{.}\) So,

\begin{gather*} \displaystyle\lim_{x \rightarrow 4^-} f(x)=\lim_{x \rightarrow -4^-} (x^3+8x^2+16x)=(-4)^3+8(-4)^2+16(-4)=0 \end{gather*}

Note that, because \(x^3+8x^2+16x\) is a polynomial, we can evaluate the limit by directly substituting in \(x=-4\text{.}\)
