Remark: We can verify that vector field (d) is indeed conservative by observing (look at the figure (d) on the right above) that the \(\hi\) component of the vector field is exactly zero and that the \(\hj\) component depends only on \(y\text{.}\) So the vector field is of the form

\begin{equation*} \vF(x,y) = a(y)\,\hj \end{equation*}

for some function \(a(y)\text{.}\) If \(A(y)\) is any antiderivative of \(a(y)\text{,}\) we have \(\vF=\vnabla A\text{,}\) so that \(\vF\) is conservative with potential \(A(y)\text{.}\)
