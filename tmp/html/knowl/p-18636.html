(c) For any fixed \(y\text{,}\) \(x^2+z^2=1\) is a circle of radius \(1\text{.}\) So we can parametrize it by \(x(\theta)=\cos\theta\text{,}\) \(z(\theta)=\sin\theta\text{,}\) \(0\le\theta \lt 2\pi\text{.}\) The \(y\)-coordinate of any point on the intersection is determined by \(y=xz\text{.}\) So we can use

\begin{equation*} \vr(\theta) = \cos\theta\,\hi +\sin\theta\,\hk + \sin\theta\,\cos\theta\,\hj \qquad 0\le\theta \lt 2\pi \end{equation*}
