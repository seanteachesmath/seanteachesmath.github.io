(a) Each (horizontal) constant \(z\) cross-section is a circle centred on the \(z\)-axis. The radius varies linearly from \(2\text{,}\) when \(z=0\) to \(0\text{,}\) when \(z=3\text{.}\) So the radius at height \(z\) is \(\frac{2}{3}(3-z)\) and we can use

\begin{align*} &\vr(\theta,z) = \frac{2}{3}(3-z)\cos\theta\ \hi +\frac{2}{3}(3-z)\sin\theta\ \hj +z\,\hk\\ &0\le\theta \lt 2\pi,\quad 0\le z\le 3 \end{align*}

as the parametrization.
