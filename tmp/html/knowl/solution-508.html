Solution 4.4.11.3.8.1.

For each of these, there are many solutions. We provide some below.

  1. \(\displaystyle 1+2+3+4+5 = \ds\sum_{n=1}^5n\)
  2. \(\displaystyle 2+4+6+8=\ds\sum_{n=1}^4 2n\)
  3. \(\displaystyle 3+5+7+9+11=\ds\sum_{n=1}^{5}(2n+1)\)
  4. \(\displaystyle 9+16+25+36+49=\ds\sum_{n=3}^7 n^2\)
  5. \(\displaystyle 9+4+16+5+25+6+36+7+49+8=\ds\sum_{n=3}^7 (n^2+n+1)\)
  6. \(\displaystyle 8+15+24+35+48= \ds\sum_{n=3}^7 (n^2-1)\)

  7. \(3-6+9-12+15-18=\ds\sum_{n=1}^6 (-1)^{n+1}3n\)

    Remark: if we had written \((-1)^n\) instead of \((-1)^{n+1}\text{,}\) with everything else the same, the signs would have been reversed.
