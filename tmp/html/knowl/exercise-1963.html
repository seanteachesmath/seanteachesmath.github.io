Exercise 4.

Consider a curve that is parametrized by arc length \(s\text{.}\)

  1. Show that if the curve has curvature \(\ka(s)=0\) for all \(s\text{,}\) then the curve is a straight line.
  2. Show that if the curve has curvature \(\ka(s) \gt 0\) and torsion \(\tau(s)=0\) for all \(s\text{,}\) then the curve lies in a plane.
  3. Show that if the curve has curvature \(\ka(s)=\ka_0\text{,}\) a strictly positive constant, and torsion \(\tau(s)=0\) for all \(s\text{,}\) then the curve is a circle.
