Solution:

  • We use integration by parts to solve the integral.
  • Set \(u(x)=x\) and \(v'(x)=\sin x\text{.}\) Then \(u'(x)=1\) and \(v(x)=-\cos x\text{,}\) and
    \begin{align*} \int x \sin x \dee{x} &= -x\cos x + \int \cos x \dee{x}\\ &= -x\cos x + \sin x + C. \end{align*}
