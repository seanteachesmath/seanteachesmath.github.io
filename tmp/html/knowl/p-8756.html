By definition, the average value in question equals

\begin{equation*} \text{Ave}=\frac1{\pi/k-0} \int_0^{\pi/k} \sin(kx) \,\dee{x} \end{equation*}

To evaluate the integral, we use the substitution \(u = kx\text{,}\) \(\dee{u}=k\,\dee{x}\text{.}\) Note that the endpoints \(x=0\) and \(x=\pi/k\) become \(u=0\) and \(u=\pi\text{,}\) respectively. So

\begin{gather*} \text{Ave}=\frac{k}{\pi} \int_0^\pi \sin(u) \,\frac{\dee{u}}{k} =\frac{1}{\pi}\Big[-\cos(u)\Big]_0^\pi =\frac{2}{\pi} \end{gather*}
