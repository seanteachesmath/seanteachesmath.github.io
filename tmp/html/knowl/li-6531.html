Item 12.4.1.27.h.

Let

\begin{equation*} \vu := 2\,\hi,\quad \vv := \hj + \sqrt{3}\,\hk\quad \vw := -\sqrt{3}\,\hj + \hk \end{equation*}
  1. Express \(\hi\text{,}\) \(\hj\text{,}\) \(\hk\) in terms of \(\vu\text{,}\) \(\vv\text{,}\) \(\vw\text{.}\)
  2. Using (i), write \(\vr(t)\) in the form
    \begin{equation*} a(t)\vu + b(t)\vv + c(t)\vw \end{equation*}
    where \(a(t)\text{,}\) \(b(t)\) and \(c(t)\) are functions you have to determine. You should find that one of these functions is zero.
  3. Draw the curve given by \(\big(a(t), b(t)\big)\) in the \(xy\)-plane.
  4. Is the drawing consistent with parts (f) and (g)? Explain.
