Example 3.10.4. The derivative of \(\log|x|\).

We can push this further still. Let \(g(x) = \log | x |\text{,}\) then  5 

  • If \(x \gt 0\text{,}\) \(|x|= x\) and so
    \begin{align*} g'(x) &= \diff{}{x} \log x = \frac{1}{x} \end{align*}
  • If \(x \lt 0\) then \(|x|= -x =-1\cdot x\) and so
    \begin{align*} g'(x) &= \diff{}{x} \log(-1 \cdot x) = \frac{1}{x} \end{align*}
    by the previous example.
  • Since \(\log 0\) is undefined, \(g'(0)\) does not exist.

Putting this together gives:

\begin{align*} \diff{}{x} \log | x | &= \frac{1}{x} \end{align*}
