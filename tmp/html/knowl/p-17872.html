Set

\begin{align*} V_b &=\Set{(x,y,z)}{0\le z \le 1-x^2-y^2,\ x^2+y^2\le 1}\\ V_c &=\Set{(x,y,z)}{x^2+y^2-1 \le z \le 1-x^2-y^2,\ x^2+y^2\le 1} \end{align*}

Then \(\partial V_b = \sigma_1\cup\sigma_3\) and \(\partial V_c = \sigma_1\cup\sigma_2\text{,}\) all with outward pointing normals. Since the divergence of \(\vF\) is

\begin{equation*} \vnabla\cdot\vF = \pdiff{}{x}[ a ( y^2 \!+\! z^2 ) + bxz ] +\pdiff{}{y}[ c ( x^2 \!+\! z^2 ) + dyz ] +\pdiff{}{z}[ x^2] = (b+d) z \end{equation*}

the divergence theorem gives

\begin{align*} \dblInt_{\sigma_1\cup\sigma_3} \vF\cdot\hn\,\dee{S} & = \tripInt_{V_b} \vnabla\cdot\vF\,\dee{V} = (b+d)\tripInt_{V_b} z\,\dee{V}\\ \dblInt_{\sigma_1\cup\sigma_2} \vF\cdot\hn\,\dee{S} & = \tripInt_{V_c} \vnabla\cdot\vF\,\dee{V} = (b+d)\tripInt_{V_c} z\,\dee{V} \end{align*}

Now on \(V_b\text{,}\) \(z\ge 0\) and \(z \gt 0\) except on \(\sigma_3\text{.}\) So \(\tripInt_{V_b} z\,\dee{V} \gt 0\) and \(\dblInt_{\sigma_1\cup\sigma_3} \vF\cdot\hn\,\dee{S}\) is zero if and only if \(d=-b\text{.}\) That's the answer to part (b).
