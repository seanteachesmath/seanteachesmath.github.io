  1. If \(C\) is a closed curve and \(\vnabla f=\vZero\text{,}\) then \(\int_C f\,\dee{s}=0\text{.}\)
  2. If \(\vr(t)\) is a parametrization of a smooth curve \(C\) and the binormal \(\vB(t)\) is constant then \(C\) is a straight line.
  3. If \(\vr(t)\) is the position of a particle which travels with constant speed, then \(\vr'(t)\cdot\vr''(t)=0\text{.}\)
  4. If \(C\) is a path from points \(A\) to \(B\text{,}\) then the line integral \(\int_C\big(\vF\times\vG\big)\cdot\dee{\vr}\) is independent of the path \(C\text{.}\)
  5. The line integral \(\int_C f\,\dee{s}\) does not depend of the orientation of the curve \(C\text{.}\)
  6. If \(S\) is a parametric surface \(\vr(u,v)\) then a normal to \(S\) is given by
    \begin{gather*} \frac{\partial\vr}{\partial u}\times \frac{\partial\vr}{\partial u} \end{gather*}
  7. The surface area of the parametric surface \(S\) given by \(\vr(u,v) = x(u,v)\,\hi + y(u,v)\,\hj + z(u,v)\,\hk\text{,}\) \((u,v)\in D\text{,}\) is given by
    \begin{gather*} \dblInt_D \left(1+\big(\tfrac{\partial z}{\partial u}\big)^2 +\big(\tfrac{\partial z}{\partial v}\big)^2\right)^{1/2} \dee{u}\dee{v} \end{gather*}
  8. If \(\vF\) is the velocity field of an incompressible fluid then \(\vnabla\cdot\vF=0\text{.}\)
  9. \(\displaystyle \vnabla\cdot\big(\vF\times\vG\big) = (\vnabla\cdot\vF)\vG + (\vnabla\cdot\vG)\vF\)
