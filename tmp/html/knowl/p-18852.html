(c) True, assuming that \(\vr(t)\) is not indentically \(\vZero\text{.}\) If \(\vr(t)\) and \(\diff{\vr}{t}\) are orthogonal at all points of the curve \(C\text{,}\) then

\begin{equation*} \diff{ }{t} \big(\vr(t)\cdot\vr(t)\big) = 2\vr(t)\cdot\diff{\vr}{t}(t) =0 \end{equation*}

So \(x(t)^2 + y(t)^2 + z(t)^2 = \vr(t)\cdot\vr(t)\) is a constant. If \(\vr(t)\) is not indentically \(\vZero\text{,}\) that constant must be strictly positive. That is \(x(t)^2 + y(t)^2 + z(t)^2 = a^2\) for some constant \(a \gt 0\text{.}\)
