Questions 13.4.2.36 and 13.4.2.37 ask you to find a path that leads to a particular value of a line integral. Many such paths are possible — you only need to find one.

36. M317 2012D.

    \begin{equation*} \int_L \vF\cdot\dee{\vr} \end{equation*}
    where \(L\) is the line segment from \((2, 2)\) to \((1, 1)\text{.}\)
  2. Find an oriented path \(C\) from \((2, 2)\) to \((1, 1)\) such that
    \begin{equation*} \int_C \vF\cdot\dee{\vr} =4 \end{equation*}
    where \(\vF\) is the vector field from (a).
