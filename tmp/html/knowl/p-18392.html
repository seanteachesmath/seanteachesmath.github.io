(a) Since

  • \(\frac{y}{x}\) is defined when \(x\ne 0\) and
  • \(x^{1+x^2}=e^{(1+x^2)\ln x}\) is defined when \(\ln x\) is defined, which is when \(x \gt 0\) (assuming that we are not allowed to use complex numbers) and
  • \(y^{1+y^2}=e^{(1+y^2)\ln y}\) is defined when \(\ln y\) is defined, which is when \(y \gt 0\) and
  • \(\cos^5(\ln z)\) is defined when \(\ln z\) is defined, which when \(z \gt 0\)

the domain of \(\vF\) is

\begin{equation*} D=\Set{(x,y,z)}{x \gt 0,\ y \gt 0,\ z \gt 0} \end{equation*}
