Solution 14.1.1.3.1.

Note that, since \(x^2+y^2=1+2z^2\) on \(S\text{,}\) the condition \(z\ge 1\) is equivalent to \(x^2+y^2\ge 3\text{,}\) \(z\ge 0\text{.}\) So the hyperboloid is \(\Set{(x,y,z)}{x^2+y^2=1+2z^2,\ 3\le x^2+y^2\le 9, z\ge 0}\text{.}\)

(a) No. Under this parametrization, the condition \(3\le x^2+y^2\le 9\) is \(3\le u^2+v^2\le 9\text{,}\) not \(2\le u^2+v^2\le 9\text{.}\)

(b) Yes. Under this parametrization, \(x=u\sin v\text{,}\) \(y=-u\cos v\) and \(z=\sqrt{\frac{u^2}{2}-\frac{1}{2}}\text{.}\) So

  • \(x^2+y^2-2z^2 = u^2-2\left(\frac{u^2}{2}-\frac{1}{2}\right) = 1\text{,}\) as desired.
  • The condition \(x^2+y^2\le 9\) is equivalent to \(u\le 3\text{,}\) since \(u\ge 0\text{.}\)
  • The condition \(x^2+y^2\ge 3\) is equivalent to \(u\ge\sqrt{3}\text{,}\) since \(u\ge 0\text{.}\)
  • \(\displaystyle z=\sqrt{\frac{u^2}{2}-\frac{1}{2}}\ge 0\)

(c) Yes. Under this parametrization, \(x=\sqrt{1+2v^2}\cos u\text{,}\) \(y=\sqrt{1+2v^2}\sin u\) and \(z=v\text{.}\) So

  • \(x^2+y^2-2z^2 = 1+2v^2 -2v^2 = 1\text{,}\) as desired.
  • The condition \(x^2+y^2\le 9\) is equivalent to \(1+2v^2\le 9\text{,}\) which is equivalent to \(v\le 2\text{,}\) since \(v\ge 0\text{.}\)
  • The condition \(x^2+y^2\ge 3\) is equivalent to \(1+2v^2\ge 3\text{,}\) which is equivalent to \(v\ge 1\text{,}\) since \(v\ge 0\text{.}\)
  • \(\displaystyle z=v\ge 0\)

(d) Yes. Under this parametrization, \(x=\sqrt{1+u}\sin v\text{,}\) \(y=\sqrt{1+u}\cos v\) and \(z=\sqrt{u/2}\text{.}\) So

  • \(x^2+y^2-2z^2 = 1+u -2(u/2) = 1\text{,}\) as desired.
  • The condition \(x^2+y^2\le 9\) is equivalent to \(1+u\le 9\text{,}\) which is equivalent to \(u\le 8\text{.}\)
  • The condition \(x^2+y^2\ge 3\) is equivalent to \(1+u\ge 3\text{,}\) which is equivalent to \(u\ge 2\text{.}\)
  • \(\displaystyle z=\sqrt{u/2}\ge 0\)

(e) No. Under this parametrization, \(x=\sqrt{u}\cos v\text{,}\) \(y=-\sqrt{u}\sin v\) and \(z=\sqrt{(u+1)/2}\text{.}\) So

  • \(x^2+y^2-2z^2 = u -2(u+1)/2 = -1\text{,}\) not \(+1\)
