  1. True or false? If \(\vr(t)\) is the position at time \(t\) of an object moving in \(\bbbr^3\text{,}\) and \(\vr(t)\) is twice differentiable, then \(|\vr''(t)|\) is the tangential component of its acceleration.
  2. Let \(\vr(t)\) is a smooth curve in \(\bbbr^3\) with unit tangent, normal and binormal vectors \(\hat\vT(t)\text{,}\) \(\hat\vN(t)\text{,}\) \(\hat\vB(t)\text{.}\) Which two of these vectors span the plane normal to the curve at \(\vr(t)\text{?}\)
  3. True or false? If \(\vF = P\hi + Q\hj + R\hk\) is a vector field on \(\bbbr^3\) such that \(P\text{,}\) \(Q\text{,}\) \(R\) have continuous first order derivatives, and if \(\vnabla\times\vF = \vZero\) everywhere on \(\bbbr^3\) , then \(\vF\) is conservative.
  4. True or false? If \(\vF = P\hi + Q\hj + R\hk\) is a vector field on \(\bbbr^3\) such that \(P\text{,}\) \(Q\text{,}\) \(R\) have continuous second order derivatives, then \(\vnabla\times(\vnabla\cdot F) = 0\text{.}\)
  5. True or false? If \(\vF\) is a vector field on \(\bbbr^3\) such that \(|\vF(x, y, z)| = 1\) for all \(x\text{,}\) \(y\text{,}\) \(z\text{,}\) and if \(S\) is the sphere \(x^2 + y^2 + z^2 = 1\text{,}\) then \(\dblInt_S \vF \cdot\hn\,\dee{S} = 4\pi\text{.}\)
  6. True or false? Every closed surface \(S\) in \(\bbbr^3\) is orientable. (Recall that \(S\) is closed if it is the boundary of a solid region \(E\text{.}\))
