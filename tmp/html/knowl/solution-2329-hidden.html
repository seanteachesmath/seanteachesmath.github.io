(a) Expressing the left hand side as an iterated integral, with \(y\) as the inner integration variable, we have

\begin{align*} \dblInt_R\frac{\partial f}{\partial y}(x,y)\ \dee{x}\,\dee{y} &=\int_0^1\dee{x}\left[\int_0^1\dee{y}\ \frac{\partial f}{\partial y}(x,y)\right]\\ &=\int_0^1\dee{x}\ \big[f(x,1)-f(x,0)\big]\\ &\hskip1in\text{by the fundamental theorem of calculus}\\ &=\int_0^1 f(x,1)\ \dee{x} - \int_0^1 f(x,0)\ \dee{x} \end{align*}

(b) Define \(F_1(x,y)=f(x,y)\) and \(F_2(x,y)=0\text{.}\) Then. by Green's theorem

\begin{align*} \dblInt_R\frac{\partial f}{\partial y}(x,y)\ \dee{x}\,\dee{y} &=-\dblInt_R\Big[\frac{\partial F_2}{\partial x}(x,y) -\frac{\partial F_1}{\partial y}(x,y)\Big]\ \dee{x}\,\dee{y}\\ &=-\int_{\partial R}\big[F_1(x,y)\,\dee{x} +F_2(x,y)\,\dee{y}\big]\\ &=-\int_{\partial R}f(x,y)\,\dee{x} \end{align*}

The boundary of \(R\text{,}\) oriented counterclockwise, is the union of four line segments.

\begin{gather*} C_1 \text{ from } (0,0) \text{ to } (1,0)\\ C_2 \text{ from } (1,0) \text{ to } (1,1)\\ C_3 \text{ from } (1,1) \text{ to } (0,1)\\ C_4 \text{ from } (0,1) \text{ to } (0,0) \end{gather*}

Now \(x\) is constant on \(C_2\) and \(C_4\) so that

\begin{equation*} \int_{C_2}f(x,y)\,\dee{x} = \int_{C_4}f(x,y)\,\dee{x} =0 \end{equation*}

So, using \(-C_3\) to denote the line segment from \((0,1)\) to \((1,1)\)

\begin{align*} \dblInt_R\frac{\partial f}{\partial y}(x,y)\ \dee{x}\,\dee{y} &=-\left[\int_{C_1}f(x,y)\,\dee{x}+\int_{C_3}f(x,y)\,\dee{x}\right]\\ &=\int_{-C_3}f(x,y)\,\dee{x}-\int_{C_1}f(x,y)\,\dee{x}\\ &=\int_0^1 f(x,1)\ \dee{x} - \int_0^1 f(x,0)\ \dee{x} \end{align*}
