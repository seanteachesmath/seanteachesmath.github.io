Solution 8.3.11.1.1.
  1. \(\displaystyle\lim_{n \to \infty}\frac{1}{n}=0\text{,}\) so the divergence test is inconclusive. It's true that this series diverges, but we can't show it using the divergence test.
  2. \(\displaystyle\lim_{n \to \infty}\frac{n^2}{n+1}=\infty\text{,}\) which is not zero, so the divergence test tells us this series diverges.
  3. \(\displaystyle\lim_{n \to \infty}\sin n\) does not exist, so in particular it is not zero. Therefore, the divergence test tells us this series diverges.
  4. For all whole numbers \(n\text{,}\) \(\sin(\pi n)=0\text{,}\) so \(\displaystyle\lim_{n \to \infty}\sin(\pi n)=0\) and the divergence test is inconclusive.
