Consider the vector field \(\vF(x, y, z) = 2x\,\hi + 2y\,\hj + 2z\,\hk\text{.}\)

  1. Compute \(\vnabla\times\vF\text{.}\)
  2. If \(C\) is any path from \((0, 0, 0)\) to \((a_1 , a_2 , a_3)\) and \(\va = a_1\,\hi + a_2\,\hj + a_3\,\hk\text{,}\) show that \(\int_C \vF\cdot \dee{\vr} = \va\cdot\va\text{.}\)
