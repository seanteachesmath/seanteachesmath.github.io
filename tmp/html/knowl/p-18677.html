(h) False. Let \(\vF\) be an everywhere defined conservative vector field with potential \(\varphi\text{.}\) Then \(\vnabla\times\vF=\vZero\) everywhere. If \(P\) and \(Q\) are two points and if \(\varphi(P)-\varphi(Q)=3\) and if \(C\) is a curve from \(Q\) to \(P\text{,}\) then \(\int_C \vF \cdot \dee{\vr} = 3\text{.}\) One example would be \(\varphi(x,y,z) = x\text{,}\) \(\vF=\hi\text{,}\) \(P=(3,0,0)\text{,}\) \(Q=(0,0,0)\text{.}\)
