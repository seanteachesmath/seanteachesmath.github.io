There is no reason to think 3.14.2.2.i or 3.14.2.2.v holds, but to be thorough we will give an example showing that they do not need to be true. If \(f(x)=x^2-10\) and \((a,b)=(0,1)\text{,}\) then \(f'(x)=2x \gt 0\) over \((0,1)\text{,}\) and \(f''(x)=2 \gt 0\) everywhere, but \(f(x) \lt 0\) for all \(x \in (0,1)\text{,}\) so 3.14.2.2.i does not hold. Also, \(f'''(x)=0\) everywhere, so 3.14.2.2.v does not hold either.
