Item b.
A curve \(C\) is said to be simple if it does not cross itself. More precisely, if \(\vr(t)\text{,}\) \(a\le t\le b\text{,}\) is a parametrization of the curve and if \(a\le t_1,t_2\le b\) obey \(t_1\ne t_2\) and \(\{t_1,t_2\} \ne\{a,b\}\text{,}\) then \(\vr(t_1)\ne \vr(t_2)\text{.}\) That is, if \(\vr(t_1)=\vr(t_2)\text{,}\) then either \(t_1=t_2\) or \(t_1=a\text{,}\) \(t_2=b\text{,}\) or \(t_1=b\text{,}\) \(t_2=a\text{.}\)
