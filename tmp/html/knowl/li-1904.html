Choice #1: If we set \(\De x = \frac{2}{n}\) and \(x_i^*= \frac{2i}{n}\text{,}\) i.e. \(x_i^* = a + i\De x\) with \(a=0\text{,}\) then
\begin{align*} \lim_{n\rightarrow\infty} \bigg( \sum_{i=1}^n e^{-1-2i/n}\cdot \frac{2}{n} \bigg) &=\lim_{n\rightarrow\infty} \bigg( \sum_{i=1}^n e^{-1-x_i^*}\De x \bigg)\\ &=\lim_{n\rightarrow\infty} \bigg( \sum_{i=1}^n f(x_i^*)\De x \bigg)\\ &\hskip0.25in\text{with $f(x) = e^{-1-x}$}\\ &=\int_a^b f(x)\,\,\dee{x}\\ &\hskip0.25in\text{with $a=x_0=0$ and $b=x_n=2$}\\ &=\int_0^2 e^{-1-x}\ \,\dee{x} \end{align*}
