Let \(S = \{1,2\}\text{.}\) What are all the subsets of \(S\text{?}\) Well â€” each element of \(S\) can either be in the subset or not (independent of the other elements of the set). So we have \(2\times2 = 4\) possibilities: neither \(1\) nor \(2\) is in the subset, \(1\) is but \(2\) is not, \(2\) is but \(1\) is not, and both \(1\) and \(2\) are. That is

\begin{align*} \es, \{1\}, \{2\}, \{1,2\} & \subseteq S \end{align*}

This argument can be generalised with a little work to show that a set that contains exactly \(n\) elements has exactly \(2^n\) subsets.
