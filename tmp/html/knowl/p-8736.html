However, consider \(f(x) = \begin{cases} 100 & \text{ if } 0 \le x \le 0.01\\ 0 & \text{ else } \end{cases}\) and \(g(x) = 0\text{.}\) Then \(f(x) \le g(x)\) for all \(x\) in \([0.01,10]\text{,}\) but the average of \(f(x)\) is \(0.1\text{,}\) while the average of \(g(x)\) is 0.
