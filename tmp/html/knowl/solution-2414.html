Solution 16.2.8.1.

(a) False. The curve \(\vr_1\) contains only points with \(z\ge0\text{.}\) Every \(\vr_2(t)\) with \(t \lt 0\) has \(z \lt 0\text{.}\)

(b) True. \(\vr_2(t^2)=\vr_1(t)\) and \(t^2\) runs from \(0\) to \(1\) as \(t\) runs from \(0\) to \(1\text{.}\)

(c) True. In general \(|\vr'(t)| = \diff{s}{t}\text{.}\) When \(t=s\text{,}\) \(\diff{s}{t}=1\text{.}\)

(d) False. The curve need not even lie in a plane. For example, as we saw in Example 12.4.4, the helix \(\vr(t) = a\cos t\,\hi +b\sin t\,\hj + bt\,\hk\) has constant curvature \(\ka=\frac{a}{a^2+b^2}\) but is not a circle.

(e) True. If the speed \(|\vv|=\sqrt{\vv\cdot\vv}\) of a moving object is constant, then

\begin{equation*} 0=\diff{ }{t}\big(\vv\cdot\vv\big)=2\vv\cdot\va \end{equation*}

(f) False. If the vector field \(\vF(x, y, z) = \frac{-y}{x^2+y^2} \hi + \frac{x}{x^2+y^2} \hj + z\hk\) were conservative, its restriction, \(\frac{-y}{x^2+y^2} \hi+ \frac{x}{x^2+y^2} \hj\text{,}\) to the \(xy\)-plane would also be conservative. But we saw in Examples 13.3.14 and 15.3.8 that the vector field \(\frac{-y}{x^2+y^2} \hi+ \frac{x}{x^2+y^2} \hj\) is not conservative.

(g) False. The vector field of part (f), with domain \(\Set{(x,y,z)}{ x^2+y^2 \gt 1}\text{,}\) provides a counterexample.

(h) False. The curve \(x^2+y^2 = 2\) can not be shrunk to a point continuously in \(\Set{ (x, y) }{ x^2 + y^2 \gt 1 }\text{.}\)

(i) True. Any curve in \(\Set{ (x, y) }{ y \gt x^2 }\) can be shrunk to a point continuously in \(\Set{ (x, y) }{ y \gt x^2 }\text{.}\)

(j) True. By the divergence theorem,

\begin{gather*} \dblInt_S \vnabla\times \vF \cdot\hn\, \dee{S} =\tripInt_E \vnabla\cdot(\vnabla\times \vF)\, \dee{V} =0 \end{gather*}

since \(\vnabla\cdot(\vnabla\times \vF)=0\) by the vector identity of Theorem 15.1.7.a.
