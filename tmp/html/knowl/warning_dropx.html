Warning 3.14.5.

Many people will suppress the \((x)\) in \(y(x)\) when doing computations like those in Example 3.14.4. This gives shorter, easier to read formulae, like \(y'=\frac{y+3x^2}{1-x-3y^2}\text{.}\) If you do this, you must never forget that \(y\) is a function of \(x\) and is not a constant. If you do forget, you'll make the very serious error of saying that \(\diff{y}{x}=0\text{,}\) which is false.
