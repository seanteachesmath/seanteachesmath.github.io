So far we have just looked at one vector field on \(\cD\text{.}\) We are now ready to consider any vector field \(\vF\) on \(\cD\) that passes the screening test \(\vnabla\times\vF=0\) on \(\cD\text{.}\) We claim that there is a function \(\varphi\) on \(\cD\) such that

\begin{equation*} \vF=\alpha_\vF\,\vTh+\vnabla\varphi\qquad\text{where}\qquad \alpha_\vF = \frac{1}{2\pi}\oint_{\cC_\veps}\vF\cdot \dee{\vr} \tag{E2} \end{equation*}

The significance of this claim is that it says that if a vector field on \(\cD\) passes the screening test on \(\cD\text{,}\) then, either it is conservative (that's the case if and only if \(\alpha_\vF=0\)) or, if it fails to be conservative, then it differs from a conservative field (namely \(\vnabla\varphi\)) only by a constant (namely \(\alpha_\vF\)) times the fixed vector field \(\vTh\text{.}\) That is, there is only one nonconservative vector field on \(\cD\) that passes the screening test, up to multiplication by constants and addition of conservative fields. This is a nice simple surprise.
