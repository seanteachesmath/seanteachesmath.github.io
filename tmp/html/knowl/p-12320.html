Let

  • \(m\) and \(n\) be natural numbers 2 
  • \(\displaystyle \va\in \bbbr^m\)
  • the function \(f(\vx)\) be defined for all \(\vx\) near 3  \(\va\) and take values in \(\bbbr^n\)
  • \(\displaystyle \vL\in\bbbr^n\)

We write

\begin{gather*} \lim_{\vx\to \va} f(\vx) = \vL \end{gather*}

if 4  the value of the function \(f(\vx)\) is sure to be arbitrarily close to \(\vL\) whenever the value of \(\vx\) is close enough to \(\va\text{,}\) without 5  being exactly \(\va\text{.}\)
