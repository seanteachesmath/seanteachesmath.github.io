Let \(a \lt b\) and let \(f(x)\) be a function which is defined and continuous on \([a,b]\text{.}\)

  • Part 1. Let \(\ds F(x)=\int_a^x f(t)\dee{t}\) for any \(x \in[a,b]\text{.}\) Then the function \(F(x)\) is differentiable and further
    \begin{align*} F'(x) &=f(x) \end{align*}
  • Part 2. Let \(G(x)\) be any function which is defined and continuous on \([a,b]\text{.}\) Further let \(G(x)\) be differentiable with \(G'(x)=f(x)\) for all \(a \lt x \lt b\text{.}\) Then
    \begin{align*} \int_a^b f(x)\dee{x} &=G(b)-G(a) & \text{or equivalently} && \int_a^b G'(x)\dee{x} &=G(b)-G(a) \end{align*}
