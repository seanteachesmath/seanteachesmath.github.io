Since the average of \(f(x)\) on the interval \([0,5]\) is \(A\text{,}\) using DefinitionÂ 7.2.2,

\begin{align*} A&=\frac{1}{5}\int_0^5 f(x)\,\dee{x}\\ 5A&=\int_0^5 f(x)\,\dee{x} \end{align*}

So, a rectangle with width 5 and height \(A\) has area \(\int_0^5 f(x)\,\dee{x}\text{.}\)

That is: if we replace \(f(x)\) with the constant function \(g(x)=A\text{,}\) then on the interval \([0,5]\text{,}\) the area under the curve is unchanged.
