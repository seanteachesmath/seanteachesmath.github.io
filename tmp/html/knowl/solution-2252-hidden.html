First observe that,

  • because \((x+y+1)^2\ge 0\text{,}\) all points on \((x+y+1)^2+z^2=4\) have \(|z|\le 2\) and that,
  • for \(|z_0|\le 2\text{,}\) the surface \((x+y+1)^2+z^2=4\) intersects the horizontal plane \(z=z_0\) on \((x+y+1)^2 = 4-z_0^2\text{,}\) i.e. on the two lines \(x+y=\pm\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\text{.}\)
  • The line \(x+y=\pm\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\) intersects the first octant if and only if \(z_0\ge0\) and \(\pm\sqrt{4-z_0^2}-1\ge 0\text{.}\)
  • Thus \(x+y=-\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\) never intersects the first octant and
  • \(x+y=\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\) intersects the first octant if and only if \(0\le z_0\le \sqrt{3}\text{.}\)
  • When \(z_0=0\text{,}\) the line \(x+y=\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\) is \(x+y=1\text{,}\) \(z=0\text{.}\)
  • When \(z_0=\sqrt{3}\text{,}\) the line \(x+y=\sqrt{4-z_0^2}-1\text{,}\) \(z=z_0\) is \(x+y=0\text{,}\) \(z=2\text{.}\)
  • So as \((x,y,z)\) runs over \(\cS\text{,}\) \((x,y)\) runs over the triangle \(x\ge0\text{,}\) \(y\ge 0\text{,}\) \(x+y\le 1\text{.}\)

Let \(G(x,y,z)=(x+y+1)^2+z^2\text{.}\) Then

\begin{equation*} \hn\,\dee{S}=\pm\frac{\vnabla G}{\vnabla G\cdot\hk}\ \dee{x}\dee{y} =\pm\frac{2(x+y+1)\hi+2(x+y+1)\hj+2z\hk}{2z}\,\dee{x}\dee{y} \end{equation*}

For the downward normal, we need the minus sign, so

\begin{align*} \vF\cdot\hn\,\dee{S} &=-\big[xy\,\hi+(z-xy)\,\hj\big]\cdot \Big[\frac{(x\!+\!y\!+\!1)\hi+(x\!+\!y\!+\!1)\hj+z\hk}{z}\Big]\,\dee{x}\dee{y}\\ &=-\frac{1}{z}\big[xy(x+y+1)+(z-xy)(x+y+1)\big]\,\dee{x}\dee{y}\\ &=-\frac{1}{z}\big[z(x+y+1)\big]\,\dee{x}\dee{y}\\ &=-(x+y+1)\,\dee{x}\dee{y} \end{align*}

The domain of integration is \(x\ge 0,\ y\ge 0,\ x+y\le 1\text{,}\) so

\begin{align*} \dblInt_\cS\vF\cdot\hn\,\dee{S} &=-\int_0^1 \dee{x}\int_0^{1-x} \dee{y} \ (x+y+1)\\ &=-\int_0^1 \dee{x}\ \Big[(1+x)(1-x)+\frac{1}{2}(1-x)^2\Big]\\ &=-\int_0^1 \dee{x}\ \Big[\frac{3}{2}-x-\frac{1}{2} x^2\Big] =-\Big[\frac{3}{2}-\frac{1}{2}-\frac{1}{6}\Big] =-\frac{5}{6} \end{align*}
