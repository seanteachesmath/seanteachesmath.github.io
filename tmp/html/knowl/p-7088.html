Solution: This integral is much more easily evaluated using the simple substitution \(u=r^2-x^2\text{.}\)

  • Set \(u=r^2-x^2\text{.}\) Then \(\dee{u}=-2x\dee{x}\text{,}\) and so
    \begin{align*} \int_a^r x\sqrt{r^2-x^2}\dee{x} &=\int_{r^2-a^2}^0 \sqrt{u}\ \frac{\dee{u}}{-2}\\ &=-\frac{1}{2}\bigg[\frac{u^{3/2}}{3/2}\bigg]_{r^2-a^2}^0\\ &=\frac{1}{3}\big[r^2-a^2\big]^{3/2} \end{align*}
