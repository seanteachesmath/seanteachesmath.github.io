(a) Parametrize the circle by \(x=a\cos\theta,\ y=a\sin\theta\text{,}\) \(0\le\theta\le 2\pi\text{.}\) Then \(\dee{x}=-a\sin\theta\,\dee{\theta}\) and \(\dee{y}=a\cos\theta\,\dee{\theta}\) so that

\begin{align*} \frac{1}{2\pi}\oint_C \frac{x\,\dee{y}-y\,\dee{x}}{x^2+y^2} &=\frac{1}{2\pi}\int_0^{2\pi} \frac{a^2\cos^2\theta\,\dee{\theta}+a^2\sin^2\theta\,\dee{\theta}} {a^2\cos^2\theta+a^2\sin^2\theta} =\frac{1}{2\pi}\int_0^{2\pi}\dee{\theta}\\ &=1 \end{align*}

(b) The boundary of the square has four sides â€” one with \(y=-1\text{,}\) one with \(x=1\text{,}\) one with \(y=1\) and one with \(x=-1\text{.}\)

To evaluate the integrals over the four sides

  • parametrize the \(y=-1\) part by \(x\) so that \(\vr(x) = x\,\hi-\hj\text{,}\) \(\vr'(x)=\hi\text{,}\) with \(x\) running from \(-1\) to \(1\text{,}\)
  • parametrize the \(x=+1\) part by \(y\) so that \(\vr(y) = \hi+y\,\hj\text{,}\) \(\vr'(y)=\hj\text{,}\) with \(y\) running from \(-1\) to \(1\text{,}\)
  • parametrize the \(y=+1\) part by \(x\) so that \(\vr(x) = x\,\hi+\hj\text{,}\) \(\vr'(x)=\hi\text{,}\) with \(x\) running from \(1\) to \(-1\text{,}\) and
  • parametrize the \(x=-1\) part by \(y\) so that \(\vr(y) = -\hi+y\,\hj\text{,}\) \(\vr'(y)=\hj\text{,}\) with \(y\) running from \(1\) to \(-1\text{,}\)

so that the integral

\begin{align*} \frac{1}{2\pi}\oint_C \frac{x\,\dee{y}-y\,\dee{x}}{x^2+y^2} &=\frac{1}{2\pi}\overbrace{\int_{-1}^1 \frac{-(-1)\dee{x}}{x^2+1}}^{y=-1{\rm\ part}} +\frac{1}{2\pi}\overbrace{\int_{-1}^1 \frac{(1)\dee{y}}{1+y^2}}^{x=+1{\rm\ part}}\\ &\hskip0.25in +\frac{1}{2\pi}\overbrace{\int^{-1}_1 \frac{-(1)\dee{x}}{x^2+1}}^{y=+1{\rm\ part}} +\frac{1}{2\pi}\overbrace{\int^{-1}_1 \frac{(-1)\dee{y}}{1+y^2}}^{x=-1{\rm\ part}}\\ &=4\frac{1}{2\pi}\arctan x\bigg|_{-1}^1 =\frac{2}{\pi}\Big[\frac{\pi}{4}+\frac{\pi}{4}\Big] =1 \end{align*}

(c) As in part (a) with \(a=\sqrt{2}\text{,}\) but with \(\theta\) running from \(0\) to \(\pi\text{,}\) the outer semicircle gives

\begin{equation*} \frac{1}{2\pi}\int_0^{\pi} \frac{a^2\cos^2\theta\,\dee{\theta}+a^2\sin^2\theta\,\dee{\theta}} {a^2\cos^2\theta+a^2\sin^2\theta} =\frac{1}{2\pi}\int_0^{\pi}\dee{\theta} =\frac{1}{2} \end{equation*}

As in part (a) with \(a=1\text{,}\) but with \(\theta\) running from \(\pi\) to \(0\text{,}\) the inner semicircle gives

\begin{equation*} \frac{1}{2\pi}\int^0_{\pi} \frac{a^2\cos^2\theta\,\dee{\theta}+a^2\sin^2\theta\,\dee{\theta}} {a^2\cos^2\theta+a^2\sin^2\theta} =\frac{1}{2\pi}\int^0_{\pi}\dee{\theta} =-\frac{1}{2} \end{equation*}

The two flat pieces each give zero, since on them \(y=0\) and \(\dee{y}=0\text{.}\) So

\begin{equation*} \frac{1}{2\pi}\oint_C \frac{x\,\dee{y}-y\,\dee{x}}{x^2+y^2} =\frac{1}{2}+0-\frac{1}{2}+0=0 \end{equation*}
