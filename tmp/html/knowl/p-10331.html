Set \(f(x)=\frac{e^{-\sqrt{x}}}{\sqrt{x}}\text{.}\) For \(x\ge1\text{,}\) this function is positive and decreasing (since it is the product of the two positive decreasing functions \(e^{-\sqrt x}\) and \(\frac1{\sqrt x}\)). We use the integral test with this function. Using the substitution \(u=\sqrt x\text{,}\) so that \(\dee{u} = \frac1{2\sqrt x}\,\dee{x}\text{,}\) we see that

\begin{align*} \int_1^\infty f(x)\,\dee{x} &=\lim_{R\rightarrow\infty}\int_1^R \bigg( \frac{e^{-\sqrt{x}}}{\sqrt{x}}\,\dee{x} \bigg)\\ &=\lim_{R\rightarrow\infty} \bigg( \int_1^{\sqrt{R}} e^{-u}\cdot2\,\dee{u} \bigg)\\ &=\lim_{R\rightarrow\infty} \bigg( {-}2e^{-u}\Big|_1^{\sqrt{R}} \bigg)\\ &=\lim_{R\rightarrow\infty} \bigg( {-}2e^{-\sqrt R} + 2e^{-\sqrt1} \bigg) = 0 + 2e^{-1}, \end{align*}

and so this improper integral converges. By the integral test, the given series also converges.
