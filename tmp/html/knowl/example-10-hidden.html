We have seen (back in Figure 1.6.1) that \(\sin(x)\) takes each value \(y\) between \(-1\) and \(+1\) for infinitely many different values of \(x\) (see the left-hand graph in the figure below). Consequently \(\sin(x)\text{,}\) with domain \(-\infty \lt x \lt \infty\) does not have an inverse function.

But notice that as \(x\) runs from \(-\frac{\pi}{2}\) to \(+\frac{\pi}{2}\text{,}\) \(\sin(x)\) increases from \(-1\) to \(+1\text{.}\) (See the middle graph in the figure above.) In particular, \(\sin(x)\) takes each value \(-1 \le y\le 1\) for exactly one \(-\frac{\pi}{2}\le x\le \frac{\pi}{2}\text{.}\) So if we restrict \(\sin x\) to have domain \(-\frac{\pi}{2}\le x\le \frac{\pi}{2}\text{,}\) it does have an inverse function, which is traditionally called arcsine (see Appendix [cross-reference to target(s) "sec_inv_trig" missing]).

That is, by definition, for each \(-1\le y\le 1\text{,}\) \(\arcsin(y)\) is the unique \(-\frac{\pi}{2}\le x\le \frac{\pi}{2}\) obeying \(\sin(x)=y\text{.}\) Equivalently, exchanging the dummy variables x and y throughout the last sentence gives that for each \(-1\le x\le 1\text{,}\) \(\arcsin(x)\) is the unique \(-\frac{\pi}{2}\le y\le \frac{\pi}{2}\) obeying \(\sin(y)=x\text{.}\)
