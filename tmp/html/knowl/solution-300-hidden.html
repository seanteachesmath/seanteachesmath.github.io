• Solution 1: Using the quotient rule,
    \begin{equation*} y'=\frac{x^3\frac{1}{x}-(\log x)\cdot 3x^2}{x^6}= \frac{x^2-3x^2\log x}{x^6}=\frac{1-3\log x}{x^4}. \end{equation*}
  • Solution 2: Using the product rule with \(y=\log x \cdot x^{-3}\text{,}\)
    \begin{equation*} y'=\frac{1}{x}x^{-3}+\log x \cdot(-3)x^{-4}=x^{-4}(1-3\log x) \end{equation*}
