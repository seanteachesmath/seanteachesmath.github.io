Solution 13.3.1.6.1.

If \(\varphi\) is a potential for \(\vF\text{,}\) then:

  • \(\pdiff{\varphi}{x}=x+y\text{,}\) so \(\varphi = \frac{1}{2}x^2+xy+\psi_1(y)\)
  • \(\pdiff{\varphi}{y}=x-y\text{,}\) so \(\varphi = xy-\frac{1}{2}y^2+\psi_2(x)\)

So, for instance, \(\varphi = \frac{1}{2}x^2+xy-\frac{1}{2}y^2\) is a potential for \(\vF\text{.}\)
