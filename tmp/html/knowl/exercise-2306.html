Exercise 9. M317 2007A.

Which of the following statements are true (T) and which are false (F)?

  1. If a smooth curve \(C\) is parameterized by \(\vr(s)\) where \(s\) is arc length, then the tangent vector \(\vr'(s)\) satisfies \(|\vr'(s)| = 1\text{.}\)
  2. If \(\vr(t)\) defines a smooth curve \(C\) in space that has constant curvature \(\kappa \gt 0\text{,}\) then \(C\) is part of a circle with radius \(1/\kappa\text{.}\)
  3. Suppose \(\vF\) is a continuous vector field with open domain \(D\text{.}\) If
    \begin{equation*} \int_C \vF \cdot \dee{\vr} = 0 \end{equation*}
    for every piecewise smooth closed curve \(C\) in \(D\text{,}\) then \(\vF\) is conservative.
  4. Suppose \(\vF\) is a vector field with open domain \(D\text{,}\) and the components of \(\vF\) have continuous partial derivatives. If \(\vnabla\times \vF= 0\) everywhere on \(D\text{,}\) then \(\vF\) is conservative.
  5. The curve defined by
    \begin{equation*} \vr_1(t) = \cos(t^2)\,\hi + \sin(t^2)\,\hj + 2t^2\,\hk,\qquad -\infty \lt t \lt \infty \end{equation*}
    is the same as the curve defined by
    \begin{equation*} \vr_2 (t) = \cos t\,\hi + \sin t\,\hj + 2t\,\hk,\qquad -\infty \lt t \lt \infty \end{equation*}
  6. The curve defined by
    \begin{equation*} \vr_1(t) = \cos(t^2)\,\hi + \sin(t^2)\,\hj + 2t^2\,\hk,\qquad 0\le t \le 1 \end{equation*}
    is the same as the curve defined by
    \begin{equation*} \vr_2 (t) = \cos t\,\hi + \sin t\,\hj + 2t\,\hk,\qquad 0\le t\le 1 \end{equation*}
  7. Suppose \(\vF(x, y, z)\) is a vector field whose components have continuous second order partial derivatives. Then \(\vnabla \cdot (\vnabla \times F) = 0\text{.}\)
  8. Suppose the real valued function \(f(x, y, z)\) has continuous second order partial derivatives. Then \(\vnabla\cdot(\vnabla f) = 0\text{.}\)
  9. The region \(D = \Set{ (x, y) }{ x^2 + y^2 \gt 1 }\) is simply connected.
  10. The region \(D = \Set{ (x, y) }{ y - x^2 \gt 0 }\) is simply connected.
