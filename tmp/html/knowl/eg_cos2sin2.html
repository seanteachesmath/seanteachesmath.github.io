Example 6.8.9. \(\int \cos^2x \sin^2x\dee{x}\).

Here we apply both 6.8.4 and 6.8.5.

\begin{align*} \int \cos^2x \sin^2x\dee{x} &= \frac{1}{4} \int \big[1+\cos(2x)\big] \big[1-\cos(2x)\big] \dee{x}\\ &= \frac{1}{4} \int \big[ 1-\cos^2(2x) \big] \dee{x}\\ \end{align*}

We can then apply 6.8.5 again

\begin{align*} &= \frac{1}{4} \int \big[ 1- \frac{1}{2}\left(1+ \cos(4x)\right) \big] \dee{x}\\ &= \frac{1}{8}\int \big[1 - \cos(4x)\big] \dee{x}\\ &= \frac{1}{8}x -\frac{1}{32}\sin(4x) +C \end{align*}

Oof! We could also have done this one using 6.8.2 to write the integrand as \(\sin^2(2x)\) and then used 6.8.4 to write it in terms of \(\cos(4x)\text{.}\)
