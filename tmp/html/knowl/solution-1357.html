Solution 8.5.3.5.1.

(a) We apply the ratio test for the series whose \(k^{\rm th}\) term is \(a_k= (-1)^k 2^{k+1} x^k\text{.}\) Then

\begin{align*} \lim_{k\to\infty} \bigg| \frac{ a_{k+1} }{ a_k } \bigg| &= \lim_{k\to\infty} \bigg| \frac{(-1)^{k+1} 2^{k+2} x^{k+1}} {(-1)^k 2^{k+1} x^k} \bigg|\\ &= \lim_{k\to\infty}|2x| = |2x| \end{align*}

Therefore, by the ratio test, the series converges for all \(x\) obeying \(|2x| \lt 1\text{,}\) i.e. \(|x| \lt \frac{1}{2}\text{,}\) and diverges for all \(x\) obeying \(|2x| \gt 1\text{,}\) i.e. \(|x| \gt \frac{1}{2}\text{.}\) So the radius of convergence is \(R = \frac{1}{2}\text{.}\)

Alternatively, one can set \(A_k = (-1)^k 2^{k+1}\) and compute

\begin{equation*} A = \lim_{k\to\infty} \bigg| \frac{A_{k+1}}{A_k}\bigg| = \lim_{k\to\infty} \bigg| \frac{(-1)^{k+1} 2^{k+2}}{(-1)^k 2^{k+1}}\bigg| = \lim_{k\to\infty} 2 =2 \end{equation*}

so that \(R=\frac{1}{A}=\frac{1}{2}\text{,}\) again.

(b) The series is

\begin{align*} \sum_{k=0}^\infty (-1)^k 2^{k+1} x^k \amp=2 \sum_{k=0}^\infty (-2x)^k =2\sum_{k=0}^\infty r^k\Big|_{r=-2x} =2\times\frac{1}{1-r}\\ \amp=\frac{2}{1+2x} \end{align*}

for all \(|r|=|2x| \lt 1\text{,}\) i.e. all \(|x| \lt \frac{1}{2}\text{.}\)
