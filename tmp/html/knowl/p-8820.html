Let \(s(t) = e^t\text{.}\)

  1. Find the average of \(s(t)\) on the interval \([0,1]\text{.}\) Call this quantity \(A\text{.}\)
  2. For any point \(t\text{,}\) the difference between \(s(t)\) and \(A\) is \(s(t)-A\text{.}\) Find the average value of \(s(t)-A\) on the interval \([0,1]\text{.}\)
  3. For any point \(t\text{,}\) the absolute difference between \(s(t)\) and \(A\) is \(|s(t)-A|\text{.}\) Find the average value of \(|s(t)-A|\) on the interval \([0,1]\text{.}\)
