(c) Again, we are using the parametrization

\begin{equation*} \vr(Y,\theta) = e^Y\sin\theta\,\hi + Y\,\hj +e^Y\cos\theta\,\hk\qquad 0\le Y\le 1,\ 0\le\theta\le 2\pi \end{equation*}

so that

\begin{equation*} \frac{\partial\vr}{\partial Y}\times\frac{\partial\vr}{\partial\theta} = \big(-e^Y\sin\theta, e^{2Y}, -e^Y\cos\theta\big), \end{equation*}

and, by (14.3.1),

\begin{gather*} \hn\,\dee{S} = \pm\frac{\partial\vr}{\partial Y}\times \frac{\partial\vr}{\partial\theta}\,\dee{Y}\dee{\theta} = \pm\big(-e^Y\sin\theta, e^{2Y}, -e^Y\cos\theta\big)\,\dee{Y}\,\dee{\theta} \end{gather*}

We choose the “\(+\)” sign so that \(\hn\) points towards the \(y\)-axis. As an example, when \(0\le\theta\le\frac{\pi}{2}\text{,}\) then \(z=e^Y\cos\theta \gt 0\) while the \(z\)-coordinate of \(\hn\) is \(-e^y\cos\theta \lt 0\text{.}\) So the integral is

\begin{align*} &\dblInt_S\vF\cdot\hn\,\dee{S}\\ &\hskip0.5in=\int_0^1\dee{Y}\int_0^{2\pi}\dee{\theta}\ \big(\overbrace{e^Y\sin\theta}^{x},0,\overbrace{e^Y\cos\theta}^{z}\big) \cdot\big(-e^Y\sin\theta, e^{2Y}, -e^Y\cos\theta\big)\\ &\hskip0.5in=-\int_0^1\dee{Y}\int_0^{2\pi}\dee{\theta}\ e^{2Y} =-2\pi\int_0^1\dee{Y}\ e^{2Y}\\ &\hskip0.5in=-\pi\big(e^2-1\big) =\pi\big(1-e^2\big) \end{align*}
