Alternatively, one can set \(A_k = (-1)^k 2^{k+1}\) and compute

\begin{equation*} A = \lim_{k\to\infty} \bigg| \frac{A_{k+1}}{A_k}\bigg| = \lim_{k\to\infty} \bigg| \frac{(-1)^{k+1} 2^{k+2}}{(-1)^k 2^{k+1}}\bigg| = \lim_{k\to\infty} 2 =2 \end{equation*}

so that \(R=\frac{1}{A}=\frac{1}{2}\text{,}\) again.
