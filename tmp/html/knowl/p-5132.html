We want to find the limit as \(n\) goes to infinity of the percentage error, \(\ds\lim_{n \rightarrow \infty} 100\frac{|S(n)-A(n)|}{|S(n)|}\text{.}\) Since \(A(n)\) is a nicer function than \(S(n)\text{,}\) let's simplify: \(\ds\lim_{n \rightarrow \infty} 100\frac{|S(n)-A(n)|}{|S(n)|} = 100\left|1-\ds\lim_{n \to \infty}\frac{A(n)}{S(n)}\right|\text{.}\)
