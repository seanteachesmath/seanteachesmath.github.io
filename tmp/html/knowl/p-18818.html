Find the correct identity, if \(f\) is a function and \(\vG\) and \(\vF\) are vector fields. Select the true statement.

  1. \(\displaystyle \vnabla\cdot(f \vF) = f \vnabla\times(\vF) + (\vnabla f ) \times\vF\)
  2. \(\displaystyle \vnabla\cdot(f \vF) = f \vnabla\cdot(\vF) + \vF\cdot \vnabla f\)
  3. \(\displaystyle \vnabla\times(f \vF) = f \vnabla\cdot(\vF) + \vF\cdot \vnabla f\)
  4. None of the above are true.
