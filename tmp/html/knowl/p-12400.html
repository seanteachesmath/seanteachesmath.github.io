(c) Switching to polar coordinates,

\begin{align*} \lim_{(x,y)\rightarrow(0,0)}\ \frac{x^2}{x^2+y^2} &=\lim_{\Atop{r\rightarrow0^+}{0\le\theta \lt 2\pi}}\ \frac{r^2\cos^2\theta}{r^2} =\lim_{\Atop{r\rightarrow0^+}{0\le\theta \lt 2\pi}}\ \cos^2\theta =\text{undefined} \end{align*}

since, for example,

  • if \(\theta=0\text{,}\) then
    \begin{equation*} \lim_{\Atop{r\rightarrow0^+}{\theta=0}}\ \cos^2\theta =\lim_{r\rightarrow0^+}\ 1 = 1 \end{equation*}
  • while if \(\theta=\frac{\pi}{2}\text{,}\) then
    \begin{equation*} \lim_{\Atop{r\rightarrow0^+}{\theta=\pi/2}}\ \cos^2\theta =\lim_{r\rightarrow0^+}\ 0 =0 \end{equation*}
