One example is \(f(x) = \left\{ \begin{array}{ll} 0&\mbox{when }0 \leq x \leq 1\\ 2&\mbox{when }1 \lt x \leq 2 \end{array}\right.\text{.}\) The IVT only guarantees \(f(c)=1\) for some \(c\) in \([0,2]\) when \(f\) is continuous over \([0,2]\text{.}\) If \(f\) is not continuous, the IVT says nothing.
