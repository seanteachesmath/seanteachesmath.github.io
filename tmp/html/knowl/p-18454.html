  1. If \(\cD\) has the property that
    \begin{equation*} \begin{split} &\text{every closed curve $\cC$ in $\cD$ is the boundary} \\ &\text{of a bounded oriented surface, $\cS$, in $\cD$} \end{split} \tag{H} \end{equation*}
    then
    \begin{equation*} \vF\text{ is conservative on $\cD$} \iff \vnabla\times \vF=0\text{ on $\cD$} \end{equation*}
  2. For any \(\cD\text{,}\) if \(\vnabla\times \vF=0\) on \(\cD\text{,}\) then \(\vF\) is locally conservative. This means that for each point \(\vx_0\in\cD\text{,}\) there is an \(\veps \gt 0\) and a function \(\varphi\) such that \(\vF=\vnabla\varphi\) on \(B_\veps(\vx_0)\text{.}\)
