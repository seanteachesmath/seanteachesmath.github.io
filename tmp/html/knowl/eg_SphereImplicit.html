Example 14.3.5. Area of a hemisphere — using an implicit equation.

This time we'll compute the area of the hemisphere by using that, if \((x,y,z)\) is on the hemisphere, then \(G(x,y,z) = a^2\) with \(G(x,y,z) = x^2 + y^2 + z^2\text{.}\) Since

\begin{equation*} \vnabla G(x,y,z) = \big(2x\,,\,2y\,,\,2z\big) \end{equation*}

14.3.3 yields

\begin{align*} \dee{S} &= \bigg|\frac{\vnabla G\big(x,y,z\big)}{\vnabla G\big(x,y,z\big)\cdot\hk}\bigg| \ \dee{x}\dee{y}\\ &= \bigg|\frac{\big(2x\,,\,2y\,,\,2z\big)}{2z}\bigg| \ \dee{x}\dee{y}\\ &= \frac{\sqrt{x^2+y^2+z^2}}{|z|} \ \dee{x}\dee{y}\\ &= \frac{a}{\sqrt{a^2-x^2-y^2}} \ \dee{x}\dee{y} \qquad\text{on } x^2+y^2+z^2=a^2 \end{align*}

So the area is \(\dblInt_{x^2+y^2\le a^2}\frac{a}{\sqrt{a^2-x^2-y^2}} \ \dee{x}\dee{y} \text{.}\) To evaluate this integral, we switch to polar coordinates, substituting \(x=r\cos\theta\text{,}\) \(y=r\sin\theta\text{.}\) This gives

\begin{align*} \text{area} &=\dblInt_{x^2+y^2\le a^2}\frac{a}{\sqrt{a^2-x^2-y^2}} \ \dee{x}\dee{y} =\int_0^a\dee{r}\ r\int_0^{2\pi}\dee{\theta}\ \frac{a}{\sqrt{a^2-r^2}}\\ &=2\pi a \int_0^a\dee{r}\ \frac{r}{\sqrt{a^2-r^2}} \end{align*}

We already showed, in Example 14.3.4, that the value of this integral is \(2\pi a^2\text{.}\)
