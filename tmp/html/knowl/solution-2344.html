Solution 15.3.1.14.1.

The given integral is of the form \(\int_C \vF\cdot\dee{\vr}\) with

\begin{equation*} \vF = \big(2xe^y + \sqrt{2} + x^2\big)\,\hi + x^2 \big(2 + e^y)\,\hj \end{equation*}

If we were to try to evaluate this integral directly, then on the \(y=x^2-4x+3\) part of \(C\text{,}\) the integrand would contain \(x^2 e^y = x^2 e^{x^2-4x+3}\text{.}\) That looks hard to integrate, so let's try Green's theorem. The parabolas \(y = x^2 - 4x + 3\) and \(y = 3 - x^2 + 2x\) intersect at \((x,y)\) with

\begin{align*} x^2 - 4x + 3 = 3 - x^2 + 2x &\iff 2x^2 -6x = 0 \iff 2x(x-3)=0\\ &\iff x=0\text{ or }x=3 \end{align*}

The curve \(C\) is the boundary of

\begin{equation*} R= \Set{(x,y)}{0\le x\le 3,\ x^2 - 4x + 3 \le y\le 3 - x^2 + 2x} \end{equation*}

It is sketched below.

By Green's theorem (Theorem 15.3.2),

\begin{align*} &\int_C \big(2xe^y + \sqrt{2} + x^2\big)\dee{x} + x^2 \big(2 + e^y)\dee{y}\\ &\hskip1in = \dblInt_R\Big[\pdiff{}{x}\big(x^2 \big(2 + e^y)\big) -\pdiff{}{y}\big(2xe^y + \sqrt{2} + x^2\big)\Big] \dee{x}\dee{y}\\ &\hskip1in = \dblInt_R \big(2x(2+e^y) - 2xe^y\big)\,\dee{x}\dee{y}\\ &\hskip1in =4 \int_0^3\dee{x}\int_{x^2 - 4x + 3}^{3 - x^2 + 2x}\dee{y}\ x\\ &=4 \int_0^3\dee{x}\ x\big[(3 - x^2 + 2x) - (x^2 - 4x + 3)\big]\\ &\hskip1in =4 \int_0^3\dee{x}\ \big(6x^2-2x^3\big) = 4\big(2\times 3^3-\frac{1}{2}3^4\big) =54 \end{align*}
