We want to add up all the tiny pieces of arclength \(\dee{s}\) along a curve \(C\text{.}\) So, the integral would simply be \(\int_C \dee{s}\text{.}\)

To see this another way, if we define \(\vr=(x(t),y(t),z(t))\) for \(a \le t \le b\) to be the equation of \(C\text{,}\) we could calculate the arclength as:

\begin{align*} \int_a^b |\vr'(t)|\dee{t}&=\int_a^b \sqrt{x'(t)^2+y'(t)^2+z'(t)^2}\dee{t} \end{align*}

This fits the form of DefinitionÂ 12.6.1 with \(f(x,y,z)=1\text{,}\) so we write it as a line integral as \(\int_C 1 \dee{s}\text{,}\) which is equivalent to \(\int_C \dee{s}\text{.}\)
