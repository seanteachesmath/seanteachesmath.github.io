Warning 13.4.9.

Note that in Theorem 13.4.8 we are assuming that \(\vF\) passes the screening test on all of \(\bbbr^2\) or \(\bbbr^3\text{.}\) We have already seen, in Example 13.3.14, that if the screening test fails at even a single point, for example because the vector field is not defined at that point, then \(\vF\) need not be conservative. We'll explore what happens in such cases in the (optional) §15.5. We'll see that something can be salvaged.
