So we just need to check if the denominator is zero. One of the facts that we should know  2  is that
\begin{gather*} -1 \leq \cos(x) \leq 1\\ \end{gather*}

and so by adding 2 we get

\begin{gather*} 1 \leq 2+\cos(x) \leq 3 \end{gather*}
Thus no matter what value of \(x\text{,}\) \(2+\cos(x) \geq 1\) and so cannot be zero.
