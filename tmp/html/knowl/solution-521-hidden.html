We need to know the first six derivatives of \(f(x)\) at \(x=1\text{.}\) Let's get started.

\begin{align*} f(x)&=x^2\log x + 2x^2+5 & \color{red}{f(1)}&\color{red}{=7}\\ f'(x)&=(x^2)\frac{1}{x}+(2x)\log x + 4x\\ &=2x\log x + 5x& \color{red}{f'(1)}&\color{red}{=5}\\ f''(x)&=(2x)\frac{1}{x}+(2)\log x + 5\\ &=2\log x + 7& \color{red}{f''(1)}&\color{red}{=7}\\ f'''(x)&=2x^{-1}& \color{red}{f'''(1)}&\color{red}{=2}\\ f^{(4)}&=-2x^{-2}& \color{red}{f^{(4)}(1)}&\color{red}{=-2}\\ f^{(5)}&=4x^{-3}& \color{red}{f^{(5)}(1)}&\color{red}{=4}\\ f^{(6)}&=-12x^{-4}& \color{red}{f^{(6)}(1)}&\color{red}{=-12} \end{align*}
