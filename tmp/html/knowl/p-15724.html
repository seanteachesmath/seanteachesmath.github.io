Consider the curve \(C\) given by

\begin{equation*} \vr(t) =\frac{t^3}{3}\,\hi + \frac{t^2}{\sqrt{2}}\,\hj + t\,\hk \qquad -\infty \lt t \lt \infty \end{equation*}
  1. Find the unit tangent \(\hat\vT(t)\) as a function of \(t\text{.}\)
  2. Find the curvature \(\kappa(t)\) as a function of \(t\text{.}\)
  3. Evaluate \(\kappa(t)\) at \(t = 0\text{.}\)
  4. Determine the principal normal vector \(\hat\vN(t)\) at \(t = 0\text{.}\)
  5. Compute the binormal vector \(\hat\vB(t)\) at \(t = 0\text{.}\)
