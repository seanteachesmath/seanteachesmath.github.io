Solution 6.8.4.17.1.
  • Solution 1: Since the power of tangent is odd, let's try to use the substitution \(u=\sec x\text{,}\) \(\dee{u} = \sec x \tan x \dee{x}\text{,}\) and \(\tan^2 x = \sec^2 x -1 = u^2-1\text{,}\) as in QuestionsÂ 14 and 15. In order to make this work, we need to see \(\sec x \tan x \dee{x}\) in the integrand, so we do a little algebraic manipulation.
    \begin{align*} \int \tan^3 x \sec^{-0.7}x \dee{x}&= \int \dfrac{\tan^3 x}{\sec^{0.7 x}} \dee{x} = \int \dfrac{\tan^3 x}{\sec^{1.7 x}}\sec x \dee{x}\\ &=\int \frac{\tan^2x}{\sec^{1.7}x}\cdot \sec x \tan x \dee{x}\\ &=\int \frac{u^2-1}{u^{1.7}} \dee{u} = \int \left(u^{0.3}-u^{-1.7}\right) \dee{u}\\ &=\frac{u^{1.3}}{1.3} + \frac{1}{0.7u^{0.7}}+C\\ &=\frac{1}{1.3}\sec^{1.3}x + \frac{1}{0.7\sec^{0.7}x}+C\\ &=\frac{1}{1.3}\sec^{1.3}x + \frac{1}{0.7}\cos^{0.7}x+C \end{align*}
  • Solution 2: Let's convert the secants and tangents to sines and cosines.
    \begin{align*} \int \tan^3 x \sec^{-0.7}x \dee{x}&= \int \frac{\sin^3 x}{\cos^3 x}\cdot \cos^{0.7}x \dee{x}\\ &=\int\frac{\sin^3 x}{\cos^{2.3}x} \dee{x}=\int \frac{\sin^2 x}{\cos^{2.3}x}\cdot\sin x \dee{x}\\ \end{align*}

    Using the substitution \(u=\cos x\text{,}\) \(\dee{u}=-\sin \dee{x}\text{,}\) and \(\sin^2 x = 1-\cos^2 x = 1-u^2\text{:}\)

    \begin{align*} & = -\int\frac{1-u^2}{u^{2.3}} \dee{u} = \int \left(-u^{-2.3}+u^{-0.3}\right) \dee{u}\\ &=\frac{1}{1.3}u^{-1.3} + \frac{1}{0.7}u^{0.7}+C\\ &=\dfrac{1}{1.3}\sec^{1.3}x + \dfrac{1}{0.7}\cos^{0.7}x+C \end{align*}
