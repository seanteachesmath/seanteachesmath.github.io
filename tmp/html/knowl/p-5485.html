  • To get the limits of summation to match the given sum, we need \(n=4\text{.}\)
  • Then to get the factor multiplying \(f\) to match that in the given sum, we need \(\frac{b-a}{n}=1\text{,}\) so \(b-a=4\text{.}\)
  • Finally, to get the argument of \(f\) to match that in the given sum, we need
    \begin{gather*} a+(k-1)\frac{b-a}{n}=a-\frac{b-a}{n} +k\frac{b-a}{n}=1+k \end{gather*}
    Subbing in \(n=4\) and \(b-a=4\) gives \(a-1 +k=1+k\text{,}\) so \(a=2\) and \(b=6\text{.}\)
