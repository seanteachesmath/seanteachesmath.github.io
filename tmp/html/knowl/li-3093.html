Item.
We should try to get some intuition about whether the integral converges or diverges. When \(t \to \infty\text{,}\) notice the integrand “looks like” the function \(\frac{1}{t^4}\text{.}\) We know \(\int_1^\infty \frac{1}{t^4} \dee{t}\) converges, because it's a \(p\)-integral with \(p=4 \gt 1\) (see Example 6.12.8). So, our integral probably converges as well. If we were only asked show it converges, we could use a comparison test, but we're asked more than that.
