The sequence has some positive terms and some negative terms, which limits the tests we can use. However, if we consider the series \(\displaystyle\sum_{n=1}^\infty \left|\dfrac{\sin n}{n^2}\right|\text{,}\) we can use the direct comparison test.

For every \(n\text{,}\) \(| \sin n| \lt 1\text{,}\) so \(0\le \left|\dfrac{\sin n}{n^2}\right| \lt \dfrac{1}{n^2}\text{.}\) Since \(\displaystyle\sum_{n=1}^\infty \dfrac{1}{n^2}\) converges, then by the direct comparison test, \(\displaystyle\sum_{n=1}^\infty \left|\dfrac{\sin n}{n^2}\right|\) converges as well.
