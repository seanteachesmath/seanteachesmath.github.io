(a) True. If the vector field is \(\vF = a\,\hi +b\,\hj + c\,\hk\text{,}\) then \(f(x,y,z) = ax + by + cz\) obeys \(\vF=\vnabla f\) and so is a potential for \(\vF\text{.}\)

(b) False. For example the vector field \(\vF= x\,\hi-y\,\hj\) obeys \(\vnabla\cdot\vF= 0\) but is not a constant vector field.

(c) True, assuming that \(\vr(t)\) is not indentically \(\vZero\text{.}\) If \(\vr(t)\) and \(\diff{\vr}{t}\) are orthogonal at all points of the curve \(C\text{,}\) then

\begin{equation*} \diff{ }{t} \big(\vr(t)\cdot\vr(t)\big) = 2\vr(t)\cdot\diff{\vr}{t}(t) =0 \end{equation*}

So \(x(t)^2 + y(t)^2 + z(t)^2 = \vr(t)\cdot\vr(t)\) is a constant. If \(\vr(t)\) is not indentically \(\vZero\text{,}\) that constant must be strictly positive. That is \(x(t)^2 + y(t)^2 + z(t)^2 = a^2\) for some constant \(a \gt 0\text{.}\)

(d) False. The curvature (see ยง12.5) is

\begin{equation*} \ka(t)= \frac{\Big|\diff{\hat\vT}{t}(t)\Big|}{|\vr'(t)|} \end{equation*}

Changing the orientation of the curve amounts to replacing \(t\) by \(-t\text{.}\) This changes the signs of \(\hat\vT\) and \(\vr'\text{,}\) but does not change \(\ka\text{,}\) because the absolute values eliminate the signs.

(e) False. For example, the vector field \(\vF=\vZero\text{,}\) with domain \(\Set{(x,y,z)}{x^2+y^2 \gt 0}\) is a conservative vector field (with potential \(0\)) whose domain is not simply connected. As a less nitpicky example, let \(\vF=\vnabla f\) with \(f=\frac{1}{x^2+y^2}\text{.}\) The biggest possible domain for this vector field is also \(\Set{(x,y,z)}{x^2+y^2 \gt 0}\text{.}\)
