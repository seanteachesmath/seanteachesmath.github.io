Theorem 6.12.20.

Let \(a\) and \(c\) be real numbers with \(a \lt c\) and let the function \(f(x)\) be continuous for all \(x\ge a\text{.}\) Then the improper integral \(\int_a^\infty f(x)\ \dee{x}\) converges if and only if the improper integral \(\int_c^\infty f(x)\ \dee{x}\) converges.
