Exercise Group 13.4.2.25–31.

Questions  13.4.2.25 and 13.4.2.26 ask you to evaluate line integrals of vector fields that are not conservative, but that can be expressed as a sum of a conservative vector field and another vector field that can be written concisely.

25. M317 2011A.

Let

\begin{equation*} \vF = \big(y^2 e^{3z} +Axy^3\big)\,\hi +(2xye^{3z}+3x^2y^2\big)\,\hj +Bxy^2e^{3z}\,\hk \end{equation*}
  1. Find all values of \(A\) and \(B\) for which the vector field \(\vF\) is conservative.
  2. If \(A\) and \(B\) have values found in (a), find a potential function for \(\vF\text{.}\)
  3. Let \(C\) be the curve with parametrization \(\vr(t) = e^{2t}\,\hi + e^{-t}\,\hj + \ln(1 + t) \,\hk\) from \((1, 1, 0)\) to \(\big(e^2\,,\,\frac{1}{e}\,,\,\ln 2\big)\text{.}\) Evaluate
    \begin{equation*} \int_C (y^2 e^{3z} + xy^3)\,\dee{x} + (2xye^{3z} + 3x^2 y^2 )\,\dee{y} + 3xy^2 e^{3z}\,\dee{z}. \end{equation*}
