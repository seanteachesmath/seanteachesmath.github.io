(a) By the vector identity of Theorem 15.1.7.a,

\begin{equation*} \nabla\cdot \vF=\nabla\cdot \nabla\times\vG=0 \end{equation*}

So we must have

\begin{align*} 0&=\nabla\cdot\vF=\nabla\cdot \big((xz+xy)\hi+\alpha(yz-xy)\hj+\beta(yz+xz)\hk\big)\\ &=(z+y)+\alpha(z-x)+\beta(y+x) \end{align*}

This is true for all \((x,y,z)\) if and only if \(\alpha=\beta=-1\text{.}\)
