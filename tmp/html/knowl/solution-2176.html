Solution 13.4.2.37.2.

There's nothing magical about the form of the path from Solution 1. It's just a path that's relatively easy to describe using one constant \(Y\text{.}\) To emphasize this point, we provide a solution with an alternate path based on an ellipse.

A partial ellipse running from \((0,0)\) to \((2,2)\) can be described by \(\vr(t)=(\cos t+1\, , \, A\sin t)\) for a constant \(A\text{,}\) with \(t\) running from \(\pi\) to 0. (To find this: we centre a circle of radius 1 at the point \((1,0)\text{,}\) then multiply its \(y\)-coordinate by \(A\text{.}\))

In this case, \(\vF(\vr(t))=(2A\sin t + 2,0)\) and \(\vr'(t)=(-\sin t , A\cos t)\text{,}\) so

\begin{align*} \vF(\vr(t))\cdot\vr'(t)&=-\sin t (2A\sin 2 +2)=-A(2\sin^2t)-2\sin t\\ &=-A(1-\cos2t)-2\sin t\\ \int \vF \cdot \dee\vr&=\int_{\pi}^0\big( A(\cos2t-1)-2\sin t \big)\dee{t}\\ &=\left[A\left(\frac12\sin(2t)-t\right) +2\cos t\right]_{\pi}^0\\ &=A\pi+4 \end{align*}

Setting \(A\pi+4=8\text{,}\) we find \(A=\frac{4}{\pi}\text{.}\) So, the half-ellipse \(\vr(t)=\left(\cos t+1\, , \, \frac{4}{\pi}\sin t\right)\text{,}\) with \(t\) running from \(\pi\) to 0, is another path that gives \(\int_C \vF\cdot\dee\vr=8\text{.}\)
