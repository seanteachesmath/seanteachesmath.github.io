Solution 3.3.3.4.1.
\begin{align*} C'(w)\amp=\lim_{h \rightarrow 0} \dfrac{C(w+h)-C(w)}{h} \approx \dfrac{C(w+1)-C(w)}{1}\\ \amp=C(w+1)-C(w) \end{align*}

which is the number of calories in \(C(w+1)\) grams minus the number of calories in \(C(w)\) grams. This is the number of calories per gram.
