Concavity:

Since we're given \(f'(x)\text{,}\) we can find \(f''(x)\text{.}\)

\begin{align*} f''(x)&=K(2-2x-2x+x^2)e^{-x}\\ &=K(2-4x+x^2)e^{-x}\\ &=K\big(x-2-\sqrt{2}\big)\big(x-2+\sqrt{2}\big)e^{-x} \end{align*}

where the last line can be found using the quadratic equation. So, \(f''(x)=0\) for \(x=2\pm\sqrt{2}\text{,}\) and \(f''(x)\) exists everywhere.

\(x\) \((-\infty,2-\sqrt{2})\) \(2-\sqrt{2}\) \((2-\sqrt{2},2+\sqrt{2})\) \(2+\sqrt 2\) \((2+\sqrt 2,\infty)\)
\(f'(x)\) positive 0 negative 0 positive
\(f(x)\) concave up IP concave down IP concave up
