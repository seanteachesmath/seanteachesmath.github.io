Solution 8.3.11.21.1.

This is a geometric series with \(r=\frac{-1}{5}\text{.}\) Since \(|r| \lt 1\text{,}\) it is convergent.

We want to use the formula \(\sum_{n=0}^\infty r^n=\frac{1}{1-r}\text{,}\) but our series does not start at 0, so we re-write it:

\begin{align*} \sum_{n=3}^\infty \left(\frac{-1}{5}\right)^n&= \sum_{n=0}^\infty \left(\frac{-1}{5}\right)^n- \sum_{n=0}^2 \left(\frac{-1}{5}\right)^n\\ \amp=\frac{1}{1-(-1/5)} - \left(1-\frac{1}{5}+\frac{1}{25} \right)\\ &=\frac{1}{6/5}-1+\frac{1}{5}-\frac{1}{25} =\frac{5}{6}+\frac{-25+5-1}{25}\\ \amp=-\frac{1}{150} \end{align*}
