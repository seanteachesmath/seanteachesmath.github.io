Any first order linear initial value problem

\begin{equation*} \diff{}{s}\vx(s) = M(s) \vx(s)\qquad \vx(0)=\vx_0 \end{equation*}

where \(\vx\) is an \(n\)-component vector and \(M(s)\) is an \(n\times n\) matrix with continuous entries, has exactly one solution. If \(n=1\text{,}\) so that \(\vx(s)\) and \(M(s)\) are just functions, this is easy to see. Just let \(\cM(s)\) be the antiderivative of \(M(s)\) that obeys \(\cM(0)=0\text{.}\) Then

\begin{align*} \diff{}{s}\vx(s) = M(s) \vx(s) &\iff e^{-\cM(s)}\diff{}{s}\vx(s) - M(s) e^{-\cM(s)} \vx(s)=0\\ &\iff \diff{}{s}\Big(e^{-\cM(s)}\vx(s)\Big) = 0 \end{align*}

by the product rule. So \(e^{-\cM(s)}\vx(s)\) is a constant independent of \(s\text{.}\) In particular \(e^{-\cM(s)}\vx(s)=e^{-\cM(0)}\vx(0)= \vx_0\) so that \(\vx(s) = \vx_0 e^{\cM(s)}\text{.}\) This argument can be generalized to any natural number \(n\text{.}\) But that is beyond the scope of this book.
