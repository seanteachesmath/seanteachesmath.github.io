If \(\varphi\) is a potential for \(\vF\text{,}\) then:

  • \(\pdiff{\varphi}{x}=\frac{x}{x^2+y^2+z^2}\text{,}\) so \(\varphi = \frac12\ln(x^2+y^2+z^2)+\psi_1(y,z)\)
  • \(\pdiff{\varphi}{y}=\frac{y}{x^2+y^2+z^2}\text{,}\) so \(\varphi = \frac12\ln(x^2+y^2+z^2)+\psi_2(x,z)\)
  • \(\pdiff{\varphi}{z}=\frac{z}{x^2+y^2+z^2}\text{,}\) so \(\varphi = \frac12\ln(x^2+y^2+z^2)+\psi_2(x,y)\)

So, for instance, \(\varphi= \frac{1}{2}\ln(x^2+y^2+z^2)\) is a potential for \(\vF\text{.}\)
