In contrast to QuestionÂ 9.3.1.1, the sign on \(\mathbf c\) does generally matter. \(\mathbf c\) is required to be a point on the line, but except in particular circumstances, there's no reason to believe that \(-\mathbf c =-\mathbf c +t\mathbf d\big|_{t=0}\) is a point on the line. Indeed \(-\mathbf c\) is on the line if and only if there is a \(t\) with \(\mathbf c + t\mathbf d = -\mathbf c\text{,}\) i.e. \(t\mathbf d = -2\mathbf c\text{.}\) That is the case if and only if \(\mathbf d\) is parallel to \(\mathbf c\text{.}\) So, only the first equation is correct in general.
