Solution 6.13.1.18.1.

(a) Substituting \(u=x^3+1\text{,}\) \(\dee{u}=3x^2\,\dee{x}\)

\begin{align*} \int\frac{x^2}{(x^3 + 1)^{101}}\dee{x} \amp=\int\frac{1}{u^{101}}\cdot \frac{\dee{u}}{3} =\frac{u^{-100}}{-100}\cdot\frac{1}{3}+C\\ \amp=-\frac{1}{300(x^3+1)^{100}} + C \end{align*}

(b) Substituting \(u=\sin x\text{,}\) \(\dee{u}=\cos x\ \dee{x}\text{,}\) \(\cos^2 x = 1-\sin^2 x = 1-u^2\text{,}\)

\begin{align*} \int \cos^3\!x\ \sin^4\!x\ \dee{x} &=\int \cos^2\!x\ \sin^4\!x\ \cos x\ \dee{x} =\int (1-u^2) u^4\ \dee{u}\\ &=\int (u^4-u^6)\ \dee{u} =\frac{u^5}{5}-\frac{u^7}{7}+C\\ &=\frac{\sin^5\!x}{5}-\frac{\sin^7\!x}{7}+C \end{align*}
