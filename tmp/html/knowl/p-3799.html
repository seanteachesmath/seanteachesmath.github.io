Since \(T_3(x)\) is the third-degree Taylor polynomial for \(f(x)\) about \(x=a\text{,}\) we know the following things to be true:

  • \(\displaystyle f(a)=T_3(a)\)
  • \(\displaystyle f'(a)=T'_3(a)\)
  • \(\displaystyle f''(a)=T''_3(a)\)
  • \(\displaystyle f'''(a)=T'''_3(a)\)

But, some of these don't look super useful. For instance, if we try to use the first bullet, we get this equation:

\begin{equation*} a^3\left[2\log a - \frac{11}{3}\right]=-\frac{2}{3}\sqrt{e^3}+3ea-6\sqrt{e}a^2+a^3 \end{equation*}
