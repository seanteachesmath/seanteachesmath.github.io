Exercise 16. M317 2016D.

Are each of the following statements True or False? Recall that \(f \in C^k\) means that all derivatives of \(f\) up to order \(k\) exist and are continuous.

  1. \(\vnabla\times(f \vnabla f ) = \vZero\) for all \(C^2\) scalar functions \(f\) in \(\bbbr^3\text{.}\)
  2. \(\vnabla\cdot(f\vF) = \vnabla f \cdot\vF + f\vnabla\cdot \vF \) for all \(C^1\) scalar functions \(f\) and \(C^1\) vector fields \(\vF\) in \(\bbbr^3\text{.}\)
  3. A smooth space curve \(C\) with constant curvature \(\kappa = 0\) must be a part of a straight line.
  4. A smooth space curve \(C\) with constant curvature \(\kappa \ne 0\) must be part of a circle of radius \(1/\kappa\text{.}\)
  5. If \(f\) is any smooth function defined in \(\bbbr^3\) and if \(C\) is any circle, then \(\int_C\vnabla f\cdot\dee{\vr}=0\text{.}\)
  6. Suppose \(\vF\) is a smooth vector field in \(\bbbr^3\) and \(\vnabla\cdot\vF=0\) everywhere. Then, for every sphere, the flux out of one hemisphere is equal to the flux into the opposite hemisphere.
  7. Let \(\vF(x, y,z)\) be a continuously differentiable vector field which is defined for every \((x, y, z)\text{.}\) Then, \(\dblInt_S\vnabla\times\vF\cdot\hn\,\dee{S}=0\) for any closed surface \(S\text{.}\) (A closed surface is a surface that is the boundary of a solid region.)
