Solution 14.2.1.13.1.

A normal vector to the surface \(x^2 + 9y^2 + 4z^2 = 17\) at the point \((x,y,z)\) is \(( 2x\,,\, 18y\,,\,8z)\text{.}\) A normal vector to the plane \(x - 8z = 0\) is \(( 1\,,\,0\,,\,-8)\text{.}\) So we want \(( 2x\,,\, 18y\,,\,8z)\) to be parallel to \(( 1\,,\,0\,,\,-8)\text{,}\) i.e. to be a nonzero constant times \(( 1\,,\,0\,,\,-8)\text{.}\) This is the case whenever \(y=0\) and \(z=-2x\) with \(x\ne 0\text{.}\) In addition, we want \((x,y,z)\) to lie on the surface \(x^2 + 9y^2 + 4z^2 = 17\text{.}\) So we want \(y=0\text{,}\) \(z=-2x\) and

\begin{gather*} 17= x^2 + 9y^2 + 4z^2 =x^2 +4(-2x)^2=17x^2 \implies x=\pm 1 \end{gather*}

So the allowed points are \(\pm(1,0,-2)\text{.}\)
