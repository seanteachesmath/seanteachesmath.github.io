Example 6.4.13. \(\int_0^1\frac{1}{(2x+1)^3}\dee{x}\).

Compute \(\int_0^1\frac{1}{(2x+1)^3}\dee{x}\text{.}\)

We could do this one using Theorem 6.4.10, but its not too hard to do without. We can think of the integrand as the function “one over a cube” with the argument \(2x+1\text{.}\) So it makes sense to substitute \(u=2x+1\text{.}\) That is

  • set \(u=2x+1\) and
  • replace \(\dee{x} \rightarrow \frac{1}{u'(x)}\dee{u} = \frac{1}{2}\dee{u}\text{.}\)
  • When \(x=0\text{,}\) we have \(u=2\times 0+1=1\) and
  • when \(x=1\text{,}\) we have \(u=2\times 1+1=3\text{.}\)

So

\begin{align*} \int_0^1\frac{1}{(2x+1)^3}\dee{x} &=\int_1^{3} \frac{1}{u^3} \cdot \frac{1}{2}\dee{u}\\ &=\frac{1}{2}\int_1^{3}u^{-3}\dee{u}\\ &=\frac{1}{2}\left[\frac{u^{-2}}{-2}\right]_1^{3}\\ &=\frac{1}{2}\left( \frac{1}{-2}\cdot \frac{1}{9} - \frac{1}{-2}\cdot\frac{1}{1} \right)\\ &=\frac{1}{2}\left( \frac{1}{2} - \frac{1}{18}\right) = \frac{1}{2}\cdot \frac{8}{18}\\ &=\frac{2}{9} \end{align*}
