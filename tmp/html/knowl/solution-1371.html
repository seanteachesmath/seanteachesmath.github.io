Solution 8.5.3.19.1.

(a) Applying \(\displaystyle\frac{1}{1+r}=\sum\limits_{n=0}^\infty (-1)^nr^n\) with \(r=x^3\) gives

\begin{gather*} \int\frac{1}{1+x^3}\ \dee{x} =\sum_{n=0}^\infty(-1)^n \int x^{3n}\ \dee{x} =\sum_{n=0}^\infty(-1)^n \frac{x^{3n+1}}{3n+1} +C \end{gather*}

(b) By part (a),

\begin{gather*} \int_0^{1/4}\frac{1}{1+x^3}\ \dee{x} =\sum_{n=0}^\infty(-1)^n \frac{x^{3n+1}}{3n+1} \bigg|_0^{1/4} =\sum_{n=0}^\infty(-1)^n \frac{1}{(3n+1)4^{3n+1}} \end{gather*}

This is an alternating series with successively smaller terms that converge to zero as \(n\rightarrow\infty\text{.}\) So truncating it introduces an error no larger than the magnitude of the first dropped term. We want that first dropped term to obey

\begin{gather*} \frac{1}{(3n+1)4^{3n+1}} \lt 10^{-5}=\frac{1}{10^5} \end{gather*}

So let's check the first few terms.

\begin{align*} \frac{1}{(3n+1)4^{3n+1}}\bigg|_{n=0}&=\frac{1}{4} \gt \frac{1}{10^5}\\ \frac{1}{(3n+1)4^{3n+1}}\bigg|_{n=1}&=\frac{1}{4^5} \gt \frac{1}{10^5}\\ \frac{1}{(3n+1)4^{3n+1}}\bigg|_{n=2}&=\frac{1}{7\times 4^7} =\frac{1}{7\times 2^{14}} =\frac{1}{7\times16\times 1024}\\ \amp=\frac{1}{112\times 1024} \lt \frac{1}{10^5} \end{align*}

So we need to keep two terms (the \(n=0\) and \(n=1\) terms).
