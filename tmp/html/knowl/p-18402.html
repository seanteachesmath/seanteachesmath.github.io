(a) By the vector identity \(\vnabla\cdot\big(\vnabla\times\vG\big)=0\) (Theorem 15.1.7.a). So we must have

\begin{align*} 0&=\vnabla\cdot\vF =\frac{\partial }{\partial x}\big(xz\big) + \frac{\partial }{\partial y}\big(axe^y z + byz\big) + \frac{\partial }{\partial z}\big(y^2 - xe^y z^2 \big)\\ &= z + \big(axe^y z+bz\big) + \big(-2xe^yz\big)\\ &= (1+b)z + (a-2)x e^y z \end{align*}

So we need \(a=2\) and \(b=-1\text{.}\)
