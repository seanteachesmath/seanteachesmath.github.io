• First we find the \(x\)-coordinates where \(y=1\text{.}\)
    \begin{align*} x^2\tan\left(\frac{\pi}{4}\right)+2x\log(1) &= 16\\ x^2\cdot 1 +2x\cdot 0 &=16\\ x^2 &= 16 \end{align*}
    So \(x=\pm 4\text{.}\)
  • Now we use implicit differentiation to get \(y'\) in terms of \(x,y\text{:}\)
    \begin{align*} x^2\tan(\pi y/4)+2x\log(y) &= 16\\ 2x\tan(\pi y/4) + x^2 \frac{\pi}{4}\sec^2(\pi y/4)\cdot y' + 2\log(y) + \frac{2x}{y} \cdot y' &= 0\,. \end{align*}
  • Now set \(y=1\) and use \(\tan(\pi/4)=1\,, \sec(\pi/4)=\sqrt{2}\) to get
    \begin{align*} 2x\tan(\pi/4) + x^2 \frac{\pi}{4} \sec^2(\pi/4)y' + 2\log(1) + 2x\cdot y' &= 0\\ 2x + \frac{\pi}{2} x^2 y' +2x y' &= 0\\ y' = -\frac{2x}{x^2 \pi/2 + 2x} &= -\frac{4}{\pi x + 4} \end{align*}
  • So at \((x,y)=(4,1)\) we have \(y' = -\dfrac{4}{4\pi+4} = -\dfrac{1}{\pi + 1}\)
  • and at \((x,y)=(-4,1)\) we have \(y' = \dfrac{1}{\pi-1}\)
