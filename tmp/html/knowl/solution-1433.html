Solution 9.1.1.1.1.
  1. The point \((x,y,z)\) satisfies \(x^2 +y^2+z^2= 2x-4y+4\) if and only if it satisfies \(x^2-2x +y^2+4y+z^2= 4\text{,}\) or equivalently \((x-1)^2 +(y+2)^2+z^2=9 \text{.}\) Since \(\sqrt{(x-1)^2 +(y+2)^2+z^2}\) is the distance from \((1, -2, 0)\) to \((x,y,z)\text{,}\) our point satisfies the given equation if and only if its distance from \((1,-2,0)\) is three. So the set is the sphere of radius 3 centered on \((1,-2,0)\text{.}\)
  2. As in part (a), \(x^2 +y^2+z^2 \lt 2x-4y+4\) if and only if \((x-1)^2 +(y+2)^2+z^2 \lt 9 \text{.}\) Hence our point satifies the given inequality if and only if its distance from \((1,-2,0)\) is strictly smaller than three. The set is the interior of the sphere of radius 3 centered on \((1,-2,0)\text{.}\)
