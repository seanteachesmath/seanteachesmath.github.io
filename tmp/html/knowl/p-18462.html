We'll start by looking at one particular vector field, which passes the screening test, but which cannot possibly be conservative. The field, which we saw in Example 13.3.14, is

\begin{equation*} \vTh =-\frac{y}{x^2+y^2}\hi + \frac{x}{x^2+y^2}\hj \end{equation*}

with domain of definition \(\cD\text{.}\) We'll first check that it passes the screening test:

\begin{align*} \vnabla\times\vTh &=\Big\{\frac{\partial }{\partial x}\Big(\frac{x}{x^2+y^2}\Big) -\frac{\partial }{\partial y}\Big(-\frac{y}{x^2+y^2}\Big)\Big\}\hk\\ &=\Big\{\Big(\frac{1}{x^2+y^2}-\frac{2x^2}{{(x^2+y^2)}^2}\Big) +\Big(\frac{1}{x^2+y^2}-\frac{2y^2}{{(x^2+y^2)}^2}\Big)\Big\}\hk\\ &=0 \end{align*}

Next we'll check that it cannot be conservative. Denote by \(\cC_\veps\) the circle \(x^2+y^2=\veps^2\text{,}\) with counterclockwise orientation. Parametrize \(\cC_\veps\) by \(\vr(\theta)=\veps\cos\theta\,\hi+\veps\sin\theta\,\hj\) with \(0\le\theta\le2\pi\text{.}\) Then

\begin{align*} &\int_{\cC_\veps}\vTh\cdot \dee{\vr} =\int_0^{2\pi} \vTh\big(\vr(\theta)\big)\cdot\frac{\dee{\vr}}{d\theta}(\theta)\ d\theta\\ &\hskip0.25in=\int_0^{2\pi} \Big(-\frac{1}{\veps}\sin\theta\,\hi +\frac{1}{\veps}\cos\theta\,\hj\Big) \cdot\big(-\veps\sin\theta\,\hi+\veps\cos\theta\,\hj\big)\ d\theta \tag{E1}\\ &\hskip0.25in=\int_0^{2\pi} \!\! d\theta\\ &\hskip0.25in=2\pi \end{align*}

is not zero. By Theorem 15.5.3, \(\vTh\) cannot be conservative on the punctured disk since the integral \(\int_{\cC_\veps}\vTh\cdot \dee{\vr}\) around the closed curve \(\cC_\veps\) is nonzero.
