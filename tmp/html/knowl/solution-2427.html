Solution 16.2.21.1.

(a) True\(\,\text{.}\) Since \(v=|\vv|=1\) is constant, we have

\begin{equation*} \va = \diff{v}{t}\hat\vT + v^2\kappa\hat\vN = 0\hat\vT + \kappa\hat\vN. \end{equation*}

Thus \(1 = |\va| = \kappa|\hat\vN|\text{,}\) i.e., \(\kappa=1\text{.}\)

(a) (Again.) Since \(\vv\cdot\vv=|\vv|^2=1\) for all \(t\text{,}\) differentiation gives \(\vv\cdot\va=0\text{,}\) i.e., \(\vv\perp\va\) always. It follows that \(|\vv\times\va| = |\vv|\,|\va|\sin\theta = 1\) always, because the angle \(\theta\) here is always \(\pi/2\text{.}\) Thus, for all \(t\text{,}\)

\begin{equation*} \kappa = { |\vv\times\va|\over |\vv|^3} = {1\over 1} = 1. \end{equation*}

(b) True\(\,\text{.}\) By the divergence theorem, if \(V\) is the solid bounded by \(S\text{,}\)

\begin{equation*} \dblInt_{S}\vnabla\times\vF\cdot\hn\,\dee{S} =\tripInt_V\vnabla\cdot\big(\vnabla\times\vF\big)\ \dee{V} =0 \end{equation*}

since \(\vnabla\cdot\big(\vnabla\times\vF\big)=0\text{.}\)

(c) False\(\,\text{.}\) If \(\vF=0\) and \(\vG\) is any nonzero, conservative field, like \(\vG=2x\hi=\vnabla(x^2)\text{,}\) then

\begin{equation*} \oint_C \vF\cdot \dee{\vr}=\oint_C \vG\cdot \dee{\vr}=0 \end{equation*}

for every closed curve \(C\text{.}\)
