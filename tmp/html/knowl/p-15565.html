If, on the other hand, \(\hat\vB(s)\) is not constant, then our curve doesn't lie in a single plane, and we can use the derivative

\begin{align*} \diff{\hat\vB}{s} &=\diff{}{s}\big(\hat\vT\times\hat\vN\big) =\diff{\hat\vT}{s}\times\hat\vN + \hat\vT\times \diff{\hat\vN}{s}\\ &=\hat\vT\times \diff{\hat\vN}{s}\qquad \Big( \text{since } \diff{\hat\vT}{s} \text{ is parallel to } \hat\vN \Big) \end{align*}

as a measure

  • of how badly the curve fails to lie in a plane,
  • i.e. how much the plane that fits the curve best near \(\vr(s)\) twists as \(s\) increases,

The cross product in \(\diff{\hat\vB}{s}=\hat\vT\times \diff{\hat\vN}{s}\) implies that \(\diff{\hat\vB}{s}\) is perpendicular to \(\hat\vT\text{.}\) In addition, \(\diff{\hat\vB}{s}\) must be perpendicular to \(\hat\vB\) because

\begin{equation*} |\hat\vB|=1 \implies 1=\hat\vB\cdot\hat\vB \implies 0 = \diff{}{s}\left[\hat\vB\cdot\hat\vB\right] = 2 \hat\vB\cdot\diff{\hat\vB}{s} \end{equation*}

So \(\diff{\hat\vB}{s}(s)\) must be parallel to \(\hat\vN(s)\text{.}\)
