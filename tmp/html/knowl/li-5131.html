Method 3: The vector from \((0,2)\) to \((1,0)\) is \(\llt 1-0\,,\,0-2\rgt = \llt 1,-2\rgt\text{.}\) As we see from the figure above, it is a direction vector for the line. One point on the line is \((0,2)\text{.}\) So a parametric equation for the line (see Equation 9.3.1) is
\begin{equation*} \llt x-0\,,\,y-2\rgt = t\llt 1,-2\rgt\qquad\text{or}\qquad x=t,\ y=2-2t \end{equation*}
