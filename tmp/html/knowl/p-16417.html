Set \(\varphi(x,y)= \arctan\frac{y}{x}\) (using the standard \(\arctan\) that takes values between \(-\frac{\pi}{2}\) and \(\frac{\pi}{2}\)). Note that \(\varphi(x,y)\) is well-defined, with all partial derivatives continuous, on \(D\) since \(x \gt 1\) there. Then

\begin{alignat*}{2} \pdiff{\varphi}{x}(x,y) &= \frac{-\frac{y}{x^2}}{1+\big(\frac{y}{x}\big)^2} &&= -\frac{y}{x^2+y^2}\\ \pdiff{\varphi}{y}(x,y) &= \frac{\frac{1}{x}}{1+\big(\frac{y}{x}\big)^2} &&= \phantom{-} \frac{x}{x^2+y^2} \end{alignat*}

so that \(\vF=\vnabla\varphi\text{.}\)
