Solution 7.2.2.16.1.
  1. We can see without calculation that the average will be zero, since \(f(x)=x\) is an odd function and \([-3,3]\) is a symmetric interval. Alternately, we can use the definition of an average to calculate
    \begin{equation*} \frac{1}{6}\int_{-3}^3 x\,\dee{x} = \left[\frac{1}{12}x^2\right]_{-3}^3 = \frac{1}{12}(9-9)=0 \end{equation*}
  2. Using the definition provided for root mean square:
    \begin{align*} \text{RMS}&=\sqrt{\frac{1}{6}\int_{-3}^3 x^2\,\dee{x}} = \sqrt{\left[\frac{1}{18}x^3\right]_{-3}^3} = \sqrt{\frac{27}{18}-\frac{-27}{18}} = \sqrt{3} \end{align*}
