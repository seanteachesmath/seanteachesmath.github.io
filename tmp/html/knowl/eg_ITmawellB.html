Example 15.1.9. Screening tests.

We have seen the vector identity Theorem 15.1.7.b before. It says that if a vector field \(\vF\) is of the form \(\vF = \vnabla\varphi\) for some some function \(\varphi\) (that is, if \(\vF\) is conservative), then

\begin{equation*} \vnabla\times\vF = \vnabla\times(\vnabla\varphi) = 0 \end{equation*}

Conversely, we have also seen, in Theorem 13.4.8, that, if \(\vF\) is defined and has continuous first order partial derivatives on all of \(\bbbr^3\text{,}\) and if \(\vnabla\times\vF=0\text{,}\) then \(\vF\) is conservative. The vector identity Theorem 15.1.7.b is our screening test for conservativeness.

Because its right hand side is zero, the vector identity Theorem 15.1.7.a is suggestive. It says that if a vector field \(\vF\) is of the form \(\vF = \vnabla\times\vA\) for some some vector field \(\vA\text{,}\) then

\begin{equation*} \vnabla\cdot\vF = \vnabla\cdot(\vnabla\times\vA) = 0 \end{equation*}

When \(\vF = \vnabla\times\vA\text{,}\) \(\vA\) is called a vector potential for \(\vF\text{.}\) We shall see in Theorem 15.1.16, below, that, conversely, if \(\vF(\vx)\) is defined and has continuous first order partial derivatives on all of \(\bbbr^3\text{,}\) and if \(\vnabla\cdot\vF=0\text{,}\) then \(\vF\) has a vector potential 8 . The vector identity Theorem 15.1.7.a is indeed another screening test.

As an example, consider the Maxwell's equations

\begin{align*} \vnabla\cdot\vB &= 0\\ \vnabla\times\vE +\frac{1}{c}\frac{\partial\vB}{\partial t}&=0 \end{align*}

that we saw in Example 15.1.2. The first equation implies that (assuming \(\vB\) is sufficiently smooth) there is a vector field \(\vA\text{,}\) called the magnetic potential, with \(\vB=\vnabla\times\vA\text{.}\) Substituting this into the second equation gives

\begin{equation*} 0= \vnabla\times\vE +\frac{1}{c}\frac{\partial\ }{\partial t}\vnabla\times\vA =\vnabla\times\Big(\vE+\frac{1}{c}\frac{\partial\vA }{\partial t}\Big) \end{equation*}

So \(\vE+\frac{1}{c}\frac{\partial\vA }{\partial t}\) passes the screening test of Theorem 15.1.7.b and there is a function \(\varphi\) (called the electric potential) with

\begin{equation*} \vE+\frac{1}{c}\frac{\partial\vA }{\partial t} = -\nabla \varphi \end{equation*}

We have put in the minus sign just to provide compatibility with the usual physics terminology.
