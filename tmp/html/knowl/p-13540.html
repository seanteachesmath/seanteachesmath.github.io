(b) As \(f_x(x,y)=-4x(y^2-x^2)\) and \(f_y(x,y)=4y(y^2-x^2)\text{,}\) we have \(f_x(0,0) = f_y(0,0) = 0\) so that \((0,0)\) is a critical point. Note that

  • \(f(0,0)=0\text{,}\)
  • \(f(x,y)\ge 0\) for all \(x\) and \(y\text{.}\)

So \((0,0)\) is a local (and also absolute) minimum.
