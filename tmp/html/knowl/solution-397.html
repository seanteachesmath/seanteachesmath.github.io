Solution 3.14.2.8.1.

The question asks for \(s''(1)\text{.}\) We start our differentiation using the quotient rule:

\begin{align*} s'(t)&=\frac{e^t(t^2+1)-e^t(2t)}{(t^2+1)^2}\\ &=\frac{e^t(t^2-2t+1)}{(t^2+1)^2} \end{align*}

Using the quotient rule again,

\begin{align*} \amp s''(t)=\frac{(t^2+1)^2{\textcolor{blue}{\diff{}{t}\{e^t(t^2-2t+1)\}}}-e^t(t^2-2t+1) {\textcolor{red}{\diff{}{t}\{(t^2+1)^2\}}}}{(t^2+1)^4}\\ &=\frac{(t^2\!+\!1)^2\cdot\left[{\textcolor{blue}{e^t(2t\!-\!2)+e^t(t^2\!-\!2\!+\!1)}}\right]- e^t(t^2\!-\!2t\!+\!1) \cdot {\textcolor{red}{2(t^2\!+\!1)(2t)}}}{(t^2\!+\!1)^4}\\ &=\frac{e^t(t^2+1)^2(t^2-1)-4te^t(t-1)^2(t^2+1)}{(t^2+1)^4} \end{align*}

so that

\begin{align*} s''(1)&=0 \end{align*}
