(a) True. In general \(|\vr'(t)| = \diff{s}{t}\text{.}\) When \(t=s\text{,}\) \(\diff{s}{t}=1\text{.}\)

(b) False. The curve need not even lie in a plane. For example, as we saw in Example 12.4.4, the helix \(\vr(t) = a\cos t\,\hi +b\sin t\,\hj + bt\,\hk\) has constant curvature \(\ka=\frac{a}{a^2+b^2}\) but is not a circle.

(c) True. See Theorem 13.4.7.

(d) False. The vector field \(\vF(x, y, z) = \frac{-y}{x^2+y^2} \hi + \frac{x}{x^2+y^2} \hj \text{,}\) with domain

\begin{equation*} \Set{(x,y,z)}{ x^2+y^2 \gt 1} \end{equation*}

provides a counterexample.

(e) False. The curve \(\vr_1\) contains only points with \(z\ge0\text{.}\) Every \(\vr_2(t)\) with \(t \lt 0\) has \(z \lt 0\text{.}\)

(f) True. \(\vr_2(t^2)=\vr_1(t)\) and \(t^2\) runs from \(0\) to \(1\) as \(t\) runs from \(0\) to \(1\text{.}\)

(g) True. \(\vnabla\cdot(\vnabla\times \vF)=0\) by the vector identity of Theorem 15.1.7.a.

(h) False. A counterexample is \(f(x,y,z) = x^2\text{.}\) It has \(\vnabla f = 2 x\,\hi\) and hence \(\vnabla\cdot(\vnabla f) = 2\text{.}\)

(i) False. The curve \(x^2+y^2 = 2\) can not be shrunk to a point continuously in \(\Set{ (x, y) }{ x^2 + y^2 \gt 1 }\text{.}\)

(j) True. Any curve in \(\Set{ (x, y) }{ y \gt x^2 }\) can be shrunk to a point continuously in \(\Set{ (x, y) }{ y \gt x^2 }\text{.}\)
