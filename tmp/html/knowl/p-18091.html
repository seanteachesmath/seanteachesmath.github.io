If we use \(D\) to denote the disk inside the circle \(C\) then we want

\begin{align*} \oint_C\vF\cdot \dee{\vr}-A\oint_C\vG\cdot \dee{\vr} &=\oint_C(\vF-A\vG)\cdot \dee{\vr}\\ &=\dblInt_D \Big[\pdiff{}{x}(\vF-A\vG)_2 -\pdiff{}{y}(\vF-A\vG)_1\Big]\,\dee{x}\dee{y} \end{align*}

to vanish for all disks \(D\text{.}\) We used Green's theorem, which is Theorem 15.3.2, in the last step. This is the case if and only if

\begin{align*} &\pdiff{}{x}(\vF-A\vG)_2 =\pdiff{}{y}(\vF-A\vG)_1\\ \iff& \pdiff{}{x}[(x+y)-A(2x-3y)] =\pdiff{}{y}[(x+3y)-A(x+y)]\\ \iff& 1-2A=3-A\\ \iff& A=-2 \end{align*}
