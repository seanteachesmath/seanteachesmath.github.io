So we would write the integral as

\begin{align*} \int_{-\infty}^\infty\frac{\dee{x}}{(x-2)x^2} &=\int_{-\infty}^{a} \frac{\dee{x}}{(x-2)x^2} +\int_{a}^0 \frac{\dee{x}}{(x-2)x^2} +\int_0^b \frac{\dee{x}}{(x-2)x^2}\\ &+\int_b^2 \frac{\dee{x}}{(x-2)x^2} +\int_2^c \frac{\dee{x}}{(x-2)x^2} +\int_c^\infty \frac{\dee{x}}{(x-2)x^2} \end{align*}

where

  • \(a\) is any number strictly less than \(0\text{,}\)
  • \(b\) is any number strictly between \(0\) and \(2\text{,}\) and
  • \(c\) is any number strictly bigger than \(2\text{.}\)

So, for example, take \(a=-1, b=1, c=3\text{.}\)
