(b) If \(\tau(s)\equiv 0\text{,}\) then \(\diff{\hB}{s}=-\tau(s)\hN(s)\equiv 0\) so that \(\hB\) is a constant. As \(\hT(s)\perp\hB\text{,}\)

\begin{equation*} \diff{}{s} (\vr(s)-\vr(0))\cdot\hB =\hT(s)\cdot\hB=0 \end{equation*}

and \((\vr(s)-\vr(0))\cdot\hB\) must be a constant. The constant must be zero (set \(s=0\)), so \((\vr(s)-\vr(0))\cdot\hB=0\) and \(\vr(s)\) always lies in the plane through \(\vr(0)\) with normal vector \(\hB\text{.}\)
