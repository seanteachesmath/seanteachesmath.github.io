Let's start by taking some derivatives. Of course, since we're differentiating sine, the derivatives will repeat every four iterations.

\begin{align*} f(x)&=\sin x & f\left(\frac{\pi}{4}\right)&=\frac{1}{\sqrt 2}\\ f'(x)&=\cos x & f'\left(\frac{\pi}{4}\right)&=\frac{1}{\sqrt{2}}\\ f''(x)&=-\sin x & f''\left(\frac{\pi}{4}\right)&=-\frac{1}{\sqrt 2}\\ f'''(x)&=-\cos x & f'''\left(\frac{\pi}{4}\right)&=-\frac{1}{\sqrt{2}} \end{align*}

So, the pattern of derivatives is \(\dfrac{1}{\sqrt{2}}\text{,}\) \(\dfrac{1}{\sqrt{2}}\text{,}\) \(-\dfrac{1}{\sqrt{2}}\text{,}\) \(-\dfrac{1}{\sqrt{2}}\text{,}\) \(\dfrac{1}{\sqrt{2}}\text{,}\) \(\dfrac{1}{\sqrt{2}}\text{,}\) \(-\dfrac{1}{\sqrt{2}}\text{,}\) \(-\dfrac{1}{\sqrt{2}}\text{,}\) etc. This is a little tricky to write in sigma notation. We can deal with the “doubles” by pairing consecutive even and odd terms, as shown below.

\begin{align*} T_{2n}(x)&=\sum_{k=0}^{2n} \frac{f^{(k)}\left(\frac{\pi}{4}\right)}{k!}\left(x-\frac{\pi}{4}\right)^k\\ &=\sum_{k=0}^{n} \left[ \underbrace{\frac{f^{(2k)}\left(\frac{\pi}{4}\right)}{(2k)!}\left(x-\frac{\pi}{4}\right)^{2k}}_{\mbox{even term}} + \underbrace{\frac{f^{(2k+1)}\left(\frac{\pi}{4}\right)}{(2k+1)!}\left(x-\frac{\pi}{4}\right)^{2k+1}}_{\mbox{odd term}}\right]\\ &=\sum_{k=0}^n \left[\frac{(-1)^k}{(2k)!\sqrt{2}}\left(x-\frac{\pi}{4}\right)^{2k} + \frac{(-1)^k}{(2k+1)!\sqrt{2}}\left(x-\frac{\pi}{4}\right)^{2k+1}\right] \end{align*}

To verify that the first sum really is the same as the second sum, write out the first few terms of each explicitly.
