Because its right hand side is zero, the vector identity Theorem 15.1.7.a is suggestive. It says that if a vector field \(\vF\) is of the form \(\vF = \vnabla\times\vA\) for some some vector field \(\vA\text{,}\) then

\begin{equation*} \vnabla\cdot\vF = \vnabla\cdot(\vnabla\times\vA) = 0 \end{equation*}

When \(\vF = \vnabla\times\vA\text{,}\) \(\vA\) is called a vector potential for \(\vF\text{.}\) We shall see in Theorem 15.1.16, below, that, conversely, if \(\vF(\vx)\) is defined and has continuous first order partial derivatives on all of \(\bbbr^3\text{,}\) and if \(\vnabla\cdot\vF=0\text{,}\) then \(\vF\) has a vector potential 8 . The vector identity Theorem 15.1.7.a is indeed another screening test.
