(b) The inverse function of \(s(t)=\sqrt{2} e^t\) is \(t(s) =\ln\left(\frac{s}{\sqrt{2}}\right)\) with \(s \gt 0\text{.}\) (As \(t\rightarrow-\infty\text{,}\) the arc length \(s\rightarrow 0\) and as \(t\rightarrow+\infty\text{,}\) the arc length \(s\rightarrow +\infty\text{.}\)) So the reparametrization is

\begin{align*} &\vR(s) = e^t (\cos t, \sin t)\Big|_{t=\ln\left(\frac{s}{\sqrt{2}}\right)} =\frac{s}{\sqrt{2}}\left(\cos\Big(\ln\Big(\frac{s}{\sqrt{2}}\Big)\Big)\,,\, \sin\Big(\ln\Big(\frac{s}{\sqrt{2}}\Big)\Big)\right)\\ &\quad\text{with } s \gt 0 \end{align*}
