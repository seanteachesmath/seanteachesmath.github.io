Solution 6.2.3.19.1.

Let \(x\) be any real number.

  • \(f(x)=f(-x)\) (since \(f(x)\) is even), and
  • \(f(x)=-f(-x)\) (since \(f(x)\) is odd).
  • So, \(f(x)=-f(x)\text{.}\)
  • Then (adding \(f(x)\) to both sides) we see \(2f(x)=0\text{,}\) so \(f(x)=0\text{.}\)

So, \(f(x)=0\) for every \(x\text{.}\)
