Let \(C\) be the parameterized curve given by

\begin{equation*} \vr(t) = \big(\cos t, \sin t, t\big),\qquad 0 \le t \le \frac{\pi}{2} \end{equation*}

and let

\begin{equation*} \vF = \big(e^{yz}\,,\, xze^{yz} + ze^y\,,\, xye^{yz} + e^y\big) \end{equation*}
  1. Compute and simplify \(\vnabla\times\vF\text{.}\)
  2. Compute the work integral \(\int_C\vF\cdot\dee{\vr}\text{.}\)
