Definition 9.2.7. Length of a Vector.
\begin{alignat*}{3} &\va=\llt a_1,a_2\rgt \qquad&&\implies\qquad &&|\va|=\sqrt{a_1^2+a_2^2}\\ &\va=\llt a_1,a_2,a_3\rgt \qquad&&\implies\qquad &&|\va|=\sqrt{a_1^2+a_2^2+a_3^2} \end{alignat*}

A unit vector is a vector of length one. We'll sometimes use the accent \(\hat{\ }\) to emphasise that the vector \(\hat\va\) is a unit vector. That is, \(|\hat\va|=1\text{.}\)
