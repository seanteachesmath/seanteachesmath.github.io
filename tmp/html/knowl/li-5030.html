Item.
A vector \(\llt a,b\rgt\) is perpendicular to \(\big \lt 1\,,\,-\frac{1}{c}\big \gt \) if and only if
\begin{align*} \llt a,b\rgt \cdot \llt 1\,,\,-\frac{1}{c}\rgt =0 &\iff a-\frac{b}{c} = 0 \iff b=ac \iff \llt a,b\rgt =a \llt 1,c\rgt \end{align*}
That is, a vector is perpendicular to \(\big \lt 1\,,\,-\frac{1}{c}\big \gt \) if and only if it is parallel to \(\llt 1,c\rgt\text{.}\)
