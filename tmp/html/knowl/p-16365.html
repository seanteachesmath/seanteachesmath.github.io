As always, we have to be careful with the flow of logic 3 . The screening test in Theorem 13.3.9 is a one-way test. If, for example, \(\frac{\partial F_1}{\partial y} \ne \frac{\partial F_2}{\partial x}\) then the vector field \(\vF\) cannot be conservative. But if \(\frac{\partial F_1}{\partial y} = \frac{\partial F_2}{\partial x}\) Theorem 13.3.9 does not guarantee that \(\vF\) is conservative. In fact there are fields that are not conservative but do obey \(\frac{\partial F_1}{\partial y}=\frac{\partial F_2}{\partial x}\text{.}\) We'll see one in Example 13.3.14, below. We shall later find some additional regularity conditions which, when combined with \(\frac{\partial F_1}{\partial y}=\frac{\partial F_2}{\partial x}\text{,}\) do imply conservativeness. See Theorem 13.4.8, below.
