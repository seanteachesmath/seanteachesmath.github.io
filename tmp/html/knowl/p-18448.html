The closed rectangle \(\cC=\big\{\ (x,y)\in\bbbr^2\ \big|\ 0\le x \le 1,\ 0 \le y \le 1\ \big\}\) is not an open subset of \(\bbbr^2\text{.}\) For example, \(\vZero=(0,0)\) is a point in \(\cC\text{.}\) No matter what \(\veps \gt 0\) we pick, the open ball \(B_\veps(\vZero)\) is not contained in \(\cC\) because \(B_\veps(\vZero)\) contains the point \((-\frac{\veps}{2},0)\text{,}\) which is not in \(\cC\text{.}\)
