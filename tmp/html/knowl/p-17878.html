We will be using the divergence theorem in both parts (a) and \(b\text{.}\) So as a prelimary calculation, let's find the divergence of \(\vH(x,y,z)=\frac{(x,y,z)-(a,b,c)}{{[(x-a)^2+(y-b)^2+(z-b)^2]}^{3/2}}\) for any \((a,b,c)\text{.}\) If \((x,y,z)\ne(a,b,c)\text{,}\)

\begin{align*} &\vnabla\cdot\vH(x,y,z)\\ &\hskip0.25in =\pdiff{}{x}\frac{x-a} {{[(x-a)^2+(y-b)^2+(z-c)^2]}^{3/2}}\\ &\hskip1.5in +\pdiff{}{y}\frac{y-b} {{[(x-a)^2+(y-b)^2+(z-c)^2]}^{3/2}}\\ &\hskip1.5in +\pdiff{}{z}\frac{z-c} {{[(x-a)^2+(y-b)^2+(z-c)^2]}^{3/2}}\\ &\hskip0.25in =\frac{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]\ -\ (x-a)\ \frac{3}{2}(2(x-a))} {{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]}^{5/2}}\\ &\hskip1.5in +\frac{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]\ -\ (y-b)\ \frac{3}{2}(2y)} {{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]}^{5/2}}\\ &\hskip1.5in +\frac{\big[(x-a)^2+(y-b)^2+z^2\big]\ -\ (z-c)\ \frac{3}{2}(2(z-c))} {{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]}^{5/2}}\\ &\hskip0.25in =\frac{3\big[(x\!-\!a)^2+(y\!-\!b)^2+(z\!-\!c)^2\big]\ -\ 3(x\!-\!a)^2-3(y\!-\!b)^2-3(z\!-\!c)^2} {{\big[(x-a)^2+(y-b)^2+(z-c)^2\big]}^{5/2}}\\ &\hskip0.25in=0 \end{align*}

If \((x,y,z)=(a,b,c)\text{,}\) \(\vH(x,y,z)\) is not defined and hence \(\vnabla\cdot\vH(x,y,z)\) is also not defined.
