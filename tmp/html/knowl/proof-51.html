Proof

The vector from the origin to the point whose polar coordinates are \((r,\theta)\) is \(\vr = r\,\hat\vr(\theta)\text{.}\) So if we parametrize a curve by giving the polar coordinates at time \(t\text{,}\)

\begin{align*} \vr(t) &= r(t)\ \hat\vr\big(\theta(t)\big)\\ \vv(t) &= \diff{r}{t}(t)\ \hat\vr\big(\theta(t)\big) + r(t)\ \diff{\hat\vr}{\theta}\big(\theta(t)\big)\ \diff{\theta}{t}(t) \notag\\ &= \diff{r}{t}(t)\ \hat\vr\big(\theta(t)\big) + r(t)\ \diff{\theta}{t}(t)\ \hat\vth\big(\theta(t)\big)\\ \va(t) & = \difftwo{r}{t}\ \hat\vr + \diff{r}{t}\ \diff{\hat\vr}{\theta}\ \diff{\theta}{t} + \diff{r}{t}\ \diff{\theta}{t}\ \hat\vth + r\ \difftwo{\theta}{t}\ \hat\vth + r\ \Big(\diff{\theta}{t}\Big)^2\ \diff{\hat\vth}{\theta}\\ &=\Big[\difftwo{r}{t}-r\ \Big(\diff{\theta}{t}\Big)^2\Big] \hat\vr +\Big[r\ \difftwo{\theta}{t} + 2 \diff{r}{t}\ \diff{\theta}{t}\Big]\hat\vth \end{align*}
