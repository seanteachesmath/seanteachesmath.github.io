(c) Yes. Under this parametrization, \(x=\sqrt{1+2v^2}\cos u\text{,}\) \(y=\sqrt{1+2v^2}\sin u\) and \(z=v\text{.}\) So

  • \(x^2+y^2-2z^2 = 1+2v^2 -2v^2 = 1\text{,}\) as desired.
  • The condition \(x^2+y^2\le 9\) is equivalent to \(1+2v^2\le 9\text{,}\) which is equivalent to \(v\le 2\text{,}\) since \(v\ge 0\text{.}\)
  • The condition \(x^2+y^2\ge 3\) is equivalent to \(1+2v^2\ge 3\text{,}\) which is equivalent to \(v\ge 1\text{,}\) since \(v\ge 0\text{.}\)
  • \(\displaystyle z=v\ge 0\)
