(a) The curve \(x^2+y^2=1\) is a circle of radius \(1\text{.}\) So we can parametrize it by \(x(\theta)=\cos\theta\text{,}\) \(y(\theta)=\sin\theta\text{,}\) \(0\le\theta \lt 2\pi\text{.}\) The \(z\)-coordinate of any point on the intersection is determined by \(z=x^2-y^2\text{.}\) So we can use the parametrization

\begin{align*} \vr(\theta) &= \cos\theta\,\hi +\sin\theta\,\hj + [\cos^2\theta-\sin^2\theta]\,\hk\\ &=\cos\theta\,\hi +\sin\theta\,\hj + \cos(2\theta)\,\hk \qquad 0\le\theta \lt 2\pi \end{align*}
