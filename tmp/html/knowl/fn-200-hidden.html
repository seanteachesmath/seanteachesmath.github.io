The MVT tells us that there is a number \(c\) between \(x\) and \(x+h\) so that
\begin{equation*} F'(c) = \frac{F(x+h)-F(x)}{(x+h)-x} = \frac{F(x+h)-F(x)}{h} \end{equation*}
But since \(F'(x) = f(x)\text{,}\) this tells us that
\begin{equation*} \frac{F(x+h)-F(x)}{h} = f(c) \end{equation*}
where \(c\) is trapped between \(x+h\) and \(x\text{.}\) Now when we take the limit as \(h \to 0\) we have that this number \(c\) is squeezed to \(x\) and the result follows.
