Solution 6.4.2.16.1.
  • The slightly sneaky method: We note that \(\displaystyle\diff{}{x} \left\{e^{x^2} \right\}= 2x\, e^{x^2}\text{,}\) so that \(\dfrac{1}{2} e^{x^2}\) is a antiderivative for the integrand \(x e^{x^2}\text{.}\) So
    \begin{gather*} \int_{-2}^2 xe^{x^2}\,dx = \bigg[\frac{1}{2}e^{x^2}\bigg]_{-2}^2 =\frac{1}{2}e^4-\frac{1}{2}e^4=0 \end{gather*}
  • The really sneaky method: The integrand \(f(x) = x e^{x^2}\) is an odd function (meaning that \(f(-x)=-f(x)\)). So by Theorem 6.2.12 every integral of the form \(\int_{-a}^a x e^{x^2}\,\dee{x}\) is zero.
