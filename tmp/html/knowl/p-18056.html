To complete our computation, we have to determine \(\int_{C_\rho} \vF\cdot\dee{\vr}\text{.}\) We can do so by repeating the same “removing a small disk containing \((0,0)\)” argument for the third time. Set

\begin{equation*} R_5 = \Set{(x,y)}{x^2 + y^2 \le 1,\ x^2+y^2\ge \rho^2} \end{equation*}

Then the boundary of \(R_5\) consists of \(C_1\) and \(-C_\rho\text{,}\) and, as \(Q_x-P_y\) is well-defined and zero throughout \(R_5\text{,}\)

\begin{align*} 0 &=\dblInt_{R_5} \big(Q_x-P_y\big)\,\dee{x}\dee{y}\\ &= \int_{C_1} \vF\cdot\dee{\vr} + \int_{-C_\rho} \vF\cdot\dee{\vr}\\ &= \pi - \int_{C_\rho} \vF\cdot\dee{\vr} \end{align*}

So \(\int_{C_4} \vF\cdot\dee{\vr} = \int_{C_\rho} \vF\cdot\dee{\vr}=\pi\text{.}\)
