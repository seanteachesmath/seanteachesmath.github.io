The bird follows the parametrized path

\begin{equation*} \vr(u)=\left(\frac{4\sqrt{2}}{3}u^{3/2},\frac{4\sqrt{2}}{3}u^{3/2}, u(2-u)\right) \end{equation*}

This is the same path as the plane, but the parameter \(u\) is not time. Let's denote by \(\vR(t)\) the position of the bird at time \(t\text{.}\) At time \(t\) the bird is at some point on the parametrized path, so there is some \(u(t)\) with

\begin{equation*} \vR(t) = \vr\big(u(t)\big) \end{equation*}
