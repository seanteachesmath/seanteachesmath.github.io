• Solution 1: Let's use the substitution \(u=\sec x\text{,}\) \(\dee{u}=\sec x \tan x\dee{x}\text{.}\) In order to make this work, we need to see \(\sec x \tan x\) in the integrand, so we start with some algebraic manipulation.
    \begin{align*} \int \tan x \sqrt{\sec x}\left(\frac{\sqrt{\sec x}}{\sqrt{\sec x}}\right) \dee{x}&=\int \frac{1}{\sqrt{\sec x}}\sec x\tan x\dee{x}\\ &=\int \frac{1}{\sqrt{u}}\dee{u}=2\sqrt{u}+C\\ &=2\sqrt{\sec x}+C \end{align*}
  • Solution 2: Let's turn our secants and tangents into sines and cosines.
    \begin{align*} \int \tan x \sqrt{\sec x}\dee{x}&=\int \frac{\sin x}{\cos x\cdot\sqrt{\cos x}}\dee{x}=\int \frac{\sin x}{\cos^{1.5}x}\dee{x}\\ \end{align*}

    We use the substitution \(u=\cos x\text{,}\) \(\dee{u}=-\sin x\dee{x}\text{.}\)

    \begin{align*} &=\int -u^{-1.5}\dee{u}=\frac{2}{\sqrt{u}}+C\\ &=2\sqrt{\sec x}+C \end{align*}
