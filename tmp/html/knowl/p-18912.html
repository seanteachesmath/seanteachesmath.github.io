Say whether each of the following statements is true or false and explain why.

  1. A moving particle has velocity and acceleration vectors that satisfy \(|\vv| = 1\) and \(|\va|=1\) at all times. Then the curvature of this particle's path is a constant.
  2. If \(\vF\) is any smooth vector field defined in \(\bbbr^3\) and if \(S\) is any sphere, then
    \begin{equation*} \dblInt_{S}\vnabla\times\vF\cdot\hn\,\dee{S}=0 \end{equation*}
    Here \(\hn\) is the outward normal to \(S\text{.}\)
  3. If \(\vF\) and \(\vG\) are smooth vector fields in \(\bbbr^3\) and if \(\ds\oint_C \vF\cdot \dee{\vr}=\oint_C \vG\cdot \dee{\vr}\) for every circle \(C\text{,}\) then \(\vF=\vG\text{.}\)
