(a) Given \(s(\phi)\text{,}\) then

\begin{equation*} \rho = \Big|\diff{s}{\phi}\Big|\qquad \ka = \Big|\diff{s}{\phi}\Big|^{-1} \end{equation*}

As we are assuming that \(0 \lt \rho=\Big|\diff{s}{\phi}\Big| \lt \infty\text{,}\) the inverse function theorem says that we can invert the function \(s(\phi)\) (at least locally) to get \(\phi\) as a function of \(s\text{,}\) and that

\begin{equation*} \ka = \Big|\diff{\phi}{s}\Big| \end{equation*}
