This time we will find the stream lines for the vortex field, \(\vv(x,y) = \Om\big(-y\hi +x\hj\big)\) of Example 13.1.4, by using 13.2.3, which is

\begin{align*} \diff{x}{t} &= -\Om y\\ \diff{y}{t} &= \Om x \end{align*}

We can convert this system of first order linear ordinary differential equations into a single second order linear constant coefficient differential equation 8 , by differentiating the first equation, to get \(\difftwo{x}{t} = -\Om\diff{y}{t}\text{,}\) and then substituting in the second equation to get

\begin{equation*} \difftwo{x}{t} + \Om^2 x = 0 \end{equation*}

This equation is a special case of the ordinary differential equation treated in Example [cross-reference to target(s) "eg_RLC" missing] of the Appendix [cross-reference to target(s) "ap_ODE" missing], entitled “Review of Linear Ordinary Differential Equations”. \begin{equation*} x(t) = A\cos(\Om t-\theta) \end{equation*}

with \(A\) and \(\theta\) being arbitrary constants 9 . Then

\begin{equation*} y(t) = -\frac{1}{\Om}\diff{x}{t} =A\sin(\Om t-\theta) \end{equation*}
