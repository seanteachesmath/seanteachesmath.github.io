Exercise 5.

Suppose a function \(y=f(x)\) satisfies \(|y| = Cx\text{,}\) for some constant \(C \gt 0\text{.}\)

  1. What is the largest possible domain of \(f(x)\text{,}\) given the information at hand?

  2. Give an example of function \(y=f(x)\) with the following properties, or show that none exists:

    • \(|y| = Cx\text{,}\)
    • \(\diff{y}{x}\) exists for all \(x \gt 0\text{,}\) and
    • \(y \gt 0\) for some values of \(x\text{,}\) and \(y \lt 0\) for others.
