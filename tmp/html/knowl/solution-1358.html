Solution 8.5.3.6.1.

We apply the ratio test for the series whose \(k^{\rm th}\) term is \(a_k= \frac{x^k}{ 10^{k+1}(k+1)! }\text{.}\) Then

\begin{align*} \lim_{k\to\infty} \bigg| \frac{ a_{k+1} }{ a_k } \bigg| &= \lim_{k\to\infty} \bigg| \frac{x^{k+1}}{10^{k+2}(k+2)!} \cdot \frac{10^{k+1}(k+1)!}{x^k} \bigg|\\ &=\lim_{k\to\infty} \left|\frac{10^{k+1}}{10^{k+2}}\right|\cdot\left|\frac{(k+1)!}{(k+2)!}\right|\cdot\left|\frac{x^{k+1}}{x^k}\right|\\ &= \lim_{k\to\infty}\frac{1}{10(k+2)}|x| = 0 \lt 1 \end{align*}

for all \(x\text{.}\) Therefore, by the ratio test, the series converges for all \(x\) and the radius of convergence is \(R = \infty\text{.}\)

Alternatively, one can set \(A_k = \dfrac{1}{10^{k+1}(k+1)!}\) and compute \(A = \displaystyle\lim_{k\to\infty} \left| \frac{A_{k+1}}{A_k}\right| = 0\text{,}\) so that \(R\) is again \(+\infty\text{.}\)
