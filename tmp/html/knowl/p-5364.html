Some comments:

  • Note that, in Definition 6.1.9, we allow \(a\) and \(b\) to be any two real numbers. We do not require that \(a \lt b\text{.}\) That is, even when \(a \gt b\text{,}\) the symbol \(\int_a^b f(x)\dee{x}\) is still defined by the formula of Definition 6.1.9. We'll get an interpretation for \(\int_a^b f(x)\dee{x}\text{,}\) when \(a \gt b\text{,}\) later.
  • It is important to note that the definite integral \(\int_a^b f(x)\dee{x}\) represents a number, not a function of \(x\text{.}\) The integration variable \(x\) is another “dummy” variable, just like the summation index \(i\) in \(\sum_{i=m}^n a_i\) (see Section 6.1.3). The integration variable does not have to be called \(x\text{.}\) For example
    \begin{gather*} \int_a^b f(x)\dee{x} = \int_a^b f(t)\dee{t} = \int_a^b f(u)\dee{u} \end{gather*}
    Just as with summation variables, the integration variable \(x\) has no meaning outside of \(f(x)\dee{x}\text{.}\) For example
    \begin{gather*} x\int_0^1 e^x\dee{x}\qquad\text{and}\qquad \int_0^x e^x\dee{x} \end{gather*}
    are both gibberish.
