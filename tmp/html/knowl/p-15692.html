(a) The curve \(x^2+y^2=1\) is a circle of radius \(1\text{.}\) So we can parametrize it by \(x(\theta)=\cos\theta\text{,}\) \(y(\theta)=\sin\theta\text{,}\) \(0\le\theta \lt 2\pi\text{.}\) The \(z\)-coordinate of any point on the intersection is determined by \(z=1-x-y\text{.}\) So we can use

\begin{equation*} \vr(\theta) = \cos\theta\,\hi +\sin\theta\,\hj + (1-\cos\theta-\sin\theta)\,\hk \qquad 0\le\theta \lt 2\pi \end{equation*}
