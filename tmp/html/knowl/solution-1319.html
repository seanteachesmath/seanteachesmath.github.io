Solution 8.3.11.35.1.

For large \(k\text{,}\) \(k^4\gg 2k^3-2\) and \(k^5\gg k^2+k\) so

\begin{equation*} \frac{k^4-2k^3+2}{k^5+k^2+k}\approx \frac{k^4}{k^5} =\frac{1}{k}. \end{equation*}

This suggests that we apply the limit comparison test with \(a_k=\frac{k^4-2k^3+2}{k^5+k^2+k}\) and \(b_k=\frac{1}{k}\text{.}\) Since

\begin{align*} \lim_{k\rightarrow\infty} \frac{a_k}{b_k} &=\lim_{k\rightarrow\infty} \frac{k^4-2k^3+2}{k^5+k^2+k}\cdot \frac{k}{1} =\lim_{k\rightarrow\infty} \frac{k^5-2k^4+k^2}{k^5+k^2+k}\\ \amp=\lim_{k\rightarrow\infty} \frac{1-2/k+1/k^3}{1+1/k^3+1/k^4}\\ &=1 \end{align*}

and since \(\sum\limits_{k=1}^\infty\frac{1}{k}\) diverges (by the \(p\)-test with \(p=1\)), the given series diverges.
