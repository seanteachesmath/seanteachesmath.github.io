Hence we have for all rational \(r \gt 0\)

\begin{align*} \lim_{x \to \infty} \frac{1}{x^r} &= 0 \end{align*}

but we have to be careful with

\begin{align*} \lim_{x \to -\infty} \frac{1}{x^r} &= 0 \end{align*}

This is only true if the denominator of \(r\) is not an even number  2 .
