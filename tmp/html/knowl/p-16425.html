If \(\varphi\) is a potential for \(\vF\text{,}\) then:

  • \(\pdiff{\varphi}{x}=\frac1x-\frac1y\text{,}\) so \(\varphi = \ln|x|-\frac{x}{y}+\psi_1(y)\)
  • \(\pdiff{\varphi}{y}=\frac{x}{y^2}\text{,}\) so \(\varphi = -\frac{x}{y}+\psi_2(x)\)

So, for instance, \(\varphi=\ln |x| - \frac{x}{y}\) is a potential for \(\vF\text{.}\)
