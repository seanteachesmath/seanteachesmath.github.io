Solution:

  • The integral \(\int_{1/2}^\infty e^{-x^2}\dee{x}\) is quite similar to the integral \(\int_1^\infty e^{-x^2}\dee{x}\) of Example 6.12.18. But we cannot just repeat the argument of Example 6.12.18 because it is not true that \(e^{-x^2}\le e^{-x}\) when \(0 \lt x \lt 1\text{.}\)
  • In fact, for \(0 \lt x \lt 1\text{,}\) \(x^2 \lt x\) so that \(e^{-x^2} \gt e^{-x}\text{.}\)
  • However the difference between the current example and Example 6.12.18 is
    \begin{align*} \int_{1/2}^\infty e^{-x^2}\dee{x}-\int_1^\infty e^{-x^2}\dee{x} &= \int_{1/2}^1 e^{-x^2}\dee{x} \end{align*}
    which is clearly a well defined finite number (its actually about \(0.286\)). It is important to note that we are being a little sloppy by taking the difference of two integrals like this â€” we are assuming that both integrals converge. More on this below.
  • So we would expect that \(\int_{1/2}^\infty e^{-x^2}\dee{x}\) should be the sum of the proper integral integral \(\int_{1/2}^1 e^{-x^2}\dee{x}\) and the convergent integral \(\int_1^\infty e^{-x^2}\dee{x}\) and so should be a convergent integral. This is indeed the case. The Theorem below provides the justification.
