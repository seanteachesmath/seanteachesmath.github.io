(b) Let \(a \gt 0\text{.}\) Write \(\sigma_a = \Set{(x,y,z)}{x^2+y^2+z^2 = a^2}\text{.}\) The outward unit normal to \(\sigma_a\) is \(\hn = \frac{\vr}{|\vr|}\) so that

\begin{align*} \int_{\sigma_a}\vF\cdot\hn\, \dee{S} &=\int_{|\vr|=a} \frac{\vr}{|\vr|^3}\cdot \frac{\vr}{|\vr|}\ \dee{S} =\int_{|\vr|=a} \frac{1}{|\vr|^2}\ \dee{S} =\frac{1}{a^2}\int_{|\vr|=a}\dee{S}\\ &=\frac{1}{a^2}\big(4\pi a^2\big) =4\pi\ne 0 \end{align*}
