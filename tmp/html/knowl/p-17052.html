We'll start by computing, in several different ways, the surface area of the hemisphere

\begin{equation*} x^2+y^2+z^2=a^2\qquad z\ge 0 \end{equation*}

(with \(a \gt 0\)). You probably know, from high school, that the answer is \(\frac{1}{2}\times 4\pi a^2=2\pi a^2\text{.}\) But you have probably not seen a derivation of this answer. Note that, since \(x^2+y^2 = a^2-z^2\) on the hemisphere, the set of \((x,y)\)'s for which there is a \(z\) with \((x,y,z)\) on the hemisphere is exactly \(\Set{(x,y)\in\bbbr^2}{x^2+y^2\le a^2}\text{.}\)
