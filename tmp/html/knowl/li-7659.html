Item 16.2.4.i.

Let \(\vF\) be a 2D vector field which is defined everywhere except at the points marked \(P\) and \(Q\text{.}\) Suppose that \(\vnabla\times\vF = 0\) everywhere on the domain of \(\vF\text{.}\) Consider the five curves \(R\text{,}\) \(S\text{,}\) \(T\text{,}\) \(U\text{,}\) and \(V\) shown in the picture.

Which of the following is necessarily true?

  1. \(\displaystyle \int_S \vF\cdot\dee{\vr} = \int_T \vF\cdot\dee{\vr}\)
  2. \(\displaystyle \int_R \vF\cdot\dee{\vr} = \int_S \vF\cdot\dee{\vr} = \int_T \vF\cdot\dee{\vr} = \int_U \vF\cdot\dee{\vr} = 0\)
  3. \(\displaystyle \int_R \vF\cdot\dee{\vr} + \int_S \vF\cdot\dee{\vr} + \int_T \vF\cdot\dee{\vr} = \int_U \vF\cdot\dee{\vr}\)
  4. \(\displaystyle \int_U \vF\cdot\dee{\vr} = \int_R \vF\cdot\dee{\vr} + \int_S \vF\cdot\dee{\vr}\)
  5. \(\displaystyle \int_V \vF\cdot\dee{\vr} = 0\)
