ExampleÂ 13.3.3 tells us \(\frac{1}{2} m |\vv(t)|^2 -\varphi\big(x(t),y(t),z(t)\big)=E\) is a constant quantity, provided \(\vF\) is conservative with potential \(\varphi\text{.}\) So, it would be nice if \(\vF\) were conservative.

If \(\vF = \vnabla\varphi\text{,}\) then

  • \(\pdiff{\varphi}{x}=0\text{,}\) so \(\varphi = \psi_1(y,z)\)
  • \(\pdiff{\varphi}{y}=1\text{,}\) so \(\varphi = y+\psi_2(x,z)\)
  • \(\pdiff{\varphi}{z}=3z^{1/3}\text{,}\) so \(\varphi = \frac{9}{4}z^{4/3}+\psi_3(x,y)\)

We can choose \(\varphi(x,y,z)=y+\frac{9}{4}z^{4/3}\text{.}\) So, \(\frac{1}{2} m |\vv(t)|^2 -\varphi\big(x(t),y(t),z(t)\big)=E\) is a constant quantity, as desired. Using the information that the particle has mass \(1/2\text{,}\) and speed \(1\) when it is at the origin:

\begin{align*} E&=\frac{1}{2}\cdot\frac{1}{2}|1|^2-\varphi\big(0,0,0\big)=\frac{1}{4}\\ \end{align*}

When the particle is at \((1,1,1)\text{:}\)

\begin{align*} \frac{1}{4}&=\frac{1}{2}\cdot\frac{1}{2}|\vv|^2-\varphi(1,1,1)=\frac{|\vv|^2}{4}-\left(1+\frac{9}{4}\right)\\ |\vv|&=\sqrt{14} \end{align*}

So, at the point \((1,1,1)\text{,}\) the particle has speed \(\sqrt{14}\text{.}\)
