2.4.2.50.b When we evaluate \(\displaystyle\lim_{x \rightarrow -4^+} f(x)\text{,}\) we only consider values of \(x\) that are greater than \(-4\text{.}\) For these values,

\begin{gather*} f(x)=\frac{x^2+8x+16}{x^2+30x-4} \end{gather*}

So

\begin{gather*} \displaystyle\lim_{x \rightarrow -4^+} f(x)=\lim_{x \rightarrow -4^+} \frac{x^2+8x+16}{x^2+30x-4} \end{gather*}

This is a rational function, and \(x=-4\) is in its domain (we aren't doing anything suspect, like dividing by \(0\)), so again we can directly substitute \(x=-4\) to evaluate the limit:

\begin{gather*} \displaystyle\lim_{x \rightarrow -4^+} f(x)= \frac{(-4)^2+8(-4)+16}{(-4)^2+30(-4)-4} =\frac{0}{-108}=0 \end{gather*}
