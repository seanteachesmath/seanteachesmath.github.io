Solution 3.14.2.7.1.

We use implicit differentiation, twice.

\begin{align*} 2x+2yy'&=0\\ 2+(2y)y''+(2y')y'&=0\\ y''&=-\frac{(y')^2+1}{y}\\ \end{align*}

So, we need an expression for \(y'\text{.}\) We use the equation \(2x+2yy'=0\) to conclude \(y'=-\dfrac{x}{y}\text{:}\)

\begin{align*} y''&=-\frac{\left(-\frac{x}{y}\right)^2+1}{y}\\ &=-\frac{\frac{x^2}{y^2}+1}{y}\\ &=-\frac{x^2+y^2}{y^3}\\ &=-\frac{1}{y^3} \end{align*}
