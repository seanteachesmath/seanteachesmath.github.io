Before we can compute \(F'(a)\text{,}\) we need to set up some ground work, and in particular the definitions of our given derivatives:

\begin{align*} f'(b) &= \lim_{H \to 0} \frac{f(b+H)-f(b)}{H} & \text{and }&& g'(a) &= \lim_{h \to 0} \frac{g(a+h)-g(a)}{h}. \end{align*}

We are going to use similar manipulation tricks as we did back in the proofs of the arithmetic of derivatives in Section 3.5. Unfortunately, we have already used up the symbols “\(F\)” and “\(H\)”, so we are going to make use the Greek letters \(\gamma, \varphi\text{.}\)
