Solution 12.7.4.2.1.

This equation stems from \(\vF=m\va\text{.}\) In that equation, \(\va\) is acceleration — the second derivative of position with respect to time. So, \(\vv\) is the derivative of position with respect to time.

We previously used \(\vv\) as the derivative of position with respect to the parameter we use to define our position — which was often called \(t\text{,}\) but was not the necessarily time. So this is a good point to keep straight.
