(b) Since

\begin{align*} \nabla\times\vG&=\nabla\times\big(xyz\hi-xyz\hj+g(x,y,z)\hk\big)\\ &=(g_y+xy)\,\hi-(g_x-xy)\,hj+(-yz-xz)\,\hk \end{align*}

we will have that \(\nabla\times\vG=\vF\) if and only if

\begin{align*} &(g_y+xy)\,\hi-(g_x-xy)\,\hj+(-yz-xz)\,\hk\\ &\hskip1in=(xz+xy)\,\hi-(yz-xy)\,\hj-(yz+xz)\,\hk \end{align*}

which is the case if and only if

\begin{equation*} g_y=xz,\quad g_x=yz \end{equation*}

The first equation, \(g_y=xz\text{,}\) is satisfied if and only if \(g=xyz+h(x,z)\text{.}\) The second equation is also satisfied if and only if \(g_x=yz+h_x(x,z)=yz\text{.}\) This is the case if and only if \(h_x(x,z)=0\text{.}\) That is, if and only if \(h\) is independent of \(x\text{.}\) Equivalently, if and only if \(h(x,z)=w(z)\) for some function \(w(z)\text{.}\) So, in fact, any function of the form \(g(x,y,z)=xyz+w(z)\) will work.
