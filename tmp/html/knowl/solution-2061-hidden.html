We'll solve this problem twice, using two different strategies. (The second strategy will be much more efficient than the first one.) Both strategies use that \(\vF=m\va\text{.}\) Since we are told that \(m=2\text{,}\) we just have to find the acceleration \(\va\) at \((1,1,1)\text{.}\)

Strategy 1: In the first strategy, we'll find the position \(\vr(t)\text{,}\) as a function of time \(t\) and then differentiate twice to get the acceleration \(\va(t)\text{.}\)

  • First we'll find any old parametrization. We are told that, on the path, \(z=x\) and \(z=y^2\text{.}\) So let's use \(y\) as the parameter. Then \(x=z=y^2\text{.}\) So the parametrization is \(\vR(y)=y^2\,\hi+ y\,\hj +y^2\,\hk\text{.}\) (We'll save the notation “\(\vr(t)\)” for the parametrization with respect to time.)
  • Next we'll reparametrize to get the time \(t\) as the parameter. Since
    \begin{align*} \diff{\vR}{y} &= 2y\,\hi+\hj + 2y\,\hk\\ \implies \diff{s}{y} &=\big|2y\,\hi+\hj + 2y\,\hk\big| =\sqrt{1+8y^2} \end{align*}
    We are told that the speed \(\diff{s}{t}=3\) for all \(t\text{.}\) So, choosing our zero point for time to coincide with our zero point for \(s\text{,}\) we have \(s=3t\text{,}\) or \(t=s/3\) so that
    \begin{equation*} \diff{t}{y} =\frac{1}{3}\sqrt{1+8y^2} \end{equation*}
    We could now integrate to get \(t\) as a function of \(y\text{.}\) But that looks quite messy. Fortunately we only need the acceleration at one point, namely \((1,1,1)\text{.}\) We'll now see that that saves quite a bit of work. Pretend that we have integrated to get \(t\) as a function of \(y\) and call the answer \(t(y)\text{.}\) Call the inverse function, which gives \(y\) as a function of \(t\text{,}\) \(y(t)\text{.}\)
  • We now have \(\vr(t) = \vR\big(y(t)\big)\text{.}\) So, by the chain rule,
    \begin{align*} \vr'(t) &= \vR'\big(y(t)\big)\ y'(t)\\ \vr''(t) &= \vR'\big(y(t)\big)\ y''(t) + \vR''\big(y(t)\big)\ y'(t)^2 \end{align*}
    We're only interest in the time, call it \(t_0\text{,}\) at which \(y(t_0)=1\text{.}\) The acceleration at time \(t_0\) is
    \begin{align*} \vr''(t_0) &= \vR'\big(y(t_0)\big)\ y''(t_0) + \vR''\big(y(t_0)\big)\ y'(t_0)^2\\ & = \vR'(1)\ y''(t_0) + \vR''(1)\ y'(t_0)^2\\ & = \big[2\,\hi+\hj + 2\,\hk\big]\ y''(t_0) + \big[2\,\hi+ 2\,\hk\big]\ y'(t_0)^2 \end{align*}
    so we just have to find \(y'(t_0)\) and \(y''(t_0)\text{.}\)
  • We know that \(\diff{t}{y} =\frac{1}{3}\sqrt{1+8y^2}\text{.}\) So by the inverse function theorem
    \begin{align*} \diff{y}{t}(t) &=\frac{3}{\sqrt{1+8y(t)^2}}\\ \difftwo{y}{t}(t) &=-\frac{1}{2}\,\frac{3\big(16y(t)y'(t)\big)}{{[1+8y(t)^2]}^{3/2}} \end{align*}
    In particular
    \begin{align*} y'(t_0) &=\frac{3}{\sqrt{1+8y(t_0)^2}} =\frac{3}{\sqrt{1+8}} =1\\ y''(t_0) &=-\frac{24\,y(t_0)y'(t_0)}{{[1+8y(t_0)^2]}^{3/2}} = - \frac{24\times 1\times 1}{{(1+8)}^{3/2}} = - \frac{8}{9} \end{align*}
  • Finally, the force is
    \begin{align*} 2\vr''(t_0) &= 2\big[2\,\hi+\hj + 2\,\hk\big]\ y''(t_0) + 2\big[2\,\hi+ 2\,\hk\big]\ y'(t_0)^2\\ &= -\frac{16}{9}\big[2\,\hi+\hj + 2\,\hk\big] +2\big[2\,\hi+ 2\,\hk\big]\\ &= \frac{4}{9}\,\hi -\frac{16}{9}\,\hj + \frac{4}{9}\,\hk \end{align*}

Strategy 2: The second strategy will be based on (see §12.5)

\begin{equation*} \va=\difftwo{s}{t}\,\hat\vT +\ka\Big(\diff{s}{t}\Big)^2\hat\vN \end{equation*}

In this problem, we are told that \(\diff{s}{t}=3\) for all \(t\text{,}\) so that \(\difftwo{s}{t}=0\) and

\begin{equation*} \va=9\ka\hat\vN \end{equation*}

So we just have to find the curvature, \(\ka\text{,}\) and unit normal, \(\hat\vN\text{,}\) at \((1,1,1)\text{.}\) We have already found one parametrization of the path in strategy 1, namely

\begin{equation*} \vR(y)=y^2\,\hi+ y\,\hj +y^2\,\hk \end{equation*}

Note that \(\vR(1) = (1,1,1)\text{.}\) Since

\begin{align*} \vR'(y)&=2y\,\hi+ \hj +2y\,\hk\\ \hat\vT(y)&=\frac{\vR'(y)}{|\vR'(y)|} =\frac{2y\,\hi+ \hj +2y\,\hk}{\sqrt{1+8y^2}}\\ \hat\vT'(y)&= \frac{2\,\hi +2\,\hk}{\sqrt{1+8y^2}} -\frac{16y}{2}\ \frac{2y\,\hi+ \hj +2y\,\hk}{[1+8y^2]^{3/2}}\\ \hat\vT'(1)&= \frac{2\,\hi +2\,\hk}{3} -8\frac{2\,\hi+ \hj +2\,\hk}{27} =\frac{2\,\hi-8\,\hj +2\,\hk}{27} \end{align*}

we have (again see §12.5)

\begin{align*} \ka(1) &=\frac{|\hat\vT'(1)|}{|\vR'(1)|}\\ \hat\vN(1) &=\frac{\hat\vT'(1)}{|\hat\vT'(1)|}\\ \vF&=m\va = 2\times 9\ka(1)\hat\vN(1) =18\frac{\hat\vT'(1)}{|\vR'(1)|} =18\frac{2\,\hi-8\,\hj +2\,\hk}{27\sqrt{1+8\times 1^2}}\\ &=\frac{4}{9}(\hi-4\,\hj+\hk) \end{align*}
