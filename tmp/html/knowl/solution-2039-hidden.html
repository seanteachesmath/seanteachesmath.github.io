(a) If \(\ka(s)\equiv 0\text{,}\) then \(\diff{\hT}{s}=\ka(s)\hN(s)\equiv 0\) so that \(\hT\) is a constant. As a result \(\diff{\vr}{s}(s)=\hT\) and \(\vr(s) =s\hT+\vr(0)\) so that the curve is the straight line with direction vector \(\hT\) that passes through \(\vr(0)\text{.}\)

(b) If \(\tau(s)\equiv 0\text{,}\) then \(\diff{\hB}{s}=-\tau(s)\hN(s)\equiv 0\) so that \(\hB\) is a constant. As \(\hT(s)\perp\hB\text{,}\)

\begin{equation*} \diff{}{s} (\vr(s)-\vr(0))\cdot\hB =\hT(s)\cdot\hB=0 \end{equation*}

and \((\vr(s)-\vr(0))\cdot\hB\) must be a constant. The constant must be zero (set \(s=0\)), so \((\vr(s)-\vr(0))\cdot\hB=0\) and \(\vr(s)\) always lies in the plane through \(\vr(0)\) with normal vector \(\hB\text{.}\)

(c) Parametrize the curve by arc length. Define the “centre of curvature” at \(s\) by

\begin{equation*} \vr_c(s)=\vr(s)+\frac{1}{\ka(s)}\hN(s) \end{equation*}

Since \(\ka(s)=\ka_0\) is a constant and \(\tau(s)\equiv 0\text{,}\)

\begin{equation*} \diff{}{s}\vr_c(s) =\hT(s)+\frac{1}{\ka_0}\big[\tau(s)\hB-\ka(s)\hT\big] =\hT(s)+\frac{1}{\ka_0}\big[0\hB-\ka_0\hT\big] =0 \end{equation*}

Thus \(\vr_c(s)=\vr_c\) is a constant and \(\big|\vr(s)-\vr_c\big|=\frac{1}{\ka_0}\) lies on the sphere of radius \(\frac{1}{\ka_0}\) centred on \(\vr_c\text{.}\) Since \(\tau(s)\equiv 0\text{,}\) the curve also lies on a plane, so it is a circle.
