We could determine whether or not the series \(\sum_{n=1}^\infty\frac{1}{n^2+2n+3}\) converges by applying the integral test. But it is not worth the effort  6 . Whether or not any series converges is determined by the behaviour of the summand  7  for very large \(n\text{.}\) So the first step in tackling such a problem is to develop some intuition about the behaviour of \(a_n\) when \(n\) is very large.

  • Step 1: Develop intuition. In this case, when \(n\) is very large  8  \(n^2\gg 2n \gg 3\) so that \(\frac{1}{n^2+2n+3}\approx\frac{1}{n^2}\text{.}\) We already know, from Example 8.3.6, that \(\sum_{n=1}^\infty\frac{1}{n^p}\) converges if and only if \(p \gt 1\text{.}\) So \(\sum_{n=1}^\infty\frac{1}{n^2}\text{,}\) which has \(p=2\text{,}\) converges, and we would expect that \(\sum_{n=1}^\infty\frac{1}{n^2+2n+3}\) converges too.
  • Step 2: Verify intuition. We can use the comparison test to confirm that this is indeed the case. For any \(n\ge 1\text{,}\) \(n^2+2n+3 \gt n^2\text{,}\) so that \(\frac{1}{n^2+2n+3}\le\frac{1}{n^2}\text{.}\) So the comparison test, Theorem 8.3.8, with \(a_n=\frac{1}{n^2+2n+3}\) and \(c_n=\frac{1}{n^2}\text{,}\) tells us that \(\sum_{n=1}^\infty\frac{1}{n^2+2n+3}\) converges.
