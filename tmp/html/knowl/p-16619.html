Note that

  • \(y=0\) on the line segment from \((1, 0, 0)\) to \((0,0,1)\) so that the integral reduces to \(\int zx \,\dee{z}\) on that line segment and
  • \(x=0\) on the line segment from \((0,0,1)\) to \((0, 1, 0)\) so that the integral reduces to \(\int yz \,\dee{y}\) on that line segment and
  • \(z=0\) on the line segment from \((0, 1, 0)\) to \((1, 0, 0)\) so that the integral reduces to \(\int xy \,\dee{x}\) on that line segment.

So it looks feasible to evaluate the integral directly. Label the sides of the triangle \(C_1\text{,}\) \(C_2\) and \(C_3\) as in the sketch above.

  • We parametrize \(C_1\) by \(\vr(t) = (1,0,0) + t[(0,0,1)-(1,0,0)] = (1-t\,,\,0\,,\,t)\text{,}\) \(0\le t\le 1\text{.}\) So
    \begin{align*} \int_{C_1} xy \,\dee{x} + yz \,\dee{y} + zx \,\dee{z} &= \int_{C_1} zx \,\dee{z} =\int_0^1 \overbrace{(1-t)}^{x} \overbrace{(t)}^{z} \overbrace{(1)}^{z'(t)}\,\dee{t}\\ &=\int_0^1 (t-t^2)\,\dee{t}\\ &=\frac{1}{2}-\frac{1}{3} =\frac{1}{6} \end{align*}
  • We parametrize \(C_2\) by \(\vr(t) = (0,0,1) + t[(0,1,0)-(0,0,1)] = (0\,,\,t\,,\,1-t)\text{,}\) \(0\le t\le 1\text{.}\) So
    \begin{align*} \int_{C_2} xy \,\dee{x} + yz \,\dee{y} + zx \,\dee{z} &= \int_{C_2} yz \,\dee{y} =\int_0^1 \overbrace{(t)}^{y} \overbrace{(1-t)}^{z} \overbrace{(1)}^{y'(t)}\,\dee{t}\\ &=\int_0^1 (t-t^2)\,\dee{t}\\ &=\frac{1}{2}-\frac{1}{3} =\frac{1}{6} \end{align*}
  • We parametrize \(C_3\) by \(\vr(t) = (0,1,0) + t[(1,0,0)-(0,1,0)] = (t\,,\,1-t\,,\,0)\text{,}\) \(0\le t\le 1\text{.}\) So
    \begin{align*} \int_{C_3} xy \,\dee{x} + yz \,\dee{y} + zx \,\dee{z} &= \int_{C_3} xy \,\dee{x} =\int_0^1 \overbrace{(t)}^{x} \overbrace{(1-t)}^{y} \overbrace{(1)}^{x'(t)}\,\dee{t}\\ &=\int_0^1 (t-t^2)\,\dee{t}\\ &=\frac{1}{2}-\frac{1}{3} =\frac{1}{6} \end{align*}

All together

\begin{align*} \int_{C} xy \,\dee{x} + yz \,\dee{y} + zx \,\dee{z} &=\sum_{\ell=1}^3\int_{C_\ell} xy \,\dee{x} + yz \,\dee{y} + zx \,\dee{z} =3\times\frac{1}{6}=\frac{1}{2} \end{align*}
