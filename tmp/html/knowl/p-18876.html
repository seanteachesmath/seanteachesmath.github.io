(c) True. To have constant curvature \(0\) the curve must have unit tangent vector \(\hat\vT(s)\) obeying

\begin{equation*} \diff{\hat\vT}{s}(s)=\vZero \end{equation*}

(See ยง12.5.) So \(\vr'(s) = \hat\vT(s)\) must be a constant vector. Call it \(\hat\vT_0\text{.}\) Integrating gives

\begin{equation*} \vr(s) = s\hat\vT_0 +\vr_0 \end{equation*}

for some constant vector \(\vr_0\text{.}\) So \(\vr(s)\) lies on the same straight line for all \(s\text{.}\)
