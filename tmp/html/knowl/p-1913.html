Let \(f(u) = u^n\) and let \(g(x)\) be any differentiable function. Set \(F(x) = f\big(g(x)\big) = g(x)^n\text{.}\) Then

\begin{align*} \diff{F}{x} = \diff{}{x} \big( g(x)^n \big) &= n g(x)^{n-1} \cdot g'(x) \end{align*}

This is precisely the result in Example 3.6.6 and Lemma 3.6.7.
