It is a piecewise function defined by

\begin{align*} |x| &= \begin{cases} x & x \geq 0 \\ -x & x \lt 0 \end{cases} \end{align*}

Hence our function \(h(t)\) is really

\begin{align*} h(t) &= \sqrt{t^2} = \begin{cases} t & t \geq 0 \\ -t & t \lt 0 \end{cases} \end{align*}

So that when we evaluate \(h(-7)\) it is

\begin{align*} h(-7) &= \sqrt{ (-7)^2 } = \sqrt{49} = 7 = -(-7) \end{align*}
