Let \(\vF\) be a vector field that is defined and continuous on all of \(\bbbr^2\) (or \(\bbbr^3\)). Then the following three statements are equivalent.

  1. \(\vF\) is conservative. That is, there exists a function \(\varphi\) such that \(\vF=\vnabla\varphi\text{.}\)
  2. The integral \(\oint_\cC\vF\cdot\dee{\vr}=0\) for any closed curve \(\cC\text{.}\)
  3. The integral \(\int\vF\cdot\dee{\vr}\) is path independent. That is, for any points \(P_0\text{,}\) \(P_1\) we have \(\int_{\cC_1}\vF\cdot\dee{\vr}= \int_{\cC_2}\vF\cdot\dee{\vr}\) for all curves \(\cC_1\text{,}\) \(\cC_2\) that start at \(P_0\) and end at \(P_1\text{.}\)

That is, if any one of the three statements are true, then all three are true.
