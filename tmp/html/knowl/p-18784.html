(d) False. For the line integral \(\int_C\big(\vF\times\vG\big)\cdot\dee{\vr}\) to be independent of the path \(C\text{,}\) the vector field \(\vF\times\vG\) has to be conservative and so has to obey \(\vnabla\times(\vF\times\vG)=\vZero\text{.}\) But

  • Not all vector fields are conservative. For example, the vector field \(\vH = x\,\hj\) obeys \(\vnabla\times\vH = \hk\) and so is not conservative.
  • We can make \(\vF\times\vG\) be any vector field through judicious choices of \(\vF\) and \(\vG\text{.}\) For example, if \(\vF= x\,\hk\) and \(\vG = \hi\text{,}\) then \(\vF\times\vG=x\,\hk\times\hi=x\,\hj=\vH\text{.}\)
