Given the form of \(\vF\text{,}\) direct evaluation looks hard. So let's try Stokes' theorem, using as \(S\) the part of the plane \(G(x,y,z)=x+2y-z=7\) that is inside \(x^2-2x+4y^2=15\text{.}\) Then

\begin{equation*} \hn\,\dee{S} = \pm \frac{\vnabla G}{\vnabla G\cdot\hk}\,\dee{x}\,\dee{y} = \pm\big(\hi+2\hj-\hk\big)\,\dee{x}\,\dee{y} \end{equation*}

As \(C\) is oriented counterclockwise when viewed from high, Stokes' theorem specifies the upward pointing normal so that \(\hn\,\dee{S} = -\big(\hi+2\hj-\hk\big)\,\dee{x}\,\dee{y}\text{.}\)
