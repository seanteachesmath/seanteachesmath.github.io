Exercises 16.2 Exercises

1. M317 2013D.

True or false?

  1. \(\vnabla\cdot(\va \times\vr) = 0\text{,}\) where \(\va\) is a constant vector in \(\bbbr^3\) , and \(\vr\) is the vector field \(\vr= (x, y, z)\text{.}\)
  2. \(\vnabla\times(\vnabla f) = 0\) for all scalar fields \(f\) on \(\bbbr^3\) with continuous second partial derivatives.
  3. \(\vnabla\cdot(f \vF) = \vnabla(f)\cdot \vF + f \vnabla\cdot\vF\text{,}\) for every vector field \(\vF\) in \(\bbbr^3\) with continuous partial derivatives, and every scalar function \(f\) in \(\bbbr^3\) with continuous partial derivatives.
  4. Suppose \(\vF\) is a vector field with continuous partial derivatives in the region \(D\text{,}\) where \(D\) is \(\bbbr^3\) without the origin. If \(\vnabla\cdot\vF \gt 0\) throughout \(D\text{,}\) then the flux of \(F\) through the sphere of radius \(5\) with center at the origin is positive.
  5. If a vector field \(\vF\) is defined and has continuous partial derivatives everywhere in \(\bbbr^3\text{,}\) and it satisfies \(\vnabla\cdot\vF = 0\text{,}\) everywhere, then, for every sphere, the flux out of one hemisphere is equal to the flux into the opposite hemisphere.
  6. If \(\vr(t)\) is a twice continuously differentiable path in \(\bbbr^2\) with constant curvature \(\kappa\text{,}\) then \(\vr(t)\) parametrizes part of a circle of radius \(1/\kappa\text{.}\)
  7. The vector field \(\vF = \left( -\frac{y}{x^2+y^2}\,,\,\frac{x}{x^2+y^2}\right)\) is conservative in its domain, which is \(\bbbr^2\text{,}\) without the origin.
  8. If a vector field \(\vF = (P, Q)\) in \(\bbbr^2\) has \(Q = 0\) everywhere in \(\bbbr^2\text{,}\) then the line integral \(\oint\vF\cdot\dee{\vr}\) is zero, for every simple closed curve in \(\bbbr^2\text{.}\)
  9. If the acceleration and the speed of a moving particle in \(\bbbr^3\) are constant, then the motion is taking place along a spiral.
Answer

(a) True

(b) True

(c) True

(d) False

(e) True

(f) That depends. If \(\ka=0\text{,}\) the curve is part of a straight line. If \(\ka \gt 0\) it is part of a circle of radius \(\frac{1}{\ka}\text{.}\)

(g) False.

(h) False.

(i) False.

Solution

(a) True. For any constant vector \(\va=(a_1,a_2,a_3)\text{,}\)

\begin{align*} \va\times\vr &=\det\left[\begin{matrix} \hi &\hj &\hk\\ a_1 & a_2 & a_3\\ x & y & z \end{matrix} \right] = (a_2 z - a_3 y)\hi -(a_1 z - a_3 x)\hj +(a_1 y - a_2 x)\hj \end{align*}

This vector field does indeed have divergence \(0\text{.}\)

(b) True. This is our conservative field screening condition Theorem 15.1.7.b.

(c) True. This is one of our vector identities, namely Theorem 15.1.4.c.

(d) False. The trap here is that \(\vF\) need not be defined at the origin. We saw, in Example 14.4.2, that the point source \(\vF_S=\frac{m\vr}{|\vr|^3}\) had flux \(4\pi m\) through every sphere centred on the origin. We also saw, in Example 15.2.7, that the divergence \(\vnabla\cdot\vF_S=0\) everywhere except at the origin (where it is not defined). So if we choose \(m\) to be a very big negative number (say \(-10^{100}\)) and add in a very small vector field with positive divergence (say \(10^{-100}(x\hi+y\hj +z\hk)\)), we will get the vector field \(\vF = -10^{100}\frac{\vr}{|\vr|^3} + 10^{-100}(x\hi+y\hj +z\hk)\) which has divergence \(\vnabla\cdot\vF=3\times 10^{-100} \gt 0\) everywhere except at the origin. The flux of this field through the specified sphere will be \(-4\pi\times 10^{100}\) plus a very small positive number.

(e) True. The statement that “the flux out of one hemisphere is equal to the flux into the opposite hemisphere” is equivalent to the statement that “the flux out of the sphere is equal to zero”. Since \(\vnabla\cdot\vF=0\) everywhere, that is true by the divergence theorem.

(f) That depends.

If \(\ka=0\text{,}\) then \(\diff{\hat\vT}{s}=0\text{,}\) so that \(\diff{\vr}{s}=\hat\vT\) is a constant. So \(\vr(s) = s\hat\vT +\vr(0)\) is part of a straight line.

If \(\ka \gt 0\text{,}\) then, because the curve is in a plane, the torsion \(\tau\) is zero and the Frenet-Serret formulae reduce to

\begin{equation*} \diff{\hat\vT}{s} = \ka\hat\vN\qquad \diff{\hat\vN}{s} = -\ka\hat\vT \end{equation*}

Now consider the centre of curvature \(\vc(s) = \vr(s) +\frac{1}{\ka}\hat\vN(s)\text{.}\) Since

\begin{equation*} \diff{\vc}{s} = \diff{\vr}{s} +\frac{1}{\ka}\diff{\hat\vN}{s} =\hat\vT(s) +\frac{1}{\ka}\big(-\ka\hat\vT(s)\big) =\vZero \end{equation*}

\(\vc(s)\) is a constant and

\begin{equation*} |\vr(s) -\vc| =\frac{1}{\ka} \end{equation*}

which says that the curve is part of the circle of radius \(\frac{1}{\ka}\) centred on \(\vc\text{.}\)

(g) False. We saw in Examples 13.3.14 and 15.3.8 that the given vector field is not conservative.

(h) False. For example, if \(P=-y\text{,}\) then \(\oint_C\vF\cdot\dee{\vr}=-\oint_C y\,\dee{x}\) is the area inside \(C\text{.}\) See Corollary 15.3.5.

(i) False.

If \(\diff{\vv}{t}=\va\) is a constant, then \(\vv(t) =\va\, t+\vv_0\text{.}\) Integrating a second time, \(\vr(t) = \frac{1}{2}\va\,t^2 +\vv_0\, t+\vr_0\text{.}\) This is not a spiral, whether or not the speed is constant. (In fact, for the speed \(|\vv(t)| =|\va\, t+\vv_0|\) to be constant, \(\va\) has to be \(\vZero\text{,}\) so that \(\vr(t) = \vv_0\, t+\vr_0\) is a straight line.)

Another way to come to the same conclusion uses

\begin{equation*} \va(t)=\difftwo{s}{t}(t)\,\hat\vT(t) +\ka(t)\Big(\diff{s}{t}(t)\Big)^2\hat\vN(t) \end{equation*}

As the speed \(\diff{s}{t}\) is a constant, it reduces to

\begin{equation*} \va(t)=\ka(t)\Big(\diff{s}{t}(t)\Big)^2\hat\vN(t) \end{equation*}

As \(\va(t)\) is a constant, its direction, \(\hat\vN(t)\text{,}\) is also a constant. The normal vector to a spiral is not constant.

2. M317 2012D.

True or false?

  1. \(\vnabla\times(\va \times\vr) = 0\text{,}\) where \(\va\) is a constant vector in \(\bbbr^3\) , and \(\vr\) is the vector field \(\vr= (x, y, z)\text{.}\)
  2. \(\vnabla\cdot(\vnabla f) = 0\) for all scalar fields \(f\) on \(\bbbr^3\) with continuous second partial derivatives.
  3. \(\vnabla(\vnabla\cdot \vF) = 0\) for every vector field \(\vF\) on \(\bbbr^3\) with continuous second partial derivatives.
  4. Suppose \(\vF\) is a vector field with continuous partial derivatives in the region \(D\text{,}\) where \(D\) is \(\bbbr^3\) without the origin. If \(\vnabla\cdot\vF = 0\text{,}\) then the flux of \(\vF\) through the sphere of radius \(5\) with center at the origin is \(0\text{.}\)
  5. Suppose \(\vF\) is a vector field with continuous partial derivatives in the region \(D\text{,}\) where \(D\) is \(\bbbr^3\) without the origin. If \(\vnabla\times\vF=\vZero\) then \(\oint_C\vF\cdot\dee{\vr}\) is zero, for every simple and smooth closed curve \(C\) in \(\bbbr^3\) which avoids the origin.
  6. If a vector field \(\vF\) is defined and has continuous partial derivatives everywhere in \(\bbbr^3\text{,}\) and it satisfies \(\vnabla\cdot\vF \gt 0\text{,}\) everywhere, then, for every sphere, the flux out of one hemisphere is larger than the flux into the opposite hemisphere.
  7. If \(\vr(t)\) is a path in \(\bbbr^3\) with constant curvature \(\kappa\text{,}\) then \(\vr(t)\) parametrizes part of a circle of radius \(1/\kappa\text{.}\)
  8. The vector field \(\vF = \left( -\frac{y}{x^2+y^2}\,,\,\frac{x}{x^2+y^2} \,,\,z\right)\) is conservative in its domain, which is \(\bbbr^3\text{,}\) without the \(z\)-axis.
  9. If all flow lines of a vector field in \(\bbbr^3\) are parallel to the \(z\)-axis, then the circulation of the vector field around every closed curve is \(0\text{.}\)
  10. If the speed of a moving particle is constant, then its acceleration is orthogonal to its velocity.
Hint

Read (d), (e), (f), (g), (h) very carefully.

Answer

(a) False

(b) False

(c) False

(d) False

(e) True

(f) True

(g) False

(h) False

(i) False

(j) True

Solution

(a) False. For any constant vector \(\va=(a_1,a_2,a_3)\text{,}\)

\begin{align*} \va\times\vr &=\det\left[\begin{matrix} \hi &\hj &\hk\\ a_1 & a_2 & a_3\\ x & y & z \end{matrix} \right] = (a_2 z - a_3 y)\hi -(a_1 z - a_3 x)\hj +(a_1 y - a_2 x)\hj \end{align*}

So

\begin{align*} \vnabla\times(\va\times\vr) &=\det\left[\begin{matrix} \hi &\hj &\hk\\ \tfrac{\partial }{\partial x} & \tfrac{\partial }{\partial y} & \tfrac{\partial }{\partial z}\\ a_2 z - a_3 y & -a_1 z + a_3 x & a_1 y - a_2 x \end{matrix} \right]\\ &=2 a_1\hi + 2 a_2\hj + 2 a_3\hk \end{align*}

\begin{align*} \dblInt_S \vF\cdot\hn\ \dee{S} &=\tripInt_V \nabla\cdot\vF\ \dee{V} =\tripInt_V \nabla\cdot\nabla \phi\ \dee{V}\\ &=\tripInt_V \big(\tfrac{\partial^2 }{\partial x^2} +\tfrac{\partial^2 }{\partial y^2} +\tfrac{\partial^2 }{\partial z^2}\big)\phi\ \dee{V} \end{align*}

22. M317 2000D, 2001A.

Three quickies:

  1. A moving particle with position \(\vr(t) = (x(t),y(t),z(t))\) satisfies
    \begin{equation*} \va = f(\vr,\vv)\vr \end{equation*}
    for some scalar-valued function \(f\text{.}\) Prove that \(\vr\times\vv\) is constant.
  2. Calculate \(\dblInt_\cS(x\,\hi - y\,\hj + z^2\,\hk)\cdot \hn\dee{S}\text{,}\) where \(\cS\) is the boundary of any solid right circular cylinder of radius \(b\) with one base in the plane \(z=1\) and the other base in the plane \(z=3\text{.}\)
  3. Let \(\vF\) and \(\vG\) be smooth vector fields defined in \(\bbbr^3\text{.}\) Suppose that, for every circle \(C\text{,}\) we have \(\oint_{C} \vF\cdot \dee{\vr}=\dblInt_S \vG\cdot \hn\,\dee{S}\text{,}\) where \(S\) is the oriented disk with boundary \(C\text{.}\) Prove that \(\vG=\vnabla\times\vF\text{.}\)
Answer

(a), (c) See the solution.

(b) \(8\pi b^2\)

Solution

(a) Define \(\vOm(t) = \vr(t)\times\vv(t)\text{.}\) Then by the product rule,

\begin{align*} \diff{\vOm}{t} &= \diff{\vr}{t}\times\vv + \vr\times\diff{\vv}{t} \cr &= \vv\times\vv + \vr\times\big(f(\vr,\vv)\vr\big). \cr &= {\bf0} + f(\vr,\vv) \big(\vr\times\vr\big) = {\bf0}. \end{align*}

It follows that \(\vOm\) is constant.

(b) By the divergence theorem, where \(\cR\) is the solid cylinder as described,

\begin{equation*} \dblInt_\cS(x\,\hi - y\,\hj + z^2\,\hk)\cdot \hn\dee{S} = \tripInt_\cR\big(1 - 1 + 2z\big)\,\dee{V} = 2\tripInt_\cR z\,\dee{V} \end{equation*}

The solid \(\cR\) clearly has reflection symmetry across the plane \(z=2\text{.}\) So the \(z\)-coordinate of the centre of mass of \(\cR\text{,}\) i.e. the average value of \(z\) over \(\cR\text{,}\) i.e.

\begin{equation*} \bar z =\frac{\tripInt_\cR z\,\dee{V}}{\tripInt_\cR\dee{V}} =\frac{\tripInt_\cR z\,\dee{V}}{{\rm Vol}(\cR)} \end{equation*}

is \(2\text{.}\) Hence

\begin{equation*} \dblInt_\cS(x\,\hi - y\,\hj + z^2\,\hk)\cdot \hn\dee{S} = 2\bar z\,{\rm Vol}(\cR) = 4\,{\rm Vol}(\cR) \end{equation*}

By basic geometry, \({\rm Vol}(\cR) = \pi r^2 h = \pi b^2 2\text{.}\) Hence

\begin{equation*} \dblInt_\cS(x\,\hi - y\,\hj + z^2\,\hk)\cdot \hn\dee{S} = 8\pi b^2 \end{equation*}

(c) By Stokes' theorem (Theorem 15.4.1),

\begin{align*} \oint_{\partial D} \vF\cdot \dee{\vr}=\dblInt_D \vG\cdot \hn\,\dee{S} &\implies \dblInt_D \vnabla\times\vF\cdot \hn\,\dee{S}=\dblInt_D \vG\cdot \hn\,\dee{S}\\ &\implies \dblInt_D \big(\vnabla\times\vF-\vG\big)\cdot \hn\,\dee{S}=0 \end{align*}

for all disks \(D\text{.}\) Because this is true for all disks \(D\text{,}\) the integrand must be zero. To see this, let \(\vH=\vnabla\times\vF-\vG\text{.}\) Suppose that \(\vH(\vx_0)\ne 0\text{.}\) Pick a unit vector \(\hn\) in the direction of \(\vH(\vx_0)\text{.}\) Let \(D\) be a very small flat disk centered on \(\vx_0\) with normal \(\hn\) (the vector we picked). Then \(\vH(\vx_0)\cdot\hn \gt 0\) and, by continuity, \(\vH(\vx)\cdot\hn \gt 0\) for all \(\vx\) on \(D\text{,}\) if we have picked \(D\) small enough. Then \(\dblInt_D\big(\vnabla\times\vF-\vG\big)\cdot \hn\,\dee{S} \gt 0\text{,}\) which is a contradiction. So we conclude that \(\vnabla\times\vF-\vG=\vZero\) and hence \(\vG=\vnabla\times\vF\text{.}\)
