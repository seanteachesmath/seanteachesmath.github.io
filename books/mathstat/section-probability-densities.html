Section 4.1 Probability densities

¶

Some words.

Subsection 4.1.1 Univariate distributions

¶

See sections 3.1, 3.2. Recommended problems: (pg80)1,3,4ab,5,6,7a

Definition 4.1.1. probability density.

If \(X\) is a continuous random variable, the function given by \(f(x) = P(X = x)\) for each \(x\) within the range of \(X\) is called the probability density.

Theorem 4.1.2. conditions for probability density.

A function can serve as the probability density for a continuous random variable \(X\) if and only if its values, \(f(x)\text{,}\) satisfy the conditions

  1. \(f(x) \ge 0\) for each value within its domain;

  2. \(\int_{\infty}^\infty f(x)\,dx = 1\text{.}\)

Definition 4.1.3.

If \(X\) is a continuous random variable, the function given by

\begin{equation*} F(x) = P(X \le x) = \int_{\infty}^x f(t) \,dt \text{for} -\infty \lt x \lt \infty \end{equation*}

where \(f(t)\) is the value of the probability density of \(X\) at \(t\text{,}\) is called the distribution function, or the cumulative distribution (function), of \(X\text{.}\)

Theorem 4.1.4. distribution function (continuous).

The values \(F(x)\) of the distribution function of a continuous random variable \(X\) satisfy the conditions

  1. \(F(-\infty) = 0\) and \(F(\infty) = 1\) (more carefully stated as \(\lim_{x\to-\infty}F(x) = 0\) and \(\lim_{x\to\infty} F(x) = 1\));
  2. if \(a \le b\text{,}\) then \(F(a) \le F(b)\) for any real numbers \(a\) and \(b\)

Subsection 4.1.2 Multivariate distributions

¶

Some words.

Definition 4.1.5. joint probability density.

If \(X\) and \(Y\) are continuous random variables, the function given by \(f(x,y) = P(X=x, Y=y)\) for each pair of values \((x,y)\) within the range of \(X\) and \(Y\) is called the joint probability density of \(X\) and \(Y\text{.}\)

Theorem 4.1.6. conditions for a joint probability density.

A bivariate function can serve as a joint probability density for a pair of continuous random variables \(X\) and \(Y\) if and only if its values, \(f(x, y)\text{,}\) satisfy the conditions

  1. \(f(x, y) \ge 0\) for each pair of values \((x, y)\) within its domain;
  2. \(\int\int f(x, y) \,dy\,dx= 1\) where the double summation extends over all possible pairs \((x, y)\text{.}\)
Definition 4.1.7. joint distribution function (continuous).

If \(X\) and \(Y\) are continuous random variables, the function given by

\begin{equation*} F(x, y) = P(X \le x, Y \le y) = \int\int f(s, t) \,ds\,dt\text{ for } \infty \lt x, y \lt \infty \end{equation*}

where \(f(s, t)\) is the value of the joint probability density of \(X\) and \(Y\) at \((s, t)\text{,}\) is called the joint density function or joint cumulative density of \(X\) and \(Y\text{.}\)

Definition 4.1.8. marginal density.

If \(X\) and \(Y\) are continuous random variables and \(f(x, y)\) is the value of their joint probability density at \((x, y)\text{,}\) the function given by

\begin{equation*} g(x) = \int f(x, y)\, dy \end{equation*}

for each \(x\) within the range of \(X\) is called the marginal density of \(X\text{.}\) Correspondingly, the function given by

\begin{equation*} h(y) = \int f(x, y)\,dx \end{equation*}

for each \(y\) within the range of \(Y\) is called the marginal density of \(Y\text{.}\)

Definition 4.1.9. conditional density.

conditional density

\begin{equation*} f(x|y) = \dfrac{f(x, y)}{h(y)}, h(y)\ne 0 \end{equation*}
\begin{equation*} w(y|x) = \dfrac{f(x, y)}{g(x)}, g(x)\ne 0 \end{equation*}
