Exercise 1. Problem 3.1.

For each of the following, determine whether the given values can serve as the values of a probability distribution of a random variable with the range \(x = 1, 2, 3, \text{ and } 4\text{:}\)

  1. \(f(1) = 0.25,f(2) = 0.75,f(3) = 0.25,\text{ and }f(4) = 0.25\)

  2. \(f(1) = 0.15,f(2) = 0.27,f(3) = 0.29,\text{ and }f(4) = 0.29\)

  3. \(f(1) = \dfrac{1}{19},f(2) = \dfrac{10}{19},f(3) = \dfrac{2}{19},\text{ and }f(4) = \dfrac{5}{19}\)
