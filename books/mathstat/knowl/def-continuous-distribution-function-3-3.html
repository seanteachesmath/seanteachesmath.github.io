Definition 4.1.9.

If \(X\) is a continuous random variable, the function given by

\begin{equation*} \displaystyle F(x) = P(X \le x) = \int_{-\infty}^x f(t) \,dt \text{ for } -\infty \lt x \lt \infty \end{equation*}

where \(f(t)\) is the value of the probability density of \(X\) at \(t\text{,}\) is called the distribution function, or the cumulative distribution (function), of \(X\text{.}\)
