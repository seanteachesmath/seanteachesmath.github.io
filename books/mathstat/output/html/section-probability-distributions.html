Section 3.1 Probability distributions

Some words.

Subsection 3.1.1 Random variables and probability distributions

See sections 3.1, 3.2. Recommended problems: (pg80)1,3,4ab,5,6,7a

Definition 3.1.1. probability distribution.

If \(\displaystyle X\) is a discrete random variable, the function given by \(\displaystyle f(x) = P(X = x)\) for each \(\displaystyle x\) within the range of \(\displaystyle X\) is called the probability distribution.

Theorem 3.1.2. conditions for probability distribution.

A function can serve as the probability distribution for a discrete random variable \(\displaystyle X\) if and only if its values, \(\displaystyle f(x)\text{,}\) satisfy the conditions

  1. \(\displaystyle f(x) \ge 0\) for each value within its domain;

  2. \(\displaystyle \sum_x f(x) = 1\text{,}\) where the summation extends over all the values within its domain

Definition 3.1.3. distribution function.

If \(\displaystyle X\) is a discrete random variable, the function given by where \(\displaystyle f(t)\) is the value of the probability distribution of \(\displaystyle X\) at \(\displaystyle t\text{,}\) is called the distribution function, or the cumulative distribution (function), of \(\displaystyle X\text{.}\)

Theorem 3.1.4. properties of a distribution function.

The values \(\displaystyle F(x)\) of the distribution function of a discrete random variable \(\displaystyle X\) satisfy the conditions

  1. \(\displaystyle F(-\infty) = 0\) and \(\displaystyle F(\infty) = 1\) (more carefully stated as \(\displaystyle \lim_{x\to-\infty}F(x) = 0\) and \(\displaystyle \lim_{x\to\infty} F(x) = 1\));
  2. if \(\displaystyle a \lt b\text{,}\) then \(\displaystyle F(a) \le F(b)\) for any real numbers \(\displaystyle a\) and \(\displaystyle b\)
Example 3.1.5. Verifying a simple probability mass function.

Identify whether the function \(f(x) = \dfrac{2x-1}{5}\) is a probability mass function for the discrete random variable \(X\) with values \(x=1,2,3,4,5\text{.}\) If it is not a valid distribution "fix it".

Solution

Notice that values of \(f(x) \ge 0\) for all possible \(x\text{,}\) but

\begin{equation*} \sum\limits_{x=1}^5 f(x) = 5 \neq 1. \end{equation*}

The function must be rescaled so that probabilities sum to one. The function \(f(x) = \dfrac{2x-1}{25}\) is a valid probability mass function for the values of the random variable mentioned above.

Example 3.1.8. CDF of a simple probability mass function.

The function \(f(x) = \dfrac{2x-1}{25}\) is a probability mass function for the discrete random variable \(X\) with values \(x=1,2,3,4,5\text{.}\) Find the cumulative distribution function.

Solution

We have

\begin{equation*} F(x) = \begin{cases} 0, \amp \quad x \lt 1\\ \frac{1}{25}, \amp \quad 1 \le x \lt 2\\ \frac{4}{25}, \amp \quad 2 \le x \lt 3\\ \frac{9}{25}, \amp \quad 3 \le x \lt 4\\ \frac{16}{25}, \amp \quad 4 \le x \lt 5\\ 1, \amp \quad 5 \le x \end{cases} \end{equation*}

The function \(F(x)\text{,}\) is piecewise constant, accumulating new probability at the values of \(x = 1, 2, 3, 4, 5\text{.}\)

Exercises 3.1.2 Exercises

1. Problem 3.1.

For each of the following, determine whether the given values can serve as the values of a probability distribution of a random variable with the range \(x = 1, 2, 3, \text{and } 4\text{:}\)

  1. \(\displaystyle f(1) = 0.25,f(2) = 0.75,f(3) = 0.25, \text{and } f(4) = 0.25\)

  2. \(\displaystyle f(1) = 0.15,f(2) = 0.27,f(3) = 0.29, \text{and } f(4) = 0.29\)

  3. \(\displaystyle f(1) = \dfrac{1}{19},f(2) = \dfrac{10}{19},f(3) = \dfrac{2}{19}, \text{and } f(4) = \dfrac{5}{19}\)

2. Problem 3.3.

Verify that \(\displaystyle f(x) = \dfrac{2x}{k(k+1)}\) for \(x=1, 2, 3, \dots, k\) can serve as the probability distribution of a random variable with the given range. Bonus: Find or describe \(F(x)\text{.}\)}

3. Problem 3.4b.

Determine \(c\) so that the function \(\displaystyle f(x) = c{5\choose x}\) for \(x = 0, 1, 2, 3, 4, 5\) can serve as a probability distribution of a random variable with the given range.

4. Problem 3.5.

For what values of \(k\) an

\begin{equation*} f(x) = (1-k)k^x \end{equation*}

serve as the values of the probability distribution of a random variable with countably infinite range \(x = 0, 1, 2, \dots\text{?}\)

5. Problem 3.7a.

Show that there are no values of \(c\) such that

\begin{equation*} f(x) = \dfrac{c}{x} \end{equation*}

can serve as the values of the probability distribution of a random variable with countably infinite range \(x = 0, 1, 2, \dots\)

6. Problem 3.6.

Construct a probability histogram for the probability distribution

\begin{equation*} f(x) = \dfrac{{2\choose x}{4\choose{3-x}}}{{6\choose3}} \text{ for }x=0, 1, 2 \end{equation*}
7. Problem 3.87.

The probability distribution of \(V\text{,}\) the number of weekly accidents at a certain intersection, is given by \(g(0) = 0.4, g(1) = 0.3, g(2) = 0.2, g(3) = 0.1\text{.}\) Construct the distribution function of \(V\) and draw its graph.

8. Problem 3.88.

Using the statement and result of the previous problem, find the probability that there will be at least two accidents in any one week using

  1. the original probabilities, and

  2. the values of the distribution function.
