A continuous random variable \(\displaystyle X\) has an exponential distribution and is referred to as an exponential random variable if and only if its probability density is given by

\begin{equation*} f(x; \theta) = \begin{cases}\dfrac{1}{\theta}e^{-\sfrac{x}{\theta}}, \amp \quad \text{ for } 0 \lt x\\0, \amp \quad \text{elsewhere}\end{cases} \end{equation*}

where \(\theta \gt 0\text{.}\)
