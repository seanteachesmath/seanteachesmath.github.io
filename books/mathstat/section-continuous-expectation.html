Section 4.2 Expectation of continuous random variables

¶

This is the start of Chapter 4 in Freund's Mathematical Statistics. In the first pass we will study the major topics of this chapter with a focus on those applying to discrete random variables.

Subsection 4.2.1 Expected value

¶

See section 4.1 - 4.2. Recommended problems: (pg 136) 7, 9, 10, 11, (pg 161) 60

Definition 4.2.1. expected value (continuous).

If \(X\) is a continuous random variable and \(f(x)\) is the value of its probability density at \(x\text{,}\) the expected value of \(X\) is given by

\begin{equation*} E[X] = \int x\cdot f(x)\,dx \end{equation*}
Theorem 4.2.2. expected value of a function of a random variable (continuous).

If \(X\) is a continuous random variable and \(f(x)\) is the value of its probability density at \(x\text{,}\) the expected value of \(g(X)\) is given by

\begin{equation*} E[g(X)] = \int g(x)\cdot f(x)\,dx \end{equation*}

Now would be a good time to review Theorem 3.1.3, Corollary 3.1.4, Corollary 3.1.5, and Theorem 3.1.7, all of which apply here as well.

Theorem 4.2.3. expected value of joint random variables (continuous).

If \(X\) and \(Y\) are continuous random variables and \(f(x, y)\) is the value of their joint probability density at \((x, y)\text{,}\) the expected value of \(g(X, Y)\) is given by

\begin{equation*} E[g(X, Y)] = \int\int g(x, y)\cdot f(x,y)\,dx\,dy \end{equation*}

Subsection 4.2.4 Expected value

¶

See section 4.3. Recommended problems: 4.3 (pg 146) 20, 22, 23, 31, 33, 34, 40, (pg 162) 69, 73, 75

Definition 4.2.4. moments about the origin (continuous).

The \(r^\text{th}\) moment about the origin of a random variable \(X\text{,}\) denoted by \(\mu_r'\text{,}\) is the expected value of \((X)^r\text{;}\) symbolically

\begin{equation*} \mu_r'=E[(X)^r] = \int_{-\infty}^\infty x^r\cdot f(x)\,dx \end{equation*}

for \(r = 0,1,2, \dots\) when \(X\) is continuous.

Definition 4.2.5. moments about the mean (continuous).

The \(\displaystyle r^\text{th}\) moment about the mean of a random variable \(\displaystyle X\text{,}\) denoted by \(\displaystyle \mu_r\text{,}\) is the expected value of \(\displaystyle (X-\mu)^r\text{;}\) symbolically

\begin{equation*} \mu_r=E[(X-\mu)^r] = \int_{-\infty}^\infty (x-\mu)^r\cdot f(x)\,dx \end{equation*}

for \(\displaystyle r = 0,1,2, \dots\) when \(\displaystyle X\) is discrete.

Subsection 4.2.6 Chebyshev's Theorem

¶

See section 4.4.

Theorem 4.2.6. Chebyshev's Theorem.

If \(\mu\) and \(\sigma\) are the mean and the standard deviation of a random variable \(X\text{,}\) then for any positive constant \(k\) the probability is at least \(1-\dfrac{1}{k^2}\) that \(X\) will take on a value within \(k\) standard deviations of the mean; symbolically

\begin{equation*} P(|x-\mu| \lt k\sigma) \ge 1 - \dfrac{1}{k^2}, \sigma \ne 0 \end{equation*}

Theorem 4.2.6 gives a lower bound on the probability that the value of the random variable is within a certain distance (specifically \(\pm k\sigma\)) of the mean.

Subsection 4.2.8 Moment-generating functions

¶

See section 4.5.

Definition 4.2.7. moment-generating function (continuous).

The moment-generating function of a random variable \(\displaystyle X\text{,}\) where it exists, is given by

\begin{equation*} \displaystyle M_X(t) = E[e^{tX}] = \int_{-\infty}^\infty e^{tx}\cdot f(x)\,dx \end{equation*}

when \(\displaystyle X\) is discrete.

Notce that a moment-generating function \(\displaystyle M_X(t)\) itself is a function of the variable \(\displaystyle t\) not \(\displaystyle x\text{.}\) As it turns out, we are most interested in values of the function at or near \(\displaystyle t=0\text{.}\)

Now would be a good time to review Theorem 3.1.14 and Theorem 3.1.16 which apply here as well.

The rules in Theorem 3.1.16 allow us to calculate moment-generating functions of simple functions of a random variable.
