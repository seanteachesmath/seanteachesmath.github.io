Solution 4.3.2.1.

  1. To evaluate \(\log_{3} (9)\text{,}\) we ask what exponent on base \(3\) will produce \(9\text{.}\) Or, 3 to what power equals 9? The exponent we need is \(\alert{2}\text{,}\) so

    \begin{equation*} \log_{3} (9) = \alert{2}~~ \text{ because } ~~3^\alert{2} = 9 \end{equation*}

    We use similar reasoning to compute the other logarithms.

  2. \(\log_{5} (125) = \alert{3}~~ \text{ because } ~~5^\alert{3} = 125\)

  3. \(\log_{4} \left(\dfrac{1}{16}\right)= \alert{-2} ~~\text{ because } ~~4^{\alert{-2}} = \dfrac{1}{16}\)

  4. \(\log_{5} \left(\sqrt{5}\right) = \alert{\dfrac{1}{2}}~~ \text{ because }~~ 5^{\alert{1/2}} =\sqrt{5}\)
