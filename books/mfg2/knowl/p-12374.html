A rowing team can maintain a speed of \(15\) miles per hour in still water. The team's daily training session includes a \(5\)-mile run up the Red Cedar River and the return downstream.

  1. Express the team's time on the upstream leg as a function of the speed of the current.

  2. Write a function for the team's time on the downstream leg.

  3. Write and simplify an expression for the total time for the training run as a function of the current's speed.
