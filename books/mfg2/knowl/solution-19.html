Solution 1.2.30.1.

  1. \(\begin{aligned}[t] (f+g)(x) = \amp f(x)+g(x)\\ = \amp (-x^2+8x-15) + (x^2-2x-15)\\ = \amp -x^2+8x-15 + x^2-2x-15\\ = \amp 6x -30 \end{aligned}\)

  2. When subtracting functions be sure to subtract \emph{all} of the second function. Putting parentheses around the second function will help you remember to do this. \(\begin{aligned}[t] (f-g)(x) = \amp f(x)-g(x)\\ = \amp (-x^2+8x-15) - (x^2-2x-15)\\ = \amp -x^2+8x-15 -x^2+2x+15\\ = \amp -2x^2 +10x \end{aligned}\)

  3. \(\begin{aligned}[t] (fg)(x) = \amp f(x) g(x)\\ = \amp (-x^2+8x-15)(x^2-2x-15)\\ = \amp -x^4 + 2x^3+15x^2 +8x^3-16x^2-120x - 15x^2+30x+225 \\ = \amp - x^4 +10x^3 -16x^2-90x+225 \end{aligned}\)

  4. \(\begin{aligned}[t] \left(\dfrac{f}{g}\right)(x) = \amp \dfrac{-x^2+8x-15}{x^2-2x-15}\\ = \amp \dfrac{(-x+3)(x-5)}{(x-5)(x+3)}\\ = \amp \dfrac{-x+3}{x+3} \end{aligned}\)
