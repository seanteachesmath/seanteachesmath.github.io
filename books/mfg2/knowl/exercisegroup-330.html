For Problems 19-22, verify the following products discussed in the text.

19.

\((x + y)^3 = x^3 + 3x^2 y + 3xy^2 + y^3\)

Answer
20.

\((x - y)^3 = x^3 - 3x^2 y + 3xy^2 - y^3\)

21.

\((x + y)(x^2 - xy + y^2) = x^3 + y^3\)

Answer
22.

\((x - y)(x^2 + xy + y^2) = x^3 - y^3\)
