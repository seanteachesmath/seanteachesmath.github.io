Exercise Group 2.5.6.61–66.

In Problems 61–64, use a graphing calculator to explore some properties of the basic functions.

61.

  1. Graph \(f (x) = x^2\) and \(g(x) = x^3\) on the domain \([0, 1]\) and state the range of each function. On the interval \((0, 1)\text{,}\) which is greater, \(f (x)\) or \(g(x)\text{?}\)

  2. Graph \(f (x) = x^2\) and \(g(x) = x^3\) on the domain \([1, 10]\) and state the range of each function. On the interval \((1, 100)\text{,}\) which is greater, \(f(x)\) or \(g(x)\text{?}\)

Answer
62.

  1. Graph \(f (x) =\sqrt{x} \) and \(g(x) =\sqrt[3]{x} \) on the domain \([0, 1]\) and state the range of each function. On the interval \((0, 1)\text{,}\) which is greater, \(f (x)\) or \(g(x)\text{?}\)

  2. Graph \(f (x) = \sqrt{x}\) and \(g(x) = \sqrt[3]{x}\) on the domain \([1, 100]\) and state the range of each function. On the interval \((1, 100)\text{,}\) which is greater, \(f(x)\) or \(g(x)\text{?}\)

63.

  1. Graph \(f (x) =\dfrac{1}{x} \) and \(g(x) = \dfrac{1}{x^2}\) on the domain \([0.01, 1]\) and state the range of each function. On the interval \((0, 1)\text{,}\) which is greater, \(f (x)\) or \(g(x)\text{?}\)

  2. Graph \(f (x) = \dfrac{1}{x}\) and \(g(x) = \dfrac{1}{x^2}\) on the domain \([1, 10]\) and state the range of each function. On the interval \((1, \infty)\text{,}\) which is greater, \(f(x)\) or \(g(x)\text{?}\)
