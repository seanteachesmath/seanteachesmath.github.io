Exercise Group 7.2.7.65–68.

For Problems 65–68,

  1. Verify that the given value is a zero of the polynomial.

  2. Find the other zeros. (Hint: Use polynomial division to write \(P(x) = (x - a)Q(x)\text{,}\) then factor \(Q(x)\text{.}\))

65.

\(P(x) = x^3 - 2x^2 + 1\text{;}\) \(a = 1\)

Answer
66.

\(P(x) = x^3 + 2x^2 - 1\text{;}\) \(a = -1\)

67.

\(P(x) = x^4 - 3x^3 - 10x^2 + 24x\text{;}\) \(a = -3\)

Answer
68.

\(P(x) = x^4 + 5x^3 - x^2 - 5x\text{;}\) \(a = -5\)
