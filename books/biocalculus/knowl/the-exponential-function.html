‘The’ exponential function.

The exponential function (or sometimes exponential function to the base \(e\)) is given as

\begin{equation*} f(x) = e^{x} \end{equation*}
Definition 1.5.2. natural logarithm.

The exponential function \(e^{x}\) as an inverse called the natural logarithm given by \(\ln(x)\text{.}\)
