Consider limits of the following two basic functions.

  1. Suppose \(f(x)=c\text{,}\) where \(c\) is a constant, for all \(x\text{.}\) Then for any value of \(a\)

    \begin{equation*} \lim_{x\to a}f(x) = \lim_{x\to a}c=c, \end{equation*}

    or, in words the limit of a constant function is that value of that constant as \(x\) approaches any value.

  2. Suppose \(f(x)=x\) for all \(x\text{.}\) Then for any value of \(a\)

    \begin{equation*} \lim_{x\to a}f(x)=\lim_{x\to a} x =a, \end{equation*}

    or, in wordsvalues of the the function \(x\) approach \(a\) as the variable \(x\) approaches \(a\).
