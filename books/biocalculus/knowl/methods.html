To find what original function a derivative ‘comes from,’ we might

  1. guess the original function

  2. graph the original function (from behaviors encoded by the derivative)

  3. approximate the graph of the original function (Euler’s method/tangent line approximations) – skipped

  4. integrate to find the original function

\begin{equation*} \frac{dp}{dt} = 1.8, \end{equation*}

where position \(p(t)\) is the quantity being differentiated with respect to time. Find the solution \(p(t)\text{,}\) that is, find the function whose derivative is \(\displaystyle\frac{dp}{dt} = 1.8\text{.}\)
