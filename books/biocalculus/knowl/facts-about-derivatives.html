A graph of \(f(x)\) can have local optima 1  at a critical point \(x=c\) if \(f'(c)=0\) or \(f'(c)\) is undefined. If the function has an optimum value, it is

  • a local maximum if the derivative switchesfrom

  • a local minimum if the derivative switchesfrom

There is no local optimum value if the derivative does not change sign.

The Second Derivative Test outlined below helps is find local optima at critical points \(x=c\text{:}\)

  • If \(f''(c)<0\text{,}\) then a local maximum value of \(f(x)\)occurs at \(x=c\) and the function value is \(f(c)\)

  • If \(f''(c)>0\text{,}\) then a local minimum value of \(f(x)\)occurs at \(x=c\) and the function value is \(f(c)\)

  • If \(f''(c)=0\text{,}\) then we have to look moreclosely at the sign of the first derivative

We have to know the locations of the critical points (from the first derivative) in order to apply the SDT!

A global optimum value is guaranteed to exist for a function on an interval. The global maximum is the largest value that the function achieves, the global minimum is the smallest value that the function achieves.Global extrema occur at critical points or endpoints of the interval.

The steps are as follows:

  1. Evaluate function at endpoints.

  2. Find all critical points.

  3. Compute the function value at critical points.

  4. Largest is the global maximum, smallest is the global minimum.
