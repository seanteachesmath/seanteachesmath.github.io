Summary.
Three very different looking formulas
\begin{equation*} \begin{aligned}[t] \text{zeroth order: }\amp A(t) = -kt + A_{0}\\ \text{first order: }\amp \ln(A(t)) = -kt + \ln(A_{0})\\ \text{second order: }\amp \frac{1}{A(t)} = kt + \frac{1}{A_{0}} \end{aligned} \end{equation*}
all result from integration of
\begin{equation*} \dfrac{dA}{dt} = -kA^n \end{equation*}
for different values of \(n\) corresponding to the reaction order. Solving each for \(A(t)\) suggests that \(A(t)\) evolves very differently in each case.
