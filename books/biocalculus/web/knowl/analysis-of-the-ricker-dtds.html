Analysis of the Ricker DTDS.
To get some basic information we will analyze the Ricker model by hand before moving onto computing.
  1. With \(r=2\) and \(K=2\text{,}\) find the equilibria \(N^*\) for the Ricker model.
  2. What is \(f'(N)\) if \(f(N) = rNe^{-KN}\text{?}\)
  3. What is \(f'(N^*)\text{?}\) Are equilibria stable? Should solutions oscillate?
  4. Repeat the analysis if \(r=8\text{.}\)
