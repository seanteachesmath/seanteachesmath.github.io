The ‘art’ of compositions.
It helps to think of a function \(h(x)=(f\circ g)(x)=f(g(x))\text{,}\) as being ‘built’ from two component functions \(f\) and \(g\) using function composition.
The function \(F(x)=e^{-x^{2}}\) is a composition of the function \(f(g)=e^g\) with the function \(g(x)=-x^{2}\text{.}\)
The function \(f(z)=\ln{(z^{3}-z)}\) is the composition of which two functions?
The function \(H(y)=\displaystyle\frac{1}{1+y^{2}}\) is the composition of which two functions?
Suppose \(h(x)=(f\circ g)(x)=f(g(x))\text{,}\) where \(f\) and \(g\) are both differentiable functions. Then
\begin{equation*} \boxed{h'(x)=f'(g(x))g'(x) \qquad \textrm{or} \qquad \displaystyle\frac{dh}{dx}=\frac{df}{dg}\frac{dg}{dx}} \end{equation*}
To use the chain rule,
  1. write the function as a composition
  2. take the derivatives of the component pieces
  3. multiply the derivatives together, giving the product the appropriate name
  4. put everything back in terms of the original variable
Find the derivative of \(F(x)=e^{-x^{2}}\text{.}\)
Find the derivative of \(f(z)=\ln{(z^{3}-z)}\text{.}\)
Find the derivative of \(H(y)=\displaystyle\frac{1}{1+y^{2}}\text{.}\)
Find the derivative of \(h(x)=e^{cx}\text{.}\)
