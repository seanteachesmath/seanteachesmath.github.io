The value of the definite integral is \(\displaystyle\int_{0}^{2} x\,dx = \left.\left(\dfrac{1}{2}x^{2}\right)\right|_{0}^{2} = \dfrac{1}{2}(2^{2}) - \dfrac{1}{2}(0^{2}) = 2\text{.}\) The average value is given by
\begin{equation*} \hat f = \dfrac{1}{2-0}\int_{0}^{2} x\,dx = \dfrac{1}{2}\left(2\right) = 1 \end{equation*}
