  1. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) faster than \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = \infty\)Ex.

  2. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) slower than \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = 0\)Ex.

  3. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) at the same rate as \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = \text{nonzero constant}\)Ex.
