Again the unknown function is \(u = A(t)\) and that

\begin{equation*} du = A'(t)\,dt\text{.} \end{equation*}

Now, this isn't immediately useful, but remembering our notation

\begin{equation*} A'(t) = \dfrac{dA}{dt}\text{,} \end{equation*}

so we can write,

\begin{equation*} du = \dfrac{dA}{dt}\,dt\text{.} \end{equation*}

Suddenly we are finished with our process of \(u\)-substittution and can integrate. Continuing from above

\begin{equation*} \begin{aligned}[t] \int\underbrace{A^{-2}}_{u^{-2}}\cdot\underbrace{\frac{dA}{ dt}\,dt}_{du} \amp = \int -k\, dt\\ \int u^{-2}\,du \amp = \int -k\,dt\\ \left(\frac{1}{-2+1}\right)u^{-2+1} \amp = -kt+c\\ -u^{-1}\amp = -kt+c\\ -\frac{1}{u}\amp = -kt+c\\ -\frac{1}{A(t)}\amp = -kt+c\\ \end{aligned} \end{equation*}

Using the initial condition,

\begin{equation*} -\frac{1}{A(0)} = -\frac{1}{A_{0}} = -k(0) + c \end{equation*}

or \(c = -\frac{1}{A_{0}}\)
