Solution 1.8.2.1.
Since \(g(x)\) is a continuous function, we know that in general \(\lim\limits_{x\rightarrow a} g(x) = g(a)\text{.}\) So,
\begin{equation*} \lim\limits_{x\rightarrow -1} g(x) = g(-1) \end{equation*}
and since \(g(x) = 5x+6\text{,}\) we have
\begin{equation*} g(-1) = 5(-1)+6 = 1. \end{equation*}
We have successfully used function evaluation to replace the more challenging limit problem.
