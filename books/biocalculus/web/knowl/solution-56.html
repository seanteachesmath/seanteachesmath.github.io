Solution 3.4.2.1.

The value of the definite integral is \(\displaystyle\int_{0}^1 x-x^{2}\,dx = \dfrac{1}{6}\text{.}\) The average value is given by

\begin{equation*} \hat g = \dfrac{1}{1-0}\int_{0}^1 x-x^{2}\,dx = \dfrac{1}{1}\left(\dfrac{1}{6}\right) = \dfrac{1}{6} \end{equation*}

Graph this and think about the areas of the corresponding shapes.
