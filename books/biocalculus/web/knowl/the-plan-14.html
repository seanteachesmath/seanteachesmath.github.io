Notation for pure-time DEs.
In general, we study a pure-time differential equation \(\displaystyle \frac{dF}{dt} = f(t)\text{.}\)
Solving this pure-time DE for the unknown function \(F(t)\) involves finding the function whose derivative is \(f(t)\text{.}\) We want to ‘undo’ the derivative to find \(F(t)\text{,}\) called the antiderivative of \(f(t)\text{.}\)
Definition 3.1.2. antiderivative.
An antiderivative of the function \(f(t)\) is a function \(F(t)\) whose derivative equal to \(f(t)\text{.}\) We say, “\(F\) of \(t\) is equal to the integral of \(f\) of \(t\) with respect to \(t\text{,}\)” and we write
\begin{equation*} F(t) = \int f(t)\,dt. \end{equation*}
Warning.
The equation \(\displaystyle\frac{dp}{dt} = 1.8\) has solution
\begin{equation*} p(t) = \int 1.8\,dt = 1.8t+c. \end{equation*}

Definition 3.1.3. indefinite integral.
The set of all antiderivatives of the function \(f(t)\) is called the indefinite integral of \(f\) and is written
\begin{equation*} \int f(t)\,dt = F(t) + c. \end{equation*}

Theorem 3.1.9. Constant product rule for integrals.
Suppose that \(\displaystyle \int f(x)\,dx=F(x)+c\text{,}\) then for any constant \(a\text{,}\)
\begin{equation*} \boxed{\int af(x)\,dx=a\int f(x)\,dx= aF(x)+c} \end{equation*}
