Functions approaching \(\infty\text{,}\) ordered by speed.
Below we consider constants \(a, \beta, n > 0\) to help build larger categories of functions.
Suppose \(\displaystyle \lim_{x\rightarrow\infty}f(x) = \infty \text{ and }\lim_{x\rightarrow\infty}g(x) = \infty.\)
  1. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) faster than \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = \infty\)
    Ex.
  2. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) slower than \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = 0\)
    Ex.
  3. \(f(x)\) approaches infinity (or \(f(x)\rightarrow\infty\)) at the same rate as \(g(x)\) if \(\displaystyle \lim_{x\rightarrow\infty}\frac{f(x)}{g(x)} = \text{nonzero constant}\)
    Ex.
