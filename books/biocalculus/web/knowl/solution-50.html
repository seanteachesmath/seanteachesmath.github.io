Solution 3.3.3.1.

Since \(f(x) = x-x^{2}\) is continuous on the interval from \(x=0\) to \(x=1\text{,}\) the definite integral \(\displaystyle\int_{0}^1 x-x^{2}\,dx\) evaluates to \(\displaystyle\int_{0}^1 x-x^{2}\,dx = \left.\left(\dfrac{1}{2}x^{2} - \dfrac{1}{3}x^{3}\right)\right|_{0}^1 = \left(\dfrac{1}{2} - \dfrac{1}{3}\right) - 0 = \dfrac{1}{6}\text{.}\) Recall that \(0^{2}=0^{3}=0\) and \(1^{2} = 1^{3} = 1\text{.}\)
