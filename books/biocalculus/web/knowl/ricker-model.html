Ricker Model.
Originally published in the 1950â€™s, the Ricker model was used to describe change in size of fish populations from one measurement to the next. Though there are other variants presented in literature, the model is
\begin{equation*} N_{t+1} = rN_te^{-KNt}, \end{equation*}
where the values \(r\) and \(K\) are parameters. This model has been used to
  • estimate growth and population parameters from data
  • predict population growth
  • optimize yields from harvested populations
