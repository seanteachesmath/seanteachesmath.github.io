Exercise 4.

Choose the type of function from the dropdown list (choices are "constant", "linear", "quadratic", "cubic", or "something else") whose characteristics match those of the leading behavior.
  1. As \(t\to\infty\text{,}\) the leading behavior of the function \(f(t) = \dfrac{3.4t^3}{1+2.4t^2}\text{,}\) given by \(f_\infty(t)\text{,}\) looks like \(\underline{\qquad\qquad}\) function.
  2. Near \(t=0\text{,}\) the leading behavior of the function \(f(t) = \dfrac{3.4t^2}{1+2.4t^3}\text{,}\) given by \(f_0(t)\text{,}\) looks like \(\underline{\qquad\qquad}\) function.
  3. As \(t\to\infty\text{,}\) the leading behavior of the function \(f(t) = \dfrac{3.4t^2}{1+2.4t^3}\text{,}\) given by \(f_\infty(t)\text{,}\) looks like \(\underline{\qquad\qquad}\) function.
  4. Near \(t=0\text{,}\) the leading behavior of the function \(f(t) = \dfrac{3.4t}{1+2.4t}\text{,}\) given by \(f_0(t)\text{,}\) looks like \(\underline{\qquad\qquad}\) function.
  5. Near \(t=0\text{,}\) the leading behavior of the function \(f(t) = \dfrac{3.4t^3}{1+2.4t^2}\text{,}\) given by \(f_0(t)\text{,}\) looks like \(\underline{\qquad\qquad}\) function.
