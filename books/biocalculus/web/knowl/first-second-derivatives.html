Derivatives.

When discussing functions and derivatives, the derivative of a function refers to its first derivative. Since derivatives of functions are functions too, we can take ‘higher derivatives’ using our current methods. Perhaps the most common higher derivative is the second derivative.

Concavity describes the shape of a graph.
  • a graph of \(f(x)\) is concave up if it ‘bends upward,’
    • if its first derivative is increasing, or
    • if \(f''(x)>0\) (the second derivative is positive)
  • a graph of \(f(x)\) is concave down if it ‘bends downward,’
    • if its first derivative is decreasing, or
    • if \(f''(x)<0\) (the second derivative is negative)

Critical points and second derivatives.
For a function \(f(x)\text{,}\) a critical point is an \(x\)-value at which the derivative is zero or not defined. Here we focus on the kind of critical point where \(f'(x)=0\text{.}\)At such a point the function commonly experiences a change in behavior.
Derivatives and important points.
Derivatives encode important information about functions.
  • critical points (where \(f'(x)=0\)) often indicate where a function changes behavior
  • inflection points (possible where \(f''(x)=0\)) indicate where a function changes shape (concavity)

The Second Derivative Test outlined below helps is find local optima at critical points \(x=c\text{:}\)
  • If \(f''(c)<0\text{,}\) then a local maximum value of \(f(x)\)occurs at \(x=c\) and the function value is \(f(c)\)
  • If \(f''(c)>0\text{,}\) then a local minimum value of \(f(x)\)occurs at \(x=c\) and the function value is \(f(c)\)
  • If \(f''(c)=0\text{,}\) then we have to look moreclosely at the sign of the first derivative

A global optimum value is guaranteed to exist for a function on an interval. The global maximum is the largest value that the function achieves, the global minimum is the smallest value that the function achieves.Global extrema occur at critical points or endpoints of the interval.
The steps are as follows:
  1. Evaluate function at endpoints.
  2. Find all critical points.
  3. Compute the function value at critical points.
  4. Largest is the global maximum, smallest is the global minimum.

Biological and scientific applications.
In general, we might want to optimize production of a chemical, yield or harvest of a plant or animal product, reproductive output, survival, growth rate. Statistics also relies heavily on the derivative and maximization to find ‘best’ values of a statistical parameter from data.
