Extra DTDS examples.
The following completely nonbiological examples illustrate some interesting mathematical behaviors. These will help us understand behaviors of more complicated biological models next time.
Each day a patient uses up half of the medication in his bloodstream, but he is also given a new dose that raises the concentration in the blood stream by 1 mg/L. Let \(M_t\) denote the concentration on day \(t\text{.}\)
  • Write a DTDS describing this situation and plot the updating function.
  • What does the solution look like if you start with \(M_{0}\) = 5 mg/L? \(M_{0}\) = 1 mg/L?
Plot the diagonal and updating function for the previous model and cobweb five terms of the solution starting from \(M_{0}\) = 5 mg/L and then from \(M_{0}\) = 1 mg/L.
What happens if you start at \(M_{0}=2\text{?}\)
A point \(m^*\) is called an equilibrium of the DTDS
\begin{equation*} m_{t+1} = f(m_t) \end{equation*}
if \(f(m^*) = m^*\) (i.e., if updating by the DTDS leaves the input \(m^*\) unchanged).
  • Find equilibria (more than 1 equilibrium) graphically by locating the points where the graphs of the updating function and diagonal intersect.
  • Find equilibria algebraically by setting \(f(m^*) = m^*\) and solving for \(m^*\) .
Compute equilibrium values for the following DTDS.
