Concept of a function:.

Functions give us a compact way to describe how one measurement (the ‘output’) depends on another (the ‘input’):

Describing a function:.

We can describe functions in four ways:

  • in words

  • in a graph

  • as data (a table)

  • as a formula

Combining functions:.

We can combine functions by

  • addition and subtraction

  • multiplication or division (if the denominator is nonzero)

  • composition

Consider two functions \(F(x)=3x^{2}+2x\) and \(H(x)=-x+1\text{.}\)

  1. adding and subtracting: “\(F\) plus/minus \(H\) of \(x\)”

    • \(\displaystyle (F + H)(x) = F(x) + H(x) =\)

    • \(\displaystyle (F - H)(x) = F(x) - H(x) =\)

    • \(\displaystyle (H - F)(x) = H(x) - F(x) =\)

  2. multiplying and dividing: “\(F\) multiplied/divided by \(H\) of \(x\)”

    • \(\displaystyle (F \cdot H)(x) = F(x) \cdot H(x) =\)

    • \(\displaystyle (F / H)(x) = F(x) / H(x) =\)

    • \(\displaystyle (H / F)(x) = H(x) / F(x) =\)

  3. composition: : “\(F\) of \(H\) of \(x\)” or “\(H\) of \(F\) of \(x\)”

    • \(\displaystyle (F\circ H)(x) = F(H(x)) =\)

    • \(\displaystyle (H\circ F)(x) = H(F(x)) =\)

    • Are \((H\circ F)(x)\) and \((F\circ H)(x)\) the same?

Note (composition of functions).

Consider two functions \(f(x)\) and \(g(x)\text{.}\) In the composition \(f(g(x))\text{,}\) \(g(x)\) is the ‘inner function’ and \(f(x)\) is the ‘outer function.’ Here, \(g\) is applied to \(x\) first, then \(f\) is applied to the result.

Consider the following composed functions. For each identify the pair of inner and outer functions involved in defining the composition.

  • \(f(x) = (7x+1)^{4}\) is composition of functions \(G(H(x))\text{,}\) define the functions \(G(H)\) and \(H(x)\text{.}\)

  • \(\displaystyle a(t) = \frac{5}{3t^{2}+t}\) is composition of functions \(b(c(t))\text{,}\) define the functions \(b(c)\) and \(c(t)\text{.}\)

  • \(w(x) = (e^x+2x)^{2}+1\) is composition of functions \(u(v(x))\text{,}\) define the functions \(u(v)\) and \(v(x)\text{.}\)

Inverse functions.

Functions allow us to compute output for a given input. Sometimes it is useful to know from which input we arrived a particular output. We can use output to find the input by using the inverse of a function.In general, we would write that the function \(f^{-1}\) is the inverse of \(f\text{,}\) meaning

\begin{equation*} f^{-1}(f(x))=x\text{ \,and\, }f(f^{-1}(x)) = x. \end{equation*}

We will make use of this when solving a variety of equations involving exponential functions.

Exercises Quick practice

1.

You measure cytotoxicity of three different concentrations of a cancer drug over time. Identify the variables and parameters.

Solution.

DV: cytoxicity (number of cells killed), IV: time (probably in hours), parameter: the concentration as varied between trials.

2.

How would you graph the results obtained in Question Exercise 1.2.1?

Solution.

It would be interesting to see cytotoxicity (number of cells killed) plotted against time with sets of points or curves colored by the drug concentration used.

3.

Simplify \(\dfrac{1}{5}+\dfrac{5}{x}\text{.}\)

Solution.

We need a common denominator to add, so \(\dfrac{1}{5}+\dfrac{5}{x} = \dfrac{1}{5}\cdot\dfrac{x}{x}+\dfrac{5}{x}\cdot\dfrac{5}{5} = \dfrac{x}{5x}+\dfrac{25}{5x} = \dfrac{x+25}{5x}\text{.}\)

4.

For \(f(x) = 3x^{2}+2x\) evaluate \(f(1)\text{,}\) \(f(2)\text{,}\) and \(f(x+2)\text{.}\)

Solution.

For \(f(x) = 3x^{2}+2x\) the values are

  • \(f(1) = 3(1)^{2}+2(1) = 5\text{,}\)

  • \(f(2) = 3(2)^{2}+2(2) = 16\text{,}\) and

  • \(f(x+2) = 3(x+2)^{2}+2(x+2)\text{.}\)

For the last of these, some simplification is still required to show that the final answer simplifies to \(f(x+2) = 3x^{2}+14x+16\text{.}\)
