Section 16 Introduction to differential equations

¶

We introduce a new category of calculus problems called integrals.

Subsection 16.1 Applications of derivatives: differential equations

¶

So far we have built methods and skills that allow us to take derivatives of known functions. When we revisit derivatives at the end of the semester, we will see quite a few more applications of derivatives. For now switch perspective and build systematic methods that allow us figure out what original function that a particular derivative comes from.

Methods. To find what original function a derivative comes from, we might

  1. guess the original function
  2. graph the original function (from behaviors encoded by the derivative)
  3. approximate the graph of the original function (Eulers method/tangent line approximations) skipped
  4. integrate to find the original function
Definition 16.1.

A differential equation (often shortened DE) describes the instantaneous rate of change of a measurement.

  • a pure-time DE is of the form \(\dfrac{dp}{dt} = f(t)\) where the instantaneous rate of change of \(p(t)\) is measured as a function of time \(f(t)\text{.}\)

  • an autonomous DE is of the form \(\dfrac{dp}{dt} = f(p)\) where the instantaneous rate of change of \(p(t)\) is described as a function of the measurement itself \(f(p)\text{.}\)

Example 16.2. Classifying DEs.

Identify whether each of the following is a pure-time or autonomous differential equation for the given variable:

  1. \(\dfrac{dp}{dt} = e^{-t}\)

  2. \(\dfrac{dn}{dt} = 2n\)

  3. \(\dfrac{dx}{dt} = 2x\Big(1-\dfrac{x}{10}\Big)\)

Solution

  1. \(\dfrac{dp}{dt} = e^{-t}\) is a pure-time DE for the variable \(p\)

  2. \(\dfrac{dn}{dt} = 2n\) is an autonomous DE for the variable \(n\)

  3. \(\dfrac{dx}{dt} = 2x\Big(1-\dfrac{x}{10}\Big)\) is an autonomous DE for the variable \(x\)

Example 16.3. Velocity during a walk.

Velocity, the instantaneous change in position, of a moving person is given by the pure-time DE \(\dfrac{dp}{dt} = 1.8\text{,}\) where position \(p(t)\) is the quantity being differentiated with respect to time. Find the solution \(p(t)\text{,}\) that satisfies the differential equation.

We have to options: Guess it, Graph it.

Solution

Guessing involves thinking of what original function has a derivative of 1.8, based on our experience of taking derivatives. If our derivative is the constant function \(1.8\text{,}\) what was the original function \(p(t)\text{?}\) In other words, "the derivative of what is \(1.8\text{?}\)"

Remark 16.4. Notation for pure-time DEs.

In general, we study a pure-time differential equation \(\dfrac{dF}{dt} = f(t)\text{.}\) Solving this pure-time DE for the unknown function \(F(t)\) involves finding the function whose derivative is \(f(t)\text{.}\) We want to undo the derivative to find \(F(t)\text{,}\) called the antiderivative of \(f(t)\text{.}\)

Definition 16.5.

An antiderivative of the function \(f(t)\) is a function \(F(t)\) whose derivative equal to \(f(t)\text{.}\) We say, ""\(F\) of \(t\) is equal to the integral of \(f\) of \(t\) with respect to \(t\)"", and we write

\begin{equation*} F(t) = \int f(t) \, dt \end{equation*}
Example 16.6.

The equation \(\dfrac{dp}{dt} = 1.8\) has solution

\begin{equation*} p(t) =\int 1.8\,dt = 1.8t + c \end{equation*}

What is c?

Definition 16.7.

The set of all antiderivatives of the function \(f(t)\) is called the indefinite integral of \(f\) and is written

\begin{equation*} \int f(t)\,dt = F(t) + c \end{equation*}
Example 16.8.

Solve pure-time DE

\begin{equation*} \dfrac{dF}{dt} = 2 \end{equation*}

by computing the indefinite integral.

Example 16.9.

Solve pure-time DE

\begin{equation*} \dfrac{dF}{dt} = 2t \end{equation*}

by computing the indefinite integral.

Theorem 16.10. Power rule for integrals.

Suppose that \(f(x) = x^n\) with \(n\ne 1\text{,}\) then

\begin{equation*} \int x^n\,dx = \left(\dfrac{1}{n+1}\right)x^{n + 1} + c \end{equation*}
Example 16.11.

Compute \(\displaystyle \int x^2\,dx\text{.}\)

Example 16.12.

Compute the indefinite integral of \(g(x) = x^{0.4}\text{.}\)

Example 16.13.

Compute \(\displaystyle \int t^5\,dt\text{.}\)

Example 16.14.

Compute the indefinite integral of \(f(t) = \dfrac{1}{t^4}\text{.}\)

Theorem 16.15. Constant product rule for integrals.

Suppose that \(\int f(x)\, dx = F(x) + c\text{,}\) then for any constant \(a\text{,}\)

\begin{equation*} \int af(x)\,dx = a \int f(x)\,dx=aF(x)+c \end{equation*}
Example 16.16.

Compute \(\displaystyle \int 12t^5\,dt\text{.}\)

Example 16.17.

Compute \(\displaystyle \int \dfrac{3}{t^4}\,dt\text{.}\)

Theorem 16.18. Sum rule for integrals.

Suppose that \(\int f(x)\, dx = F(x) + c\) and \(\int g(x)\, dx = G(x) + c\text{,}\) then

\begin{equation*} \int f(x) + g(x)\, dx = \int f(x)\, dx + \int g(x)\, dx = F(x) + G(x) + c \end{equation*}
Example 16.19.

Compute \(\displaystyle \int 12t^5 + \dfrac{3}{t^4}\,dt\text{.}\)

Example 16.20.

Consider a seed in a flower in a very tall tree. From an initial height of \(14.7\) meters, the seed pops straight into the air with an initial velocity of \(9.8 \text{meters per second}\) \(v(0) = 9.8 \text{m}\) and an acceleration due to gravity of \(a(t) = 9.8 \text{m}\) (in the downward direction). Find the velocity function \(v(t)\) and use that to find its position function \(s(t)\) of the seed. Then address the following questions

  1. When will the seed land?

  2. What is the velocity of the seed when it smashes into the ground?
