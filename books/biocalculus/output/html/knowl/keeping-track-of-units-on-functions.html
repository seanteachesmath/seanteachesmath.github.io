Keeping track of units on functions.

When appropriate we want to make good use of the benefits of function notation and function manipulation, while preserving the unit of the underlying measurement. In particular, we want to be able to use given information to determine when we are evaluating a function (units of the input) or solving an equation (units of the output).

Consider two mathematical functions used to represent the change in these quantities, \(r(t) = \textit{radius of a colony as a function of time}\) and \(A(r) = \textit{area of a colony as a function of radius}\text{.}\)

  • What are dimensions and sensible units for each?

  • How can we combine these functions (measurements) to define other realistic measurements?
