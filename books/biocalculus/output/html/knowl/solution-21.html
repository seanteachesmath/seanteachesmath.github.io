Solution 1.7.3.1.

It might help you to make a table of values (see Example 1.7.13) values are

  1. \(\displaystyle \lim_{x\to0^-}\dfrac{1}{x^{3}} = -\infty\text{,}\)

  2. \(\displaystyle \lim_{x\to0^+}\dfrac{1}{x^{3}} = +\infty\text{,}\) and finally

  3. since the previous two answers for one-sided limits do not match, the two-sided limit \(\displaystyle \lim_{x\to0}\dfrac{1}{x^{3}}\) does not exist, written in shorthand as \(\displaystyle \lim_{x\to0}\dfrac{1}{x^{3}} = \text{DNE}\) or \(\displaystyle \lim_{x\to0}\dfrac{1}{x^{3}}\text{ DNE}\)
