Section 3.1 Introduction to differential equations

The Plan.

We introduce a new category of calculus problems called integrals.

Applications of derivatives: differential equations.

So far we have built methods and skills that allow us to take derivatives of known functions. When we revisit derivatives at the end of the semester, we will see quite a few more applications of derivatives. For now switch perspective and build systematic methods that allow us figure out what original function that a particular derivative ‘comes from.’

Methods.

To find what original function a derivative ‘comes from,’ we might

  1. guess the original function

  2. graph the original function (from behaviors encoded by the derivative)

  3. approximate the graph of the original function (Euler’s method/tangent line approximations) – skipped

  4. integrate to find the original function

A differential equation (often shortened ‘DE’) describes the instantaneous rate of change of a measurement.

\({}^\dagger\) we will return to autonomous DEs at the end of the chapter

Velocity, the instantaneous change in position, of a moving person is given by the pure-time DE

\begin{equation*} \frac{dp}{dt} = 1.8, \end{equation*}

where position \(p(t)\) is the quantity being differentiated with respect to time. Find the solution \(p(t)\text{,}\) that is, find the function whose derivative is \(\displaystyle\frac{dp}{dt} = 1.8\text{.}\)

Notation for pure-time DEs.

In general, we study a pure-time differential equation \(\displaystyle \frac{dF}{dt} = f(t)\text{.}\)

Solving this pure-time DE for the unknown function \(F(t)\) involves finding the function whose derivative is \(f(t)\text{.}\) We want to ‘undo’ the derivative to find \(F(t)\text{,}\) called the antiderivative of \(f(t)\text{.}\)

Definition 3.1.2. antiderivative.

An antiderivative of the function \(f(t)\) is a function \(F(t)\) whose derivative equal to \(f(t)\text{.}\) We say, “\(F\) of \(t\) is equal to the integral of \(f\) of \(t\) with respect to \(t\text{,}\)” and we write

\begin{equation*} F(t) = \int f(t)\,dt. \end{equation*}
Warning.

The equation \(\displaystyle\frac{dp}{dt} = 1.8\) has solution

\begin{equation*} p(t) = \int 1.8\,dt = 1.8t+c. \end{equation*}

What is \(c\text{?}\)

Definition 3.1.3. indefinite integral.

The set of all antiderivatives of the function \(f(t)\) is called the indefinite integral of \(f\) and is written

\begin{equation*} \int f(t)\,dt = F(t) + c. \end{equation*}

Solve pure-time DE

\begin{equation*} \frac{dF}{dt} = 2 \end{equation*}

by computing the indefinite integral.

Solve pure-time DE

\begin{equation*} \frac{dF}{dt} = 2t \end{equation*}

by computing the indefinite integral.

Suppose that \(f(x) = x^n\) with \(n\neq-1\text{,}\) then

\begin{equation*} \boxed{\int x^n\,dx=\left(\frac{1}{n+1}\right)x^{n+1}+c} \end{equation*}

Theorem 3.1.9. Constant product rule for integrals.

Suppose that \(\displaystyle \int f(x)\,dx=F(x)+c\text{,}\) then for any constant \(a\text{,}\)

\begin{equation*} \boxed{\int af(x)\,dx=a\int f(x)\,dx= aF(x)+c} \end{equation*}

\begin{equation*} \boxed{\int f(x)+g(x)\,dx=\int f(x)\,dx +\int g(x)\,dx= F(x)+G(x)+c} \end{equation*}

Exercises 3.1.1 Quick practice

1.

Is \(\dfrac{dq}{dt} = 2.1q(1-q)\) a pure-time or autonomous differential equation?

Solution

In \(\dfrac{dq}{dt} = 2.1q(1-q)\text{,}\) the unknown quantity that is being differentiated, \(q = q(t)\text{,}\) appears on the right-hand side, and does so in the absence of the variable \(t\) on the right-hand side, this is indicative of an autonomous equation.

2.

Compute \(\displaystyle\int 3x+2\,dx\text{.}\)

Solution

The indefinite integral \(\displaystyle\int 3x+2\,dx\) evaluates to \(\displaystyle\int 3x+2\,dx = \dfrac{3}{2}x^{2}+2x+c\text{.}\)

3.

What quantity is obtained when a pure-time growth rate indicating a rate of change of length is integrated?

Solution

Integrating the rate of change of length gives length itself as a function of time.

4.

Compute \(\displaystyle\int 0\,dx\text{.}\)

Solution

The indefinite integral \(\displaystyle\int 0\,dx\) evaluates to \(\displaystyle\int 0\,dx = c\text{.}\)
