Consider the following laws for for real numbers \(a\text{,}\) \(m\text{,}\) and \(n\text{.}\)

  1. \(\displaystyle a^{m}\cdot a^{n} = a^{m+n}\)

  2. \(\displaystyle (a^n)^m = a^{nm}\)

  3. \(\displaystyle a^{-n} = \frac{1}{a^{n}}\)

  4. \(\displaystyle \frac{a^n}{a^m} = a^{n-m}\)

  5. \(\displaystyle a^1 = a\)

  6. \(\displaystyle a^0 = 1\)
