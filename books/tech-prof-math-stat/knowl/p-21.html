Paragraph

We began by talking about inserting graphics into a LaTeX document. We downloaded a graph of our choosing from the shared drive, and placed it in the same folder as our .tex files, and then learned how to add it to our document using the package graphicx as well as how to change the width (by adding [width=x] to the includegraphics command) of the graphic. We then learned how to make captions and reference it within a document using the caption... command and the label... and ref... commands.. Finally, we learned how to make tables inside of a document to show off information using the command begintable , such as explaining the types of discontinuity in the figure we added. We learned to justify the tables to the right, left, or center by adding a r, l, or c to the top of the table, respectively. We also learned the centering command to center graphics added to our documents.
