BibTeX a a tremendously powerful and useful bibliography processing system. It also has auxiliary programs (e.g., BibDesk) that are very useful for reference/bibliography management. With no special commands we can use the \cite{citekey} command to reference a item by its citekey. The citekey is the code associated with the bibliographic information given in the .bib file. The file is specified by the command \bibliography{filename.bib} and the style is given by the command \bibliographystyle{name}. Reasonable choices for bibliography styles are

  1. plain - reference list sorted alphabetically, in-text reference refers to numerical order

  2. unsrt - reference list sorted in order of in-text referencing

  3. alpha - reference list sorted alphabetically, in-text reference refers to unique, abbreviated author-year code
