Today we learned how to create new commands using preexisting ones. We learned how to create matrices and vectors. We learned how to type up a piecewise defined term, such as the absolute value of \(x\text{.}\) We learned how some commands automatically treat the input into those commands as ‘math’ and others as ‘stuff’ that we have to define as ‘math.’ This was done in continuation of the slide show which was started on Tuesday.
