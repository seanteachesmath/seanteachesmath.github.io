We have \(f_\text{tan}(x) = f(x_0) + f'(x_0)(x-x_0)\text{,}\) and letting \(x_1\) be the approximate solution to this equation, set \(f_\text{tan}(x_1) = 0\text{,}\) which gives

\begin{equation*} \begin{aligned}[t] 0 \amp = f(x_0) + f'(x_0)(x_1 - x_0)\\ -f'(x_0)(x_1 - x_0) \amp = f(x_0)\\ (x_1 - x_0) \amp = \frac{f(x_0)}{-f'(x_0)}\\ x_1 \amp = x_0 - \frac{f(x_0)}{f'(x_0)}\\ \end{aligned} \end{equation*}

Repeating this from \(x_1\) to generate a value for \(x_2\) allows us to generalize the rule

\begin{equation*} x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \end{equation*}

Applying this method gives a sequence of values that most often approach a solution to the original equation.

Algorithm 3.3.1. R code for approximating solutions to \(f(x) = 0\) by Newton's Method. 
x0 <- 1
xs <- x0
for(i in 1:n){
  x1 <- x0 - f(x0)/fp(x0)     ## calculate new value
  xs <- c(xs, x1)   ## store the new number
  x0 <- x1          ## replace the old number by the new number, repeat
}
