Given that we spent some time on sequences and functions, we can use the command sapply(object, function) INDEX R , built-in , sapply to apply a given function to an object. In this case our object was a series of \(x\) values, and our function was a mathematical function of \(x\text{.}\) This result gives the function values evaluated the points in the list of \(x\) values. Such a command might be useful to plot values of a function at a set of points. In fact, we plotted these function values and added a legend INDEX R , graphics , legend to the graph.
