Section 1.3 Environments: Figure and Table

Subsection 1.3.1 Preview

We will continue learning about some other ‘environments’ in LaTeX to achieve particular formats for our written content.

Subsection 1.3.2 Review

Today we introduced the first homework assignment in which you will write about the limit definition of the derivative.

We learned includegraphics and the beginfigure[htbp]...endfigure

environment, we adjusted the size and spacing of the image itself and its positioning (centering) within the figure. The figure options htbp refer to ‘here’, ‘top’, ‘bottom’, or a new ‘page’ and control where a figure (or table) is placed relative to the text that surrounds it. Based on the particular preferences of a given computer we sometimes include or do not include the file extension in includegraphics..., so learn what works for you on a given computer and stick with it. If you get ‘File not found’ errors it is because you either spelled the file name wrong, added or omitted a file extension, or forgot to include the file in the same directory as your .tex file.

Much like includegraphics was nested within beginfigure[htbp]...endfigure, we ended with a brief look at begintabular...endtabular which we nested inside begintable[htbp]...endtable to achieve the proper referencing and labeling. Within each of these we introduced the caption command caption... and the label command labeltype::... . The labeltype::... will prove useful when combined with the referencing command reftype::... to help us count references to a given float type by their name. This means we never write a text reference to a figure as Figure 1 or Fig. 1 we will always instead use Figure [fig::...] where ... refers to some meaningful text label applied to the figure. Above,   is the command for a nonbreaking space, this prints a space between the text ‘Figure’ and the reference number, but will not allow that space to break across lines or pages. For each of these we use centering INDEX LaTeX , positioning , ‘centering’ to display the object centered in the page.

To display graphics we needed to add usepackagegraphicx INDEX LaTeX , packages , graphicx in the preamble. Additionally, we introduced usepackagegeometry INDEX LaTeX , packages , geometry in the preamble to adjust margins.

Subsection 1.3.3 Student perspective

We began by talking about inserting graphics into a LaTeX document. We downloaded a graph of our choosing from the shared drive, and placed it in the same folder as our .tex files, and then learned how to add it to our document using the package graphicx as well as how to change the width (by adding [width=x] to the includegraphics command) of the graphic. We then learned how to make captions and reference it within a document using the caption... command and the label... and ref... commands.. Finally, we learned how to make tables inside of a document to show off information using the command begintable , such as explaining the types of discontinuity in the figure we added. We learned to justify the tables to the right, left, or center by adding a r, l, or c to the top of the table, respectively. We also learned the centering command to center graphics added to our documents.
