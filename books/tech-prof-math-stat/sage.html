Subsection 4.7 Miscellaneous programming (11/10)

Subsubsection 4.7.1 Preview

Programming DAGGER

Subsubsection 4.7.2 Review

We looked at programming in sage with if-else statements, and defining a program to define a piecewise calculation. Since we were not able to graph the object of our program, we computed and plotted points. It turned out there was a function called Piecewise(...) that took a rather complicated argument. With a bit of debugging, we were able to plot and differentiate the function.

We also briefly explored programming special functions like nChooseR() to compute the combinations of objects and compared this to binomial coefficients.

Subsubsection 4.7.3 Student perspective

We first learned how to use if-else commands. And we tried using a series of if-else at the same time. Then we learnt piecewise functions and learnt to print a series of numbers using \(f(i)\) and srange. We had trouble differentiating a function because we used diff(f,x) instead of only diff(f). This was because we previously defined function f, therefore we can differentiate by only using diff(f).Next we learnt how to compute permutations on sage. We first defined nChooseR and then print the answers according to the definition. We also learnt using [0..2] which means number 0 to 2, instead of typing all numbers out. Finally we computed the Pascal’s triangle by using the nChooseR function and where \(n = 10\) and ‘R = 0 to 10’ or [0..10].
