In a similar fashion, we can quickly calculate and store Fibonacci numbers. This is done by calculating terms according to \(x_{n+2} = x_{n+1} + x_{n}\) where \(x_{2} = x_{1} = 1\) (some, more modern, authors begin with \(x_{0} = 0\)). Consequently, the first few terms are

\begin{equation*} 1, 1, 2, 3, 5, 8, 13, \dots. \end{equation*}

Algorithm 3.3.2. R code for calculating Fibonacci numbers. 
x0 <- 1
x1 <- 1
xs <- c(x0, x1)
for(i in 1:n){
  x2 <- x0 + x1     ## calculate the sum
  xs <- c(xs, x2)   ## store the new number
  x0 <- x1          ## replace the oldest number by the second oldest
  x1 <- x2          ## replace the second oldest by the newest, repeat
}
