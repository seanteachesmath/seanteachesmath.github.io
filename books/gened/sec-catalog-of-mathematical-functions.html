Subsection 2.7.1 Linear-functions

¶

...

Linear functions are those of the form \(y = f(x) = mx+b\text{,}\) we call \(m\) the slope and \(b\) the \(y\)-coordinate of the \(y\)-intercept. We can solve equations of evaluate functions. Graphs of linear functions are straight lines. If the slope is positive, the line increases from left to right (bigger \(x\) results in bigger \(y\)). If the slope is negative, the line decreases from left to right (bigger \(x\) results in smaller \(y\)). If the slope is zero, the line is flat or constant. Linear functions are often used in regression to determine the slope and intercept of a line that best summarize information in a scatterplot of points.

Example 2.7.1. ....

A popular example of a linear function is

\begin{equation*} F = \dfrac{9}{5}C + 32 \end{equation*}

which describes the temperature in degrees Fahrenheit as a function of temperature in degrees Celsius. We might remember that the freezing point of water is \(32 {}^\circ\text{F}\) and that the boiling point is \(212 {}^\circ\text{F}\text{.}\) We might be interested in representing these values in Celsius, and can pursue this by solving each of the following equations for \(C\text{,}\)

\begin{equation*} \begin{aligned}[t] 32 \amp = \dfrac{9}{5}C+32\\ 212 \amp = \dfrac{9}{5}C+32 \end{aligned} \end{equation*}

To solve each, we subtract \(32\) from each side and then multiply by \(\dfrac{5}{9}\) to find that the freezing point of water os \(0 {}^\circ\text{C}\) and the bioling point is \(100 {}^\circ\text{C}\text{.}\) For example,

\begin{equation*} \begin{aligned}[t] 32 \amp = \dfrac{9}{5}C+32\\ \cancel{32-32} \amp = \dfrac{9}{5}C+\cancel{32 - 32}\\ 0 \amp = \dfrac{9}{5}C\\ \dfrac{5}{9}\cdot0 \amp = \cancel{\dfrac{5}{9}\cdot\dfrac{9}{5}}C\\ C \amp = 0 \end{aligned} \end{equation*}
Checkpoint 2.7.2.
Repeat the calculation to confirm the boiling point of water in Celsius.
Example 2.7.3. ....

Consider you are taking a \sout{taxi} Lime scooter ride. The \sout{taxi"> Lime scooter charges a \(\$1.00\) `unlock fee' then charges an additional \(\$0.29\) per minute. Write the linear function that describes cost \(c\) as function of time spend riding \(t\text{.}\)

Solution
We can imagine in words that the cost is equal to the initial charge plus the cost per minute multiplied by the number of minutes. This is
\begin{equation*} c = c(t) = (\text{initial charge}) + (\text{cost per minute})(\text{time spent riding}) \end{equation*}
or
\begin{equation*} c = c(t) = 1.00 + 0.29t \end{equation*}

We can ask our usual sorts of questions about this function. For example ''What if we made the poor decision to skip class and go on a 75-minute camps joyride? How much would this cost?''

Example 2.7.4. ....

''What if we made the poor decision to skip class and go on a 75-minute campus joyride instead? How much would this cost?''

Solution
To answer this question, we evaluate \(c(75)\) to find the associated cost.
\begin{equation*} \begin{aligned}[t] c(75) \amp = 1.00 + 0.29(75)\\ \amp= 22.75 \end{aligned} \end{equation*}

Or, ''How long could I ride if I had a \$5 credit?''

Example 2.7.5. ....

''How long could I ride if I had a \$5 credit?''

Solution
To answer this question, we solve \(c(t) = 5\) for the unknown value of \(t\text{.}\)
\begin{equation*} \begin{aligned}[t] %5 \amp = c(t) \\ 5 \amp= 1.00 + 0.29t\\ 4.00 \amp= 0.29t\\ t\amp= \dfrac{4.00}{0.29} = 13.79\\ \end{aligned} \end{equation*}

Subsection 2.7.2 Quadratic functions

¶

...

The simplest quadratic function is \(f(x) = x^2\text{,}\) whose graph is a `u'-shaped parabola (as opposed to `n'-shaped). This term is an example of a power function, built from a variable raised to a number as a power,

\begin{equation*} x^2 \text{is of the form} (\text{variable})^\text{number}. \end{equation*}

This form has important consequences for the properties of the formula and its use in applications. With similar terms we can construct polynomial functions such as,

\begin{equation*} g(x) = 3x^2+x^4 \end{equation*}

which are sums of constants multiplied by powers of a variable.

Quadratic functions show up frequently in applications, for example in physics. We often study problems such as \(s(t) = -4.9t^2 + 10t + 12\) where \(s(t)\) represents the height of an object thrown from a height of \(12 \text{meters}\) with an initial velocity of \(10 \text{meters per second}\text{.}\) Because the number in front of \(t^2\) is negative, the graph of the corresponding parabola opens downward (as an upside down `u'). There are a few ways that we can learn a little more about the graph to determine the position of the curve within the \(t-s\) plane (that is, in a plot of \(s\) against \(t\)).

Notice that when \(t = 0\text{,}\) \(s(0) = -4.9(0)^2 + 10(0) + 12 = 10\text{;}\) the graph of \(s(t)\) crosses the vertical axis at a value of \(12\text{.}\) But given the shape of the curve (an upside-down parabola), does the graph intersect the point \((0, 12)\) on its downstroke, at its vertex, or on its upstroke? There are a number of ways to answer this question

  1. `cheat' and graph the formula using a computer (e.g., with Desmos online, or even Google!) or graphing calculator

  2. put the function in vertex form

  3. apply methods of calculus

  4. explore the formula using a hodge-podge of approaches

We will take the last of these approaches. A question is (not unrelated to how we might reason about functions using calculus or even more sophisticated approaches) is ''what does the graph look like when \(t\) is ''small''?'' Here we mean \(t\approx0\) to give an estimate of what we consider to be ''small'' values of \(t\text{.}\) Here thinking numerically about the relative contribution of each term gives some indication of the value of the function in general. The values of each term and the exact and approximate values of each function are given in Table \ref{tab::position}.

The results in Tab. \ref{tab::position} suggests that the height is initially increasing (by the exact values, or by the approximation). The approximation, neglecting the contribution of the \(t^2\) term for small values of \(t\) suggests that the position initially increases in an approximately linear fashion. At some point though, the quadratic term starts to play a role. It seems quite strange that \(s(10) = -378\text{,}\) this would suggest that the object is \(378 \text{meters}\) below the surface of the ground. It should be the case that at some point prior, the model stops making sense. In fact, we could use the quadratic formula to determine when the object lands. Solving for \(s(t) = 0\) gives the value(s) of time \(t\) for which the height from the ground is zero.

\begin{equation*} \begin{aligned}[t] x \amp= \dfrac{-10\pm\sqrt{10^2 - 4(-4.9)(12)}}{2(-4.9)}\\ \amp = \dfrac{-10\pm\sqrt{335.2}}{-9.8}\\ x \amp = -0.847,\,\,2.889\\ \end{aligned} \end{equation*}

The negative value does not make as much sense because this suggests that the object was on the ground \text{before} we threw it upwards - we are only really concerned with the future after \(t=0\text{.}\) From our hodge-podge analysis we have that the object starts from a height of \(12 \text{meters}\text{,}\) increases approximately linearly for a brief span, and with the shape of a parabola eventually comes back down and lands at \(t\approx2.899\text{.}\) The mathematical equation is valid for all time, but after this the equation as a mathematical object of the flight of the object stops making sense.

Subsection 2.7.3 Polynomials

¶

...

As mentioned in the previous section,

\begin{equation*} g(x) = 3x^2+x^4 \end{equation*}

is an example of a polynomial. This category can be written as a sum of multiples of powers of a variable, for example

\begin{equation*} p_n(x) = \sum_{i=0}^n a_i x_i \end{equation*}

where in the example for \(g(x)\) we would have \(n=4\) with \(a_0 = a_1 = a_3 = 0\text{,}\) and \(a_2 = 3\) and \(a_4 = 1\text{.}\) In summation form,

\begin{equation*} g(x) = \sum_{i=0}^4 a_i x^i = 0x^0 + 0x^1 + 3x^2 + 0x^3 + 1x^4 = 3x^2 + x^4 \end{equation*}

Applications of polynomials are numerous, this stems partly from the fact that they are `easy' to evaluate. The word `easy' refers to the fact that there might be many, many terms to keep track of, but that it only requires elementary operations of addition or subtraction and multiplication. As a consequence, polynomials play an important role in how we evaluate most functions using calculators or computers.

As a few examples of polynomials, consider \(f(x) = x^3\) and \(g(x) = x^3-x\text{.}\) In the second of these, the subtraction of \(x\) has a dramatic impact on the graph (see Fig. \ref{fig::cubics}).

Side-by-side, the graph of \(f(x) = x^3\) is relatively less interesting than that of \(g(x) = x^3 - x\text{.}\) Given a sketch, it appears that \(x^3\) only has one zero, while \(x^3-x\) has three zeroes. In general it isn't as easy to use something like the quadratic formula for solutions to polynomial equations with higher powers. Here however, we get a bit lucky and can do some factoring of \(x^3-x\text{.}\) It turns out that in solving this, we can write

\begin{equation*} x(x^2-1)= 0 \end{equation*}

as a first step. To solve this equation either factor can be zero. From the first factor we have \(x=0\text{,}\) additionally from the second we can solve \(x^2-1=0\) to find the locations of other solutions. We have three strategies, guessing solutions, attempting to factor, or applying the quadratic formula (the second equation is a quadratic after all).

If we factor, we might notice this as a difference of squares (\(x^2\) and \(1\) can each be written as squares, of \(x\) and \(1\text{,}\) respectively). We have

\begin{equation*} x^2-1 = (x+1)(x-1) \end{equation*}

From here we set each factor to zero to find the \(x\) values that solve this equation and the original equation. The value are \(x = -1\) and \(x=1\text{,}\) or \(x=\pm 1\text{.}\) Alternatively we could apply the quadratic formula to this quadratic with \(a = 1\text{,}\) \(b=0\text{,}\) and \(c=-1\)

\begin{equation*} \begin{aligned}[t] x \amp= \dfrac{-0\pm\sqrt{0^2 - 4(1)(-1)}}{2(1)}\\ \amp = \dfrac{\pm\sqrt{4}}{2}\\ x \amp = \pm 1\\ \end{aligned} \end{equation*}

Taking a peek back at the original formula for \(g(x) = x^3-x\) we might notice we could have originally factored this as \(g(x) = x(x+1)(x-1)\) (we can multiply this out to check that it agrees with the original formula). The factored form illustrates that for \(g(x) = 0\) any one of the factors can be zero, each of which agrees with our solutions above.

Subsection 2.7.4 Power functions

¶

...

In some sense power functions are simpler than polynomials, they only contain a single term. But often that term is less familiar than those that occur in polynomials. For example, we might be interested in \(f(x) = x^{0.5}\) or \(g(x) = x^{0.75}\) where are non-integer powers of the variable of interest. These functions take on very different shapes and are useful in a variety of situations. The square root function is an example of a power function \(\sqrt{x} = x^{0.5}\) which has the shape of a `sideways parabola', \(x^1 = x\) is another (a straight line), \(x^2\) which has the shape of a normal parabola, as well as \(x\) raised to any other power. We can also put a constant factor out in front, for example

\begin{equation*} f(x) = 1.4x^{0.24} \end{equation*}

Power functions are often used as formulas to represent simple nonlinear patterns. Three classic examples are species-area accumulation curves, the skull length-backbone length relationship in fossil fish, and the connection between the metabolism-body mass relationship of mammals and the accidental overdoes of an elephant with LSD.

Subsection 2.7.7 Kleiber's Law curve

¶

...

In terms of energy use (i.e., metabolism), a mouse must eat half of its body mass per day in order to survive, an elephant much less. In fact, the energy metabolism, \(E\text{,}\) of an animal as a function of its body mass, \(M\text{,}\) is described by

\begin{equation*} E = (\#)M^{0.75} \end{equation*}

This relationship holds across a remarkable range of organism sizes and was originally built by trying to understand the relationship between the surface area (amount of skin) and volume (amount within the skin).

Subsection 2.7.8 Rational functions

¶

...

Rational functions are any ratios of two separate polynomials. For example,

\begin{equation*} r(x) = \dfrac{-4.9x^2+10x+12}{x^3-x} \end{equation*}

is a rational function. These functions can take on a variety of interesting and useful shapes as graphs.

Subsection 2.7.9 Exponentials and Logarithms

¶

...

Most importantly exponential functions are of the form

\begin{equation*} (\text{number})^\text{variable} \end{equation*}

in contrast to power functions which are of the form

\begin{equation*} (\text{variable})^\text{number} \end{equation*}

Exponential functions (see Fig. \ref{fig::exp}) seem to be frequently confused with quadratic functions (see Fig. \ref{fig::power}). Looking at the graphs however, the quadratic has a symmetry (for \(x^2\) a fold along the vertical axis) that the exponentials lack.

Exponential decay functions are plotted in Figure \ref{fig::expdecay}.

Example 2.7.6. ....

Exponential functions show up in a variety of places, such as population growth. A return to paperfolding provides a somewhat surprising connection to exponential functions. Taking the thickness of a piece of paper after fold number \(n\) to be represented by \(t_n\text{,}\) we can track the thickness of the paper as we fold. For example,

\begin{equation*} t_1 = 2t_0 = 2(0.01) = 0.02 \end{equation*}

and

\begin{equation*} t_2 = 2t_1 = 2(0.02) = 0.04 \end{equation*}

After \(n\) folds we actually have doubled \(n\) times, and can write

\begin{equation*} t_n = 2^n(t_0) \end{equation*}

This is precisely exponential growth, as illustrated in Table \ref{tab::paper}. Since the width is divided in half after each fold, we can also calculate the remaining width of paper in a similar manner.

An easily overlooked complication lies in Table \ref{tab::paper}: by the sixth fold, the paper is actually thicker than it is wide. There is no longer enough paper to fold! \textit{Assuming} that we could fold the paper 100 times, it would be \(1.268\times10^{28} \text{mm}\) (or \(1.268\times10^{22} \text{kilometers}\)) thick! These results, are illustrated in Fig. \ref{fig::paper_exp}. The diameter of the earth is \(1.2742\times10^4 \text{kilometers}\text{,}\) so this result is shocking if not almost incomprehensible.

As a follow-up to exponential functions are logarithmic functions. Recall that we can think of the logarithm of some number of interest as the power to which we raise another number to get that number of interest. For example, in terms of powers of \(10\text{,}\)

\begin{equation*} 10^4 = 10,000 \text{can be expressed as} \log_{10}(10,000) = 4 \end{equation*}

Each of these translates to the understanding that \(4\) factors of \(10\) is ten thousand or \(10\) to the fourth power is \(10,000\text{.}\) Similarly the relationship

\begin{equation*} 2^4 = 16 \text{can be expressed as} \log_2(16) = 4 \end{equation*}

In addition to thinking of single calculations of logarithms of numbers, we can talk about logarithm \textit{functions}. We plot the graph of the natural logarithm in Fig. \ref{fig::ln}. The natural logarithm \(\ln(x)\) gives the exponent to to which we raise \(e\) to arrive at the argument of the natural logarithm.
