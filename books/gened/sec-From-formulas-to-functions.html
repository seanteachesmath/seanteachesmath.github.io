Section 2.2 From formulas to functions

¶

...

Given a formula we are really just a step away from thinking of functions. While a function has some extra mathematical properties, we can really think of a function \(f(x)\) as the label for a particular calculation involving the quantity \(x\text{.}\) The major string attached to this designation is that a particular \(x\)-value can only give rise to a single calculated value.

Imagine referring to the calculation of the quantity \((50+x)(50+x)\text{,}\) it would be a bit of a drag if, any time that we wanted to make reference to the calculation, we had to refer to the complete formula. Instead we could assign this special calculation to a function \(f(x) = (50+x)(50+x)\text{,}\) at which point we could refer to our action by either the formula or the function name. Setting \(x = 3\) and calculating we could say ''lets take the quantity \(50\) plus \(3\) times the quantity \(50\) plus \(3\)'' to calculate a particular value. As a much more convenient alternative we could simply say \(f(3)\text{,}\) the value of the function whose formula defines the calculation.

Now, a major source of confusion about functions lies in the notation itself. In particular, parentheses show up in two places, with two different meanings,

\begin{equation*} \underbrace{f(x)}_{\text{function notation}}= \underbrace{(50 + x)(50 + x)}_{\text{grouping and multiplication}} \end{equation*}

We read the notation \(f(x)\) as ''\(f\) of \(x\)'' referring to the value generated by a particular choice of \(x\text{;}\) these parentheses serve as `function notation'. In contrast, each pair of parentheses on the right (''\((50 + x)\)'') serve as grouping the two terms to be added; abutting these two factors indicates a step of multiplying the results of the two completed addition operations.

As an added example of the use and misuse of parenthesis with respect to function notation consider the natural logarithm function \(\ln(x)\text{.}\) When solving equations that require the application of natural logarithms for simplification, novice users often write things like \(3\cdot \ln\) when they mean, or should mean, \(\ln(3)\text{.}\) This can be contrasted with how we use the square root symbol `\(\sqrt{\phantom{8}}\)'. To take the square root of \(8\text{,}\) we would always write `\(\sqrt{8}\)' and not `\(8\cdot\sqrt{\phantom{8}}\text{.}\)'

Sometimes we say \(y = f(x)\) to indicate that a particular function \(f(x)\) generates a value called \(y\text{,}\) to form an ordered pair \((x, y)\text{.}\) Occasionally we try to use more meaningful symbols than \(y\text{,}\) \(f\text{,}\) and \(x\text{.}\) For example, in discussing our pace we could somewhat generically write a function \(f(x) = 0.3218x\text{.}\) This gives very little insight into the meaning of the symbol \(x\) or the result of the calculation. We might then say \(p = f(x) = 0.3218\text{,}\) just as we have said \(y = f(x) =0.3218\) in many instances before. Better yet would be \(p = p(x) = 0.3218\text{,}\) where the function and variable name are the same - one fewer thing to keep track of - and indicative of the measurement of `pace'. Better still would be \(p = p(t) = 0.3218t\) were the variable \(t\) could be a reminder of the measurement of time.

Given a function there are really two things we might do

  1. ''plug in'' or evaluate with a known input value

  2. set up and solve an equation for an unknown input value

The first of these tends to be a bit easier, the second can be much harder (or in some cases impossible using or typical methods).

In this case, given \text{any} value for \(x\) we can evaluate our function \(f(x) = (50+x)(50+x)\text{.}\) For example, taking \(x=6\) we write

\begin{equation*} f(6) = (50+6)(50+6) = (56)(56) = 3136 \end{equation*}

and read ''\(f\) of \(6\) equals the quantity \(50\) plus \(6\) times the quantity \(50\) plus \(6\text{.}\)'' We plug in \(6\) for every instance of \(x\) on the left-hand side and the right hand side. We then use arithmetic operations to calculate the value of the expression on the right-hand side. For \(x=0\) we find

\begin{equation*} f(0) = (50+0)(50+0) = (50)(50) = 2500. \end{equation*}
