If we are ultimately interested in modeling using mathematical functions, we need to build up a catalog of mathematical functions. Before we diversify too much, we will spend a but of time looking at quadratic functions. Quadratic functions

\begin{equation*} f(x) = ax^2 + bx + c \end{equation*}

coupled with the quadratic formula

\begin{equation*} x = \dfrac{-b\pm\sqrt{b^2-4ac}}{2a} \end{equation*}

which gives the \(x\) values that solve \(f(x) = 0\text{.}\) What is surprising, perhaps, is just how often and in how many places the quadratic formula shows up as the lynchpin of so many mathematical problems. Partly this is because so many problems are ultimately studied in two dimensions. For example, we are used to visualizing so many problems in the \(xy\)-plane.
