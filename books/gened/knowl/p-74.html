Here `roots' refers to the zeros of a function \(f(x)\) or the solutions to the equation \(f(x) = 0\text{.}\) This is the set of all numbers (associated to a particular function) for which plugging in these values returns the function value of zero. This can be most easily pictured as where the graph of a function crosses the horizontal axis, though this isn't always enough.
