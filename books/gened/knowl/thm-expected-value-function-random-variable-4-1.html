Theorem 1.1.2. expected value of a function of a random variable.

If \(\displaystyle X\) is a discrete random variable and \(\displaystyle f(x)\) is the value of its probability distribution at \(\displaystyle x\text{,}\) the expected value of \(\displaystyle g(X)\) is given by

\begin{equation*} E[g(X)] = \sum_x g(x)\cdot f(x) \end{equation*}
