The Greek letters are called `parameters'; these are symbols used to represent various numbers like the transmission parameter \(\beta\) or the recovery rate \(\gamma\) (see Table \ref{tab::epi}). Sometimes it is easier to write and study the model generically without having to worry about the precise decimal values that go in these positions, so we use the Greek letters as placeholders. This is similar to how we use the formula \(y=mx+b\) to describe a linear function between variables \(y\) and \(x\text{.}\) Whether or not we have acknowledged this before, \(m\) and \(b\) are parameters of the linear formula (`slope' and `\(y\)-intercept', respectively). Changes to these numbers might change the appearance, but do not change the fundamental nature of the relationship between \(y\) and \(x\text{.}\) % ">
