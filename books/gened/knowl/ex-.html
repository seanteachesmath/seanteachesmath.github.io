Checkpoint 2.4.17.

We approximated \(\sqrt{3}\) to 8 correct digits in only 4 steps starting from \(x_0=1\text{.}\) Do you think that the choice of \(x_0\) matters? For each of \(x_0\) equal to each integer from \(1\) to \(10\) see how many steps it takes to reach 8 correct digits from that number. Graph your results with points illustrating the number of steps required (vertical) against the value of \(x_0\) (horizontal). This can be done by hand but will be a miserable amount of work, instead see if you can use spreadsheet software to automate the calculation.
