Similarly, we can add the remaining equations,

\begin{equation*} \begin{aligned} \frac{d^{} {N}}{d {t}^{}} & = -k_+ N X_0 + k_- X_1\\ \frac{d^{} {X_0}}{d {t}^{}} & = -k_+ N X_0 + k_- X_1 + k_i X_i\\ \frac{d^{} {X_1}}{d {t}^{}} & = k_+ N X_0 - k_- X_1 - k_i X_i\\ \frac{d^{} {C}}{d {t}^{}} & = k_i X_1\end{aligned} \end{equation*}

Given this model, our first goal is to make some simplifications through astute observations. Both \(X_0\) and \(X_1\) are receptor states, and if we add the two differential equations,

\begin{equation*} \frac{d^{} {X_0}}{d {t}^{}}+\frac{d^{} {X_1}}{d {t}^{}}=0, \end{equation*}

we see that the total receptor concentration is conserved, \(R\text{.}\) With this we can write \(X_0=R-X_1.\)

\begin{equation*} \begin{aligned} \frac{d^{} {N}}{d {t}^{}} & = -k_+ N (R-X_1) + k_- X_1\\ \frac{d^{} {X_1}}{d {t}^{}} & = k_+ N (R-X_1) - k_- X_1 - k_i X_i\end{aligned} \end{equation*}

Reshuffling a bit,

\begin{equation*} \begin{aligned} \frac{d^{} {N}}{d {t}^{}} & = -k_+ N R+(k_+N + k_-) X_1\\ \frac{d^{} {X_1}}{d {t}^{}} & = k_+ N R- (k_+ N + k_- + k_i )X_i\end{aligned} \end{equation*}
