Recall,

\begin{equation*} \frac{d^{} {N}}{d {t}^{}}=(b-m)N. \end{equation*}
\begin{equation*} \begin{aligned} \frac{dN}{N}&=r\,dt\\ \int_0^s\frac{dN}{N}&=\int_0^s (b-m)\,dt\\ \ln(N)\Big|_0^s&=(b-m)t\Big|_0^s\\ \ln(N(s))-\ln(N_0)&=(b-m) s\\ \ln(N(s))&=\ln(N_0)+(b-m) s\\ N(s)&=\exp(\ln(N_0)+(b-m) s)\\ &=\exp(\ln(N_0))\exp((b-m) s)\\ N(s)&=N_0\exp((b-m) s)\\\end{aligned} \end{equation*}

Replacing \(s\) by \(t\text{,}\)

\begin{equation*} N(t)=N_0\exp((b-m) t), \end{equation*}

and we can see that when \(b<m\text{,}\) \(N(t)\rightarrow0\text{.}\) Otherwise, when \(b>m\text{,}\) \(N(t)\rightarrow\infty\text{.}\) If \(b=m\text{,}\) then \(r=0\) and \(N(t)\) never changes from \(N_0\text{.}\) We might be interested in using the formula for \(N(t)\) to compute the half-life (commonly \(\tau_{1/2}\)) or doubling time (commonly \(\tau_2\)).
