We notice that our model can be written as a set of two updating functions each depending on two variables,

\begin{equation*} \begin{aligned} Y_{t+1}&=F(Y_t, Z_t)=rY_t-pZ_tY_t\\ Z_{t+1}&= G(Y_t, Z_t)=gY_tZ_t+mZ_t, \quad\text{this should be a $+m$ not a $-m$!}\end{aligned} \end{equation*}

Now, the motivation and methodology are the same as with 1D models, we are interested in thinking about \(Y_t=Y^*+y_t\) and \(Z_t=Z^*+z_t\text{,}\) which means each solution (e.g., \(Y_t\)) is a combination of equilibrium value (e.g., \(Y^*\)) and some small perturbation (e.g., \(y_t\)). The Jacobian is a matrix of partial derivatives of our updating functions with respect to the state variables. We could derive this as we did with a Taylor series expansion of a 1D model.
