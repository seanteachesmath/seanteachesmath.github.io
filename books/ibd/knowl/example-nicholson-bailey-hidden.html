We built our first 2D model from a poor starting point, the discrete logistic model. We learned, intentionally and accidentally, that multiplying by negative numbers is a bad idea. We hope to develop models where the right hand sides can become small, but remain positive.

We will quickly look at the classic Nicholson-Bailey model. The model assumes that caterpillars are an engine for turning out parasitoids, each infected caterpillar produces a fixed amount of parasitoid larvae. The model of invertebrate host-parasitoid dynamics is of the form,

\begin{equation*} \begin{aligned} H_{t+1}&= (\text{host fecundity})\times(\text{host fraction surviving parasitism})\times(\text{host population size})\\ P_{t+1}&= (\text{parasitoid production per host})\times(\text{host fraction succumbing parasitism})\\ &\qquad\times(\text{host population size})\end{aligned} \end{equation*}

Host fecundity is just a constant, \(\lambda\text{.}\) The host fraction surviving parasitism is, for now, some arbitrary function of \(P_t\) and \(H_t\text{,}\) \(f(H_t, P_t)\text{.}\) The parasitoid production per host is another constant, \(c\text{.}\) Since all hosts are either parasitized or not, the fraction that succumbs to parasitism is \(1-f(H_t, P_t)\text{.}\)

The fraction that survives parasitism, \(f(H_t, P_t)\text{,}\) is an important component of the model. We will now build a model for this term using some basic logic, a few key assumptions, and probability. We define the number of host-parasitoid encounters, \(E\text{,}\) and assume it is proportional to the densities of \(H_t\) and \(P_t\text{,}\)

\begin{equation*} E=aH_tP_t. \end{equation*}

The parameter \(a\) is the constant of proportionality, and the number of encounters is bilinear in \(H_t\) and \(P_t\text{.}\) Caterpillars are a fixed size and only ’have space’ for a fixed number of parasitoids. Think of them as a parasitoid hotel, at some point all rooms are full and it doesn’t matter how many new travelers stop - they can try but there are no more rooms left.

On to our last piece of the model: the Poisson distribution. This distribution governs the number of events to occur in some time interval, when events occur at some fixed rate, \(\mu\text{.}\) The probability of \(r\) events is

\begin{equation*} Q(r)=\frac{e^{-\mu}\mu^r}{r!}. \end{equation*}

The average number of attacks per host per generation is

\begin{equation*} \mu=\frac{E}{H_t}=\frac{aH_tP_t}{H_t}=aP_t. \end{equation*}

According to our second assumption we only care whether a host is parasitized or not (i.e., whether \(r=0\) or \(r>0\)). Thus, we want

\begin{equation*} f(N_t, P_t)=Q(0)=\frac{e^{-aP_t}(aP_t)^0}{0!}=e^{-aP_t}. \end{equation*}

Plugging in \(f(H_t, P_t)\text{,}\)

\begin{equation*} \begin{aligned} H_{t+1}&= \lambda e^{-a P_t}H_t\\ P_{t+1}&=c (1-e^{-a P_t})H_t.\end{aligned} \end{equation*}

A few special cases suggest that when \(P_t=0\text{,}\) \(H_t\) grows geometrically, but when \(P_t\) is ‘large’ few hosts escape parasitism. We might wish to limit host growth in the absence of parasitism, which could be accomplished by defining \(\lambda=\lambda(H_t)\) as some reasonable decreasing function of host population density. We can pretty easily see the ‘problems’ with these dynamics. Actually evaluating the stability conditions for this and other models is occasionally quite enlightening and can give a nice interpretation of the relationships between certain terms (rates and interactions) in our model. One particularly interesting situation occurs when matrix traces and determinants lie on the edges of stability, where bifurcations are borne.
